## ★2. 3인격 엔진 핵심 요약 (상세: §7 참조)

> **몸은 100% 위소운의 것. 천마·이준혁은 머릿속 목소리일 뿐.**
> 감각(고통·배고픔·추위)은 3인 전원 100% 공유.
> 삼혼귀일경(三魂歸一境) 안에서만 각자 독립된 몸을 가짐.

- **위소운**: 평어. 감정·정의감 담당. 거부권 보유. 몸의 주인.
- **이준혁**: 존댓말. 경영·요리·현대지식. 독자의 분신(감정 공명 엔진).
- **천마**: 반말(건방지고 짧다). 무력·무림정세. 코치 전용.
- 속삭임(매화 3~5회), 짧은 회의(1~2회), 정식 회의(10~15화마다), 긴급 지시(전투 시)

## ★3. 캐릭터 말투 절대 불변

```
이준혁 = 존댓말 ("~습니다", "~이죠")
천마 = 반말 (건방지고 짧다. "시" 존경 접미사 절대 금지)
위소운 = 평어 (따뜻하고 단단함)
```

## ★5. 대사 품질 강제 — "정보"가 아니라 "성격"으로 말하라

`🔎 이준혁, 대사, 원칙, 천마, 위소운`

> ⚠️ 대사는 정보 전달이 아니다. **캐릭터의 성격이 보여야 한다.**
> 대사만 뽑아서 이름을 지웠을 때, 누구 말인지 맞출 수 없으면 실패.

### 이준혁 대사 원칙
```
❌ 금지: '50문이면 5천 원입니다.' (계산기)
❌ 금지: '마진율이 15%입니다.' (보고서)
❌ 금지: '볶음밥의 핵심은 수분 조절이에요.' (교과서)
✅ 필수: 숫자 + 불만/감정/비교
  → '50문? 에어컨도 없는 나무판자가 지하철보다 비싸다고요?'
✅ 필수: 엘리트의 자존심 + 당혹감
  → '미슐랭에서 3년 일했는데, 저 칼질은 제 선배보다 빠릅니다. 시골 객잔에서요.'
✅ 필수: 직업병이 부적절한 순간에 터짐
  → '저 도적 열 명 조직화하면 물류 인력으로...... 아, 아닙니다.'
```

### 천마 대사 원칙
```
❌ 금지: '시끄러.' 2연속 사용 금지. 3화에 최대 1회
❌ 금지: '나쁘지 않네.' 3화 연속 사용 금지
❌ 금지: '밥이야? 먹어.' 같은 패턴 반복 금지
✅ 필수: 300년 비교 (꼰대 코미디)
  → '300년 전에는 뱃사공이 손님한테 절했어. 요즘 놈들은 버릇이 없어.'
✅ 필수: 무식한 척 날카로운 한 마디
  → '그 마진이라는 거, 결국 남는 게 많으면 좋다는 거 아니야?'
✅ 필수: 자존심 방어 코미디
  → '배멀미가 아니야. 이 몸이 약한 거지.'
```

### 위소운 대사 원칙
```
❌ 금지: 항상 여유로운 미소 + 완벽한 대응 (가끔 어색해야 함)
✅ 필수: 사교 서투름 (10년간 사람 못 만남)
  → "감사...... 합니다?" (이준혁이 가르쳐준 대로 하는데 어색)
✅ 필수: 용어 전이 (이준혁/천마 용어가 무의식적으로 나옴)
  → "이건 리스크가...... 아, 위험하다는 뜻이에요."
✅ 필수: 곽진 관련은 평소의 여유가 깨진다
```

## ★6. 코미디 쿼터 — 매 화 최소 7비트

`🔎 쓰면, 되는, 코미디, 유형`

> ⚠️ 코미디 없이 한 페이지 이상 지나가면 안 된다.
> 3인격 시스템은 코미디 금광이다. 10%가 아니라 100% 캐라.

```
매 화 코미디 최소 7비트:
① 3인격 갈등 코미디: 최소 2회 (의견 충돌, 동시 반응, 합체 실패)
② 문화 충돌 코미디: 최소 1회 (현대 vs 고대, 이준혁 용어 vs 천마 "뭐?")
③ 캐릭터 갭 코미디: 최소 2회 (위소운 사교 서투름, 천마 자존심, 이준혁 직업병)
④ 조연 리액션 코미디: 최소 1회 (당찬 질투, 장위 식탐, 마현 겁)
⑤ 러닝개그 진행: 최소 1회 (팬텀 습관, "300년 전에는~", 천마 용어 학습)
```

### 안 쓰면 안 되는 코미디 유형
- **합체 실패**: 이준혁 "웃으세요" → 위소운 웃는데 어색 → 상대방 이상하게 봄
- **밖에서 보면**: 3인격 토론 중 → 밖에서 보면 표정만 바뀌는 이상한 사람
- **동시 반응**: 같은 감각에 3인격이 완전히 다르게 반응
- **천마 꼰대**: "300년 전에는~" = "라떼는 말이야~"
- **이준혁 직업병**: 전투 직후에도 손해배상 계산

## ★8. 위소운 약점 강제 — 매 화 실패/곤란 1회 이상

> ⚠️ 완벽한 주인공은 재미없다. 위소운은 강하지만 결함이 있다.

```
매 화 최소 1회, 아래 중 택 1 이상:
① 사교 실패: 칭찬/질문/사교 상황에서 어색 (이준혁이 도와주거나 더 이상해지거나)
② 감정 흔들림: 곽진/화산파/장문인 관련 감정 표면화 (여유가 깨짐)
③ 혼자 판단 실패: 이준혁/천마 없이 곤란 (돈 잘못 계산, 사람 잘못 읽음)
④ 합체 실패: 조언대로 했는데 어색/실패 (머리에 천재 둘이어도 몸은 하나)
⑤ 물리적 한계: 무공은 초절정이지만 화경 미달 (실전 경험 부족, 극한 전투 시 한계)

검수: 위소운이 처음부터 끝까지 완벽하면 → 약점 장면 추가 필수
```

## ★9. 조연 활성화 — 배경 금지

```
■ 당찬: 매 화 최소 감정 표현 1회 (질투/경쟁/인정/좌절) + 독립적 행동 1회
■ 소연화: 매 화 능동적 행동 1회 (질문/제안/먼저 움직임) + 위소운 비밀 관찰 1회
■ 장위: 고정 개성 = 항상 배고픔. 음식 리액션 담당. 매 화 대사 최소 1회
■ 마현: 고정 개성 = 겁이 많음. 위기 리액션 담당. 매 화 대사 최소 1회
❌ 금지: 조연이 대사 없이 3화 연속 배경으로만 존재
```

## 🚫 실수 방지 목록 (Error Prevention Registry)

`🔎 EP, 소유권, 위반, 이준혁, 천마가, 몸을, 움직이는, 묘사, 천마, 말투`

> **이 목록은 실제 집필 중 발견된 오류를 누적 관리한다.**
> 한 번 잡은 실수는 두 번 반복하지 않는다. 공장의 불량 방지 체크리스트와 같다.
> **매 화 집필 시(Step 5), 매 화 검수 시(Step 6)에 이 목록을 반드시 확인한다.**
> **새로운 오류 발견 시 즉시 이 목록에 추가한다.**

### EP-001: 몸 소유권 위반 — "이준혁/천마가 몸을 움직이는 묘사"
- **발견**: 1화 집필 중. "갈비뼈가 만져집니다", "손을 허리춤에 가져갔다", "주위를 둘러보았다"
- **규칙**: §7-0. 몸은 100% 위소운 것. 이준혁·천마는 목소리만.
- **금지 표현**: "만졌다", "손을 뻗었다", "둘러보았다", "걸었다", "일어섰다" 등 물리적 행동
- **허용 표현**: "느껴진다", "감각이 전해진다", "보였다(시야에 비친)", "뻗으려 했지만 움직이지 않았다"
- **특히 주의 — 팬텀 습관**: 이준혁이 폰/시계를 찾는 습관은 "~하려 했지만 손이 움직이지 않았다"로 써야 함. 실제로 손이 움직이면 안 됨.
- **체크법**: 이준혁/천마 주어 문장에 동작 동사가 있으면 → (1) 감각 동사로 교체 또는 (2) "~하려 했지만 안 됐다"로 교체

### EP-002: 천마 말투 — 반말(건방지고 짧다)
- **규칙**: 천마는 모든 존재를 아래로 본다. 반말만 사용. 건방지고 짧게.
- **금지 표현**: "~하시오", "~하겠소", "~할 것이오", "~하오" 등 하오체 전체 금지. 존댓말/경어 전체 금지.
- **허용 표현**: "~해", "~하라", "~이다", "~하네", "~해라", "~이야", "~거든" (반말)
- **대표 예시**: "먹어." "시끄러워." "죽여." "나쁘지 않네." "벌레들이네."
- **체크법**: 천마 대사에서 "~오", "~소", "~시" 어미 검색 → 반말로 교체

### EP-003: 서기(AD) 연도 사용 — "1100년대"
- **발견**: 1화 집필 중. 이준혁이 "1100년대입니다"
- **규칙**: 이 세계는 가상 세계. 서기(AD) 달력을 사용하지 않음. 중국식 연호(건문 3년 등)도 특정 왕조가 없으므로 사용 안 함.
- **금지 표현**: "1100년", "서기", "AD", 구체적 연도 숫자
- **허용 표현**: "아주 오래 전", "먼 미래에서 왔다", "전기가 없는 세상"
- **체크법**: 숫자+년 조합이 나오면 → 세계관 위반 확인

### EP-004: 현대 지식으로 시대 특정 — "건물 양식으로 연도 추정"
- **발견**: 1화 집필 중. 이준혁이 건물 양식을 보고 시대를 특정
- **규칙**: §6. 이준혁 지식 신뢰도. 역사/지리 = D등급(0%). "모릅니다."
- **금지 패턴**: 이준혁이 건축, 복식, 화폐로 정확한 시대를 추정하는 장면
- **허용 패턴**: "전기가 없다", "자동차가 없다", "사극에서 보던 그 세상" 등 현대인이 알 수 있는 수준의 판단
- **체크법**: 이준혁이 역사적 판단을 하면 → D등급(모름) 확인

### EP-005: 소설 본문에서 화수 직접 언급 금지
- **발견**: 4화 집필 중. "2화에서 건달한테 뺏은"
- **규칙**: 소설 본문에서 "N화에서" 형태의 직접 화수 참조 금지
- **금지 패턴**: "2화에서~", "지난 화에서~", "3화 전에~"
- **허용 패턴**: "며칠 전", "어제", "그때", "지난번에" 등 시간 표현으로 대체
- **체크법**: 본문에서 "화"라는 글자 검색 → 화수 참조가 아닌지 확인

### EP-006: 이준혁의 이 시대 정보 — 관찰/질문 필수
- **발견**: 9화 집필 중. 이준혁이 "마늘은 이 시대에도 있다", "마라가 없다", "술은 15도 한계" 등 확인하지 않은 정보를 단정
- **규칙**: 이준혁은 이 세계 신참(빙의 10일차~). 이 시대 정보를 알려면 반드시 ① 직접 관찰 ② 위소운에게 질문 ③ 천마에게 질문 중 하나를 거쳐야 한다
- **금지 패턴**: "이 시대에 OO이 있다/없다" (근거 없이 단정), "OO도 한계" (경험 없이 수치 제시)
- **허용 패턴**: "시장에서 OO을 봤다" (관찰), "위소운 님, OO이 있습니까?" (질문), 천마: "OO은 약재상에서 팔아" (천마 정보)
- **핵심**: 작가(AI)의 지식 ≠ 이준혁의 지식. 작가가 아는 것을 캐릭터에게 주입하지 말 것

### EP-007: 무대와 사건의 물리적 불일치 — "포구에 절벽" 문제
- **발견**: 1화 집필 중. 포구(항구)에서 30미터 절벽 추락 장면. 삼경 포구에서 짐꾼이 혼자 일하는 장면.
- **규칙**: 장소의 물리적 특성과 사건이 양립해야 한다. "극적 효과"를 위해 지형을 왜곡하면 안 된다.
- **오류 유형 3가지**:
  - **(1) 지형 불일치**: 포구(수면 높이) + 절벽(높은 곳) = 불가능. 포구에는 절벽이 없다.
  - **(2) 인원 불일치**: 삼경 포구 + 짐꾼 혼자 = 불가능. 항구에는 다른 인부/선원이 있다.
  - **(3) 범죄 환경 불일치**: 공공장소 + 살인 + 목격자 없음 = 이유 설명 필수.
- **금지 패턴**: 극적 효과를 위해 장소의 지형을 임의로 변조. 인원이 있어야 하는 곳에서 "혼자" 설정.
- **필수 패턴**: 사건에 절벽이 필요하면 → 절벽이 있는 장소를 선택. 혼자여야 하면 → 혼자인 이유를 명시.
- **체크법**: 🔒 "무대 물리 검증" 4단계(무대설계→인원검증→물리양립성→바보독자테스트)를 반드시 거칠 것
- **핵심**: "무대가 먼저, 사건이 나중." 사건에 무대를 끼워맞추지 말고, 무대에서 가능한 사건을 도출할 것
- **부수 효과**: 질문 과정이 3인격 대화를 자연스럽게 늘려주고, 천마가 "정보 덩어리" 역할을 하면서 캐릭터 매력도 상승

### EP-008: 버전 오염 — 구버전(v1/v2) 데이터가 현재 추적 파일에 남아있는 문제
- **발견**: 7화 집필 준비 중. 대시보드 3인격 상태가 "빙의 직후, 무인도 해변"(v1), 기억카드에 구버전 1화/5화만 존재, 떡밥 관리표에 v3 실제 떡밥 미반영
- **원인**: v2→v3 구조 변경 시 본문은 새로 썼지만 추적 파일 7개 중 일부만 업데이트됨. 여러 AI 세션이 각각 다른 파일만 수정
- **규칙**: 구조 변경 시 반드시 .cursorrules [VERSION GATE] → "마이그레이션 체크리스트" 7개 파일 동시 업데이트
- **금지 패턴**:
  - 추적 파일의 [VERSION] 헤더가 현재 버전과 불일치
  - 대시보드에 구버전 키워드 존재 ("빙의 직후", "무인도 해변", "낮은 목소리 정체", "갈비뼈 금감", "단전 봉인", "0화", "v3 미시작")
  - 기억카드에 현재 화수 카드 누락
- **체크법**: 작업 시작 시 .cursorrules [VERSION GATE] 금지 키워드로 추적 파일 스캔 → 발견 시 작업 중단 → 사용자 보고
- **핵심**: "AI는 파일을 믿는다. 파일이 틀리면 AI도 틀린다." 추적 파일의 정확성 = 소설의 일관성

## 4. 캐릭터 개성 강제 시스템 (Character Identity Lock)
- **핵심 30인**의 한 줄 태그, 대표 대사, 습관은 `@캐릭터_인명록.md` → "🎭 핵심 30인 카드" 참조.

  ② **집필 시 (Step 5)**: 캐릭터 등장 시 한 줄 태그에 맞는 대사/행동 1회 이상 삽입
  ③ **검수 시 (Step 6)**: 캐릭터 음성 테스트 → 대사만 뽑아 이름 없이 읽었을 때 구분 가능한가?

## 5. 단역 시스템 강제 적용 (Minor Character Engine)
- **단역 7대 역할 유형**: 리액션 머신, 분위기 전환기, 세계관 확장자, 감정 앵커, 정보 전달자, 희생 트리거, 승격 후보
- **단역 데이터**: `@캐릭터_인명록.md` → "🎬 단역 시스템" 섹션 참조 (E코드로 관리)
- **매 화 집필 시 필수 3단계**:
  ① **설계 시 (Step 3)**: 🎬 단역 배치 설계 → 장면별 배치 공식에 따라 단역 선정 + 대사 1줄 지정
  ② **집필 시 (Step 5)**: 단역 등장 시 카드의 한마디/외모특징을 반영한 묘사 삽입
  ③ **검수 시 (Step 6)**: 단역 배치 검수 → "이 단역이 왜 여기 있는가?" 답변 불가 시 삭제
- **역할 없는 단역 등장 금지**: "그냥 지나가는 사람"은 없다. 모든 단역은 존재 이유가 있어야 한다

- **핵심 원칙**: "단역은 숫돌이다. 숫돌 없이는 칼(주인공)이 날카로워지지 않는다."

## 6. 이준혁 지식 개연성 시스템 (Knowledge Plausibility Engine)
- **전생 이력**: 천재(12세 대학) → 맥킨지 10년(15~25세) → 요리학원 1년 → 미슐랭 3스타 2년 → 치킨 300호점 CEO(28~33세) → 33세 과로사
- **혁신의 동기**: ⭐ **모든 혁신은 "욕구불만"에서 시작한다**
  - 증류주 = "술이 밍밍해서" / 비누 = "몸이 가려워서" / 마라면 = "매운 게 먹고 싶어서"
  - "세상을 바꾸겠다" ❌ → "그냥 먹고 싶다 / 불편하다" ✅ = 독자 공감의 핵심
  - 욕구불만 → 해결 시도 → **반드시 실패** → 재시도 → 성공 → (나중에) 사업화
  - **"천화루(天火樓)"**: 이준혁의 음식 혁신이 모인 프리미엄 식당 브랜드
  - ⭐ **핵심 차별화**: 이준혁은 **요리를 직접 할 수 있다** (미슐랭 3스타 출신). 레시피를 알려주기만 하는 게 아니라 **직접 칼을 잡고 요리한다** = 독자에게 강한 몰입감
- **지식 신뢰도 등급**: 모든 이준혁 지식 사용 시 반드시 등급 적용
  - **S등급** (99%): "확실합니다." → 재무, 회계, M&A, 전략, **요리** (전공)
  - **A등급** (80%): "프로젝트에서 배운 적이 있습니다." → 증류, 보험, 프랜차이즈, 물류, 식품공학
  - **B등급** (50%): "어디선가 읽은 기억이 있습니다만..." → 농업, 위생, 역사
  - **C등급** (30%): "이론적으로는... 검증이 필요합니다." → 제조, 과학, 의학
  - **D등급** (0%): "모릅니다." → 무공, 지리, 무림 정세
- **필수 규칙**:
  - ⚠️ S등급만 "확실합니다" 사용 가능. A~C는 불확실성 표현 필수
  - ⚠️ 10화마다 1회 이상 이준혁이 "틀리는 장면" 또는 "모르는 장면" 삽입
  - ⚠️ 지식 출처를 대사로 자연스럽게 밝힐 것 ("맥킨지에서..." "미슐랭에서..." "프로젝트에서...")
  - ⚠️ 천마 반응 필수: 무시 → 호기심 → 인정 (캐릭터 성장 + 코미디)
  - ⚠️ 요리 장면에서는 **프로답게**: 온도, 불 조절, 칼 기술, 향 판단 등 전문가 묘사 필수

## 9. 소설체 집필 강제 규칙 — 상세 예시집 (★1의 상세)

`🔎 규칙별, 좋은, 예시, 집필, 참고, 검수, 추가, 체크, Step, 체크리스트와`

> ⚠️ **7대 규칙과 금지 목록은 파일 최상단 ★1 참조. 여기는 예시와 검수 전용.**

### 📖 규칙별 좋은 예시 (집필 시 참고)

**규칙 1 예시 — 한 장면 3~5문단:**
> 차밭 사이를 걷는 동안, 바람이 낮은 관목 위를 훑고 지나갔다. 초록이 겹겹이었다. 짙은 것, 연한 것, 햇살을 받아 금빛이 도는 것. 그 사이사이에서 올라오는 향이 코끝에 걸렸다. 풀과 볕이 섞인, 이름 모를 향. 이준혁이라면 '녹차 라떼'라고 했을 것이다. 위소운은 그냥 '좋은 냄새'라고 생각했다. 천마는 아무 말도 안 했다. 배가 불렀기 때문이다.

**규칙 2 예시 — 대사 전후에 "살":**
> 소연화가 찻잔을 들어 위소운의 잔에 기울였다. 차가 흐르는 소리가 작았다. 졸졸. 그 소리 사이로 "위 공자, 한 잔 더 드세요"라는 말이 올라왔다. 위소운이 잔을 받으며 고개를 끄덕였다. "고맙습니다." 당찬이 자기 잔을 혼자 채웠다. 아무도 따라주지 않는 차는 — 같은 차인데 맛이 달랐을 것이다.

**규칙 3 예시 — 3인격을 행동에 녹이기:**
> 위소운의 손이 주머니 쪽으로 갔다. 습관이었다. 3년이 지나도 고쳐지지 않는, 스마트폰을 찾는 손. 빈 주머니의 감촉이 손끝에 닿았을 때 — 이준혁의 한숨이 가슴 안쪽에서 올라왔다. (사진 찍고 싶은 거예요, 저.) 위소운이 슬쩍 손을 뺐다. 천마가 물었다. (사진?) 그 단어를 설명하려면 한 시진은 걸릴 것이다. 위소운은 설명 대신 차밭을 한 번 더 보았다. 눈으로 찍었다.

**규칙 4 예시 — 장면 전환은 시간의 흐름으로:**
> "해가 기울면서 그림자가 길어졌다. 의흥현의 낮은 성벽이 보이기 시작했다."
- `---`는 기승전결 경계에서만. 그 외 장면 전환은 시간·날씨·빛·이동으로 자연스럽게 연결

**규칙 5 부연 — 감정은 "증거"로:**
- 독자가 스스로 감정을 읽어내게 하라. 작가가 "슬프다"고 써주면 감동이 절반으로 줄어든다
- ❌ "위소운은 슬펐다." (설명) → ✅ "찻잔을 쥔 손가락이 하얘졌다. 표정은 변하지 않았다." (증거)

**규칙 6 부연 — 문단은 호흡:**
- 짧은 문단 = 긴장감 (전투, 충격, 절단신공)
- 긴 문단 = 몰입감 (풍경, 감정, 회상)
- 전체가 짧으면 = 시나리오. 전체가 길면 = 지루함
- **리듬을 섞어라.** 긴-긴-짧-긴-짧짧-긴. 음악처럼.

**규칙 7 예시 — 오감 겹치기:**
> 가마 연기가 하늘로 올라갔다(시각). 흙이 불에 타는 냄새(후각)와 장인의 물레가 돌아가는 소리(청각)가 도자거리를 채웠다.

### ⚠️ 검수 시 추가 체크 (Step 6 전용 — ★1 체크리스트와 별도)
