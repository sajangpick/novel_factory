### 무림대회 설정 (武林大會)

> ⚠️ **무림대회는 30세 이하 누구나 참가 가능한 열린 대회다.**
> 구파일방+오대세가 전용이 아니다. 무림의 모든 젊은 무인이 실력을 증명하는 자리.
> 이것이 대회를 더 풍성하고, 위소운의 우승을 더 가치 있게 만든다.

#### 대회 기본 정보

| 항목 | 내용 |
|------|------|
| **명칭** | 무림대회 (武林大會) / 개봉무림대회 |
| **장소** | 개봉(開封) — 무림의 중심 도시 |
| **주관** | 무림맹 (武林盟) |
| **참가 자격** | ⭐ **30세 이하, 누구나**. 정파/사파/무소속 불문. 단, 마교는 공식 불가(몰래 참가 가능) |
| **주기** | 3년마다 1회 |
| **규모** | 300~500명 지원 → **3연승 예선** → 본선 **32명** → 토너먼트 → 결승 |
| **상금/명예** | 우승자 = "무림제일신예(武林第一新銳)". 상금 + 각 세력 스카우트 경쟁 |
| **시드** | ⭐ **없음**. 명문이든 무소속이든 모두 3연승 예선을 통과해야 한다 |

#### 예선 — 3연승 무대 방식 (挑戰三連勝)

> ⚠️ **핵심 규칙**: 출전 희망자가 무대에 올라가면, 관중/다른 지원자가 도전자로 나선다.
> **연속 3명을 물리치면 32강 진출 확정.** 1패라도 하면 탈락. 시드 없음.

| 항목 | 내용 |
|------|------|
| **무대** | ⭐ 개봉 무림맹 광장. **비무대 1개만 운영** — 모든 관중의 시선이 한 곳에 집중. 경기장의 열기가 폭발. |
| **진행** | 출전자가 무대에 올라감 → 도전자(다른 출전자) 3명을 연속으로 이겨야 통과 |
| **도전자** | 자발적 지원. 누구든 손을 들 수 있음. 강한 상대를 골라 끌어내리려는 전략도 가능 |
| **휴식** | 1전 끝나면 차 한 잔(약 5분) 정도의 짧은 휴식만 허용 |
| **탈락** | 3연승 중 1패라도 하면 **즉시 탈락**. 재도전 불가 |
| **32명 확정** | 32명이 채워지면 예선 종료 |
| **관전** | 관중석 자유 관람. 이 과정에서 출전자들의 실력이 공개됨 |

**3연승 예선이 만드는 드라마:**
- 명문파 제자라도 3연승 못 하면 탈락 → "이름값"이 통하지 않는 공정한 시스템
- 강한 출전자를 일부러 도전해서 탈락시키려는 전략 → "저 녀석을 끌어내려라!"
- 3번째 상대가 의도적으로 강한 도전자를 보내는 연합 전략
- 위소운처럼 무명인이 3연승하면 → 관중이 술렁임 → "저 놈이 누구야?"
- 마교/사파 출전자는 **일부러 평범하게 이김** (실력 숨김)

**주요 예선 장면:**
| 장면 | 내용 | 스토리 효과 |
|------|------|-----------|
| **위소운 예선** | 무명 무소속이 무대에 올라감. 3명을 가볍게 이김. | "저 놈 누구야?" — 첫 주목 |
| **곽진 예선** | 화산파 대제자. 압도적 3연승. 관중 환호. | 정파 스타의 등장. 위소운과 대비 |
| **남궁현 예선** | 3명 모두 검을 뽑기도 전에 끝남. | "소검신은 다르다" — 최강 후보 확인 |
| **야율흑 예선** | 무표정으로 "적당히" 이김. 실력 감춤. | 천마: "저 놈... 숨기고 있어." |
| **탈락 드라마** | 명문 제자가 3번째 상대에게 패배. | 대회의 잔인함. "여기서는 이름이 통하지 않는다" |

#### 본선 — 32강 토너먼트 (단판 승부)

```
32강 (32명, 16경기) → 16강 (16명, 8경기) → 8강 (8명, 4경기) → 4강 (4명, 2경기) → 결승 → 우승
```

#### ⭐⭐ 무림대회 축제 일정표 — "월드컵처럼 즐기는 무림의 축제"

> ⚠️ **핵심 원칙**: 무림대회는 단순 토너먼트가 아니라 **무림 최대의 축제**다.
>
> **현실적 이유**:
> - 경기가 1초에 끝나기도, 1~2시간 걸리기도 함 → 관중이 기다려야 함
> - 무대 파손 시 수리 시간 필요
> - 선수는 한 번 싸우면 회복 시간 필요 (부상, 내공 소모)
> - 전국에서 모인 무인·상인·구경꾼 = **네트워킹과 교류의 장**
>
> 이준혁: "월드컵도 경기만 보러 가지 않습니다. 축제, 이벤트, 팬 존, 만찬...
> 이 대회 기간이 위소운 님의 **인맥 구축 골든 타임**입니다."

##### 전체 일정 — 총 20일

> ⭐ **비무대 1개만 운영** — 관중 전원이 한 곳에 집중. 경기장 열기 폭발.
> 예선: 매일 5~6명씩 선발 → 6일간 32명 확정.
> 본선: 32강~결승 + 쇼케이스 + 쉬는 날(네트워킹).

```
[무림대회 전체 일정표 — 비무대 1개, 총 20일]

━━━ 예선 (挑戰三連勝, 비무대 1개) ━━━
1일차   예선 1일 (6명 선발) + 개막식. 300~500명 지원자 중 도전 시작.
2일차   예선 2일 (6명 선발). 명문 제자들 본격 도전.
3일차   예선 3일 (6명 선발). 무명 무인들의 이변 속출.
4일차   예선 4일 (5명 선발). 위소운 3연승 통과. "저 놈 누구야?"
5일차   예선 5일 (5명 선발). 곽진·남궁현 등 대물 통과.
6일차   예선 6일 (4명 선발, 32명 확정). 마지막 자리 쟁탈전.
7일차   ⭐ 쉬는 날 — 32강 추첨식 + 9대문파 무공 시연회 + 개막 축하연
━━━━━━━━━━━━━━━━━━━━
━━━ 32강 (16경기, 4경기/일) ━━━
8일차   32강 1일차 (4경기) + 5대세가 시연 (남궁세가·사천당가)
9일차   32강 2일차 (4경기) + 시연 (중소문파 합동)
10일차  32강 3일차 (4경기) + 시연 (태행산 표국 호위술)
11일차  32강 4일차 (4경기) + 시연 (곤륜파·개방)
12일차  ⭐ 쉬는 날 — 무림 교류회 (장터+만찬+상인 축제)
━━━━━━━━━━━━━━━━━━━━
━━━ 16강 (8경기, 4경기/일) ━━━
13일차  16강 1일차 (4경기) + 9대문파 시연 2차 (점창·아미)
14일차  16강 2일차 (4경기) ← ⭐ 위소운 vs 곽진 (이 날)
15일차  ⭐ 쉬는 날 — 자유 교류일 (1:1 만남·비무·장터)
━━━━━━━━━━━━━━━━━━━━
━━━ 8강 (4경기) ━━━
16일차  8강 (4경기) + 5대세가 시연 2차 (하북팽가·모용세가)
17일차  ⭐ 쉬는 날 — 무림맹 주최 VIP 대연회
━━━━━━━━━━━━━━━━━━━━
━━━ 4강 + 결승 ━━━
18일차  4강 (2경기) + 은퇴 고수 명인전
19일차  ⭐ 쉬는 날 — 결승 전야. 개봉 전체가 축제 분위기.
20일차  ⭐ 결승전 + 시상식 + 폐막 축하연
```

> 이준혁: "올림픽이 16일, 월드컵이 29일. 20일이면 적당합니다.
> 예선부터 결승까지 20일... 이 기간이 위소운 님의 **인맥 구축 골든 타임**입니다."

> ⚠️ **예선 6일 = 전국 무인의 실력이 매일 공개되는 무림의 쇼윈도**
> 관중은 매일 비무대를 보면서 "강한 놈", "약한 놈"을 골라냄.
> 명문 제자의 탈락, 무명 무인의 이변 = 매일 드라마.
> 위소운이 4일차에 통과하면 → "저 무명이 3일째 강자들 사이에서 살아남았다?"
>
> ⚠️ **쉬는 날 = 스토리의 금광**
> 경기일은 전투 묘사. 쉬는 날은 인간 드라마.
> 위소운이 친구를 사귀고, 이준혁이 사업 파트너를 만나고,
> 천마가 옛 무림의 흔적을 발견하는 날.

##### 쇼케이스 — 무공 시연회 (경기 사이 볼거리)

> **목적**: 관중 즐거움 + 문파 자랑 + 홍보 + 무대 수리 시간 확보
> 경기와 경기 사이, 또는 경기 전후에 15~30분간 시연.
> 월드컵 하프타임 쇼와 같은 역할.

| 일차 | 시연 주체 | 시연 내용 | 스토리 효과 |
|------|----------|----------|-----------|
| 7일차 | **소림사** | 나한권법 72수 시연 (10인 합동) | 정파 대표의 위엄. 관중 환호 |
| 7일차 | **무당파** | 태극검법 쌍인무(雙人舞) 시연 | 검이 춤추듯 아름다움. 여성 관중 열광 |
| 7일차 | **화산파** | 매화검법 시연 (곽진 주도) | ⭐ 위소운 감정 흔들림. 천마: "저 검... 익숙하지?" |
| 8일차 | **남궁세가** | 천잔검법 개인 시연 (남궁현) | 결승 상대의 위엄 사전 공개 |
| 8일차 | **사천당가** | 암기술 표적 시연 (당소령) | 암기의 정밀함. 관중 경악 |
| 9일차 | **중소문파 합동** | 각 지역 대표 무공 3분 시연 | ⭐ 각 지역 무공 소개 = 세계관 확장 |
| 10일차 | **태행산 표국** | 호위 호송술 시연 (엄표) | 실전적 무공. 관중 박수 |
| 11일차 | **곤륜파** | 곤륜검법 시연 (풍설) | 서역의 바람 같은 검 |
| 11일차 | **개방** | 타구봉법 합동 시연 | 거지들의 봉법에 관중 열광 |
| 13일차 | **점창파** | 뇌검법 시연 (하진) | 빠른 검법의 극치 |
| 13일차 | **아미파** | 아미검법 시연 (당영란) | 유려하고 날카로운 여검 |
| 16일차 | **하북팽가** | 팽가도법 시연 (팽대산) | 도(刀)의 호쾌함 |
| 16일차 | **모용세가** | 모용가 차경법 시연 | "상대의 무공을 되돌리다" — 관중 경악 |
| 18일차 | **은퇴 고수 명인전** | 무림맹 장로급 시연 | 절정~화경급 무공 공개. 관중 경외 |

> ⭐ **위소운에게 쇼케이스의 의미 — 기류감응 도약의 핵심**:
> "다양한 문파의 무공을 한자리에서 관전" = 기류감응 폭발적 성장
> 관중석에서 각 문파의 시연을 보며 기의 흐름을 읽는다
> → 만류귀종 이론 90%→99%로 완성
> 이준혁: "무림판 MBA 수업이네요. 경쟁사 분석을 한 자리에서."
> 천마: "좋은 눈이야. 300년 전 나도 이렇게 배웠다."

#### ⭐⭐⭐ 무림대회 네트워킹 — 위소운이 만나는 핵심 인물들

> ⚠️ **무림대회 20일은 위소운·이준혁에게 "인맥 구축 골든 타임"이다.**
> 전국 각지에서 모인 문파주·세가주·상인·표국주·유지들을 한 자리에서 만날 수 있는 유일한 기회.
> 예선 6일간 매일 관전하면서 자연스럽게 관중석에서 인연이 시작된다.
> 여기서 만난 사람들이 → 12개 성 지사 파트너의 핵심 후보가 된다.
>
> 이준혁: "비즈니스 컨벤션입니다. 20일이면 충분합니다. 예선부터 결승까지."

##### 만남별 상세 장면 설계

**예선 기간 (1~6일차) — 관중석에서 시작되는 인연:**
- 매일 비무대 관전 → 관중석에서 자연스러운 대화 시작
- 엄표(태행산): 위소운 4일차 3연승 관전 후 감탄. "저 검법... 처음 봅니다."
- 조비연(산동): 엄표와 함께 관전 중 소개. "비연표국 조비연이오."
- 이준혁: 예선 6일간 관중석 인물 파악 완료. '사전 조사는 끝났습니다.'

**7일차 — 32강 추첨식 + 개막 축하연 (첫 공식 인상):**
- 위소운: 예선 3연승 무명으로 주목받기 시작. 32강 추첨에서 곽진과 같은 블록!
- 백운학(항주): "소협의 검법이 인상적이었소." → 위소운 어색하게 인사
- 이준혁: (내면) '저 사람 항주라고? 항주 부동산 시세가 궁금합니다.'

**12일차 — 무림 교류회 (핵심 네트워킹):**
- ⭐ 이 날이 가장 중요. 32강이 끝나고 전체 참가자+관중+상인이 모이는 대형 장터+교류회
- 여상진(소주): 이준혁이 증류주 시음을 권함 → "이... 이건 뭡니까?!" → 사업 동기 점화
- 곽대용(만리표국): 이준혁의 "분업 제안"에 "흥미롭군. 더 듣겠네" → 15일차 약속
- 이준혁: 32강 결과로 위소운의 이름값이 올라간 상태. 협상력 상승.

**15일차 — 자유 교류일 (깊은 만남):**
- 진무영(장안): 위소운과 비무 약속 → 비무 후 서로 인정 → "형제라 부르겠소"
- 임소하(홍주): 차 모임에서 위소운이 이준혁이 알려준 차 평가 멘트 시도 → 반은 성공 반은 어색
- 이준혁: 곽대용과 분업 협상 본격화. "장거리는 만리표국, 식품 단거리는 천화표국"

**17일차 — 무림맹 VIP 대연회 (VIP 만찬):**
- ⭐ 위소운이 8강 진출자로서 VIP 석에 초대됨
- 당유란(사천): 손약령과 약재 이야기 → "이런 재료 감별법은 처음..." → 사천 루트 연결
- 해란(복건): "해상 물류? 복건에서 개봉까지?" → 장기적 해상 루트 시발점
- 이준혁의 증류주+요리 소규모 시식회 → 고위층 입소문 → VIP 홍보 전략 연동

**20일차 — 폐막 축하연 (우승 후):**
- 위소운이 "무림제일신예" 우승자로서 만찬의 주인공
- 오학성(광동): "우승자와 사업을?" → 먼저 명함(서신)을 건넴
- 모든 파트너 후보들이 우승자 위소운에게 접근 → 이준혁이 우선순위 정리
- 이준혁: "20일 만에 12개 성의 파트너 후보를 확보했습니다. ROI 무한대."

| **3인격 코미디** | 대기실에서 3인격 작전회의. 천마 "벌레들이네." / 이준혁 "상대 분석 완료." / 위소운 "...긴장돼요." |

| ⭐ **마교 출신 간파** | 위소운이 기류감응으로 야율흑의 마교 기류를 간파(4강). 천마 긴장. 떡밥 투하. |

#### 위소운의 무공 등급 — 대회 시점

| 항목 | 상태 |
|------|------|
| **현재 등급** | **초절정(超絶頂)** — 화경 문턱 |
| **내공** | 태허귀일결 + 정파제일고수(소무진) 내공심법 융합. 체질 문제 해결 완료. |
| **검법** | 화산 매화검법(기본) + 소무진 검법 + 자창 응용 = **독자적 검법** |
| **만류귀종** | 이론 90%, 실전 20%. 대회를 통해 이론 99%까지 도달. 결승전에서 첫 의식적 사용. |
| **기류감응** | 1단계(교전 중 감지) → 대회 중 ⭐ **2단계(관전만으로 파악)**으로 도약 |
| **약점** | 실전 경험 부족 (삼혼귀일경은 실전이지만 "진짜 죽는" 긴장감이 다름) |
| **대회 중 성장** | 기류감응 도약 + 만류귀종 이론 완성 + 실전 극한에서 화경의 경계를 엿봄. |
| **대회 후** | 사업 비중 증가로 수련 시간 감소. 그러나 실전 경험+기류감응으로 자연스럽게 화경 진입. |

> ⭐ **무림대회의 무학적 의미**: 위소운에게 대회는 "이기는 장소"가 아니라 **"눈을 뜨는 장소"**.
> 다양한 문파의 무공을 관전하며 "모든 무공의 근원이 하나"임을 체감하는 것이 진짜 수확.
