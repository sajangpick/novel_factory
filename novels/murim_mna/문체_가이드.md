## 📐 문체 3원칙

### 원칙 1: 간결한 문장 (Short & Sharp)

**기본 규칙**: 한 문장 = 25자 이내. 길어야 40자.

| 등급 | 문장 길이 | 사용 빈도 | 예시 |
|------|----------|----------|------|
| A급 | ~15자 | 30% | "검이 빛났다." / "침묵이 흘렀다." |
| B급 | ~25자 | 50% | "위소운은 천천히 칼을 들어올렸다." |
| C급 | ~40자 | 15% | "창고에서 타오르는 불길은 그의 재산뿐만 아니라 분노까지 태웠다." |
| D급 | 40자+ | 5% | 특수 상황 전용 (내면 독백, 감정 폭발, 서사 전환) |

```
❌ 나쁜 예: "위소운은 자신의 적이었던 유진산이 과거 자신을 배신했다는 사실을
   떠올리며 주먹을 불끈 쥐었고 그 순간 눈에서 살기가 번뜩였다." (60자)

✅ 좋은 예:
  유진산. 배신자.
  주먹이 떨렸다. 
  눈에서 살기가 번뜩였다. (총 3문장, 각 15자 이내)
```

---

### 원칙 2: 경영학적 해석 (Business Lens)

**기본 규칙**: 무협적 상황을 경영학 용어로 번역하는 이중 구조

| 상황 | 무협적 표현 | 경영학적 번역 (이준혁 시점) |
|------|-----------|------------------------|
| 문파 흡수 | "문파를 멸했다" | "적대적 인수합병(M&A) 완료" |
| 전투 승리 | "적을 베었다" | "경쟁사 도태. 시장 점유율 확보" |
| 수련 성공 | "무공이 올랐다" | "핵심 역량(Core Competency) 업그레이드" |
| 배신 | "칼을 등에 꽂았다" | "내부 횡령. 감사 실패" |
| 동맹 | "형제의 의를 맺었다" | "전략적 제휴(Strategic Alliance)" |
| 후퇴 | "한 발 물러섰다" | "전략적 손절. 포트폴리오 리밸런싱" |
| 패배 | "무릎을 꿇었다" | "파산. 법정관리 신청" |
| 승리 | "천하를 얻었다" | "시장 지배력 확보. IPO 완료" |

**사용 규칙**:
- 이준혁이 내면에서 경영학적 해석을 제공
- 모든 전투/사건에 대해 1회 이상 경영학 번역 삽입
- 단, 감동 장면에서는 경영 해석을 자제 (분위기 깨짐 방지)

```
✅ 좋은 예:
  검이 적의 목에 닿았다.
  이준혁이 속삭였다.
  '적대적 M&A 완료. 인수가: 목숨 1개.'

✅ 감동 장면 예외:
  천마가 사라졌다.
  위소운의 눈에서 눈물이 흘렀다.
  이준혁은 아무 말도 하지 않았다. (← 경영 해석 생략 = 감정 존중)
```

### 원칙 3: 무협적 장엄함 (Martial Grandeur)

**기본 규칙**: 전투/각성 장면에서는 스케일을 극대화

| 수준 | 묘사 범위 | 사용 시점 | 예시 |
|------|----------|----------|------|
| 소(小) | 신체 반응 | 일상 긴장 | "손이 떨렸다." |
| 중(中) | 주변 환경 | 전투 | "바닥에 균열이 갈렸다." |
| 대(大) | 자연 현상 | 각성/최종전 | "하늘이 둘로 갈라졌다." |
| 극(極) | 세계 반응 | 텐션10 전용 | "중원 전체가 한 순간 침묵했다." |

```
✅ 소: 위소운의 눈이 가늘어졌다.
✅ 중: 발을 내딛자 바닥이 갈라졌다. 주변의 나뭇잎이 일제히 흩날렸다.
✅ 대: 검기가 하늘을 갈랐다. 구름이 양쪽으로 찢어졌다.
✅ 극: 천마패도. 그 일격에 중원 전체가 진동했다. 수백 리 밖의 무림인들이 고개를 돌렸다.
```

## 🎭 캐릭터별 문체 (페르소나 가이드)

### 위소운 (주인공)

| 항목 | 규칙 |
|------|------|
| 말투 | 기본 존댓말. 친한 사이엔 반말. 적에겐 냉담한 존댓말 |
| 특징 | 말이 적다. 행동이 먼저. "..." 자주 사용 |
| 감정 표현 | 직접 말하지 않음 → 행동으로 보여줌 |
| 금기 | 장황한 연설 금지. 한 번에 3문장 이상 대사 금지 |

```
✅ 위소운 대사:
"...알겠습니다."
"좋아."
"그 사람은 내가 처리하겠습니다."

❌ 위소운이 절대 하지 않는 대사:
"여러분, 우리는 이 위기를 반드시 극복해야 합니다! 함께라면 할 수 있습니다! 
 우리의 미래는 우리가 만드는 것입니다!" (← 이건 위소운이 아님)
```

---

### 이준혁 (경영 인격)

| 항목 | 규칙 |
|------|------|
| 말투 | 항상 존댓말 (경어체). "~입니다" "~습니다" |
| 특징 | 데이터, 숫자, 확률을 반드시 포함. 감정을 숨기려 함 |
| 감정 표현 | 숫자로 번역 → "위험도 87%입니다" = "무섭다" |
| 금기 | 감탄사 금지 ("와!" "대박!" 등). 항상 냉정 유지 |
| 변화 | 후반부로 갈수록 감정이 살짝 드러남 (성장) |

```
✅ 이준혁 대사 (초반):
"생존 확률 23%입니다. 높지 않지만 0%는 아닙니다."
"적대적 M&A 완료. 인수가: 시가의 40%."
"위험합니다, 대표님. 감정은 판단을 흐립니다."

✅ 이준혁 대사 (후반, 감정 성장 반영):
"확률은 없습니다. 있는 것은... 의지뿐입니다." (← 처음으로 비논리적 발언)
"감사합니다, 대표님." (← 처음으로 감정 표현)
"행복했습니다." (← 마지막 대사. 데이터 없음 = 완전한 감정)
```

---

### 천마 (무력 인격)

| 항목 | 규칙 |
|------|------|
| 말투 | 하오체 + 고어체. "~하는구나" "~로다" "~이 몸은" |
| 특징 | 자신을 "이 몸"이라 지칭. 타인을 "벌레" "개미" 등으로 비하 |
| 감정 표현 | 분노 = 살기로 표현. 호감 = "쓸 만하군"으로 표현 |
| 금기 | 겸손 금지. "죄송합니다" "실례합니다" 절대 사용 불가 |
| 변화 | 후반부로 갈수록 위소운에 대한 애정이 살짝 드러남 |

```
✅ 천마 대사 (초반):
"벌레 같은 놈들이 판을 치는구나."
"이 몸이 죽는다고? 그런 재밌는 농담은 300년 만이구나."
"이 몸이 지는 싸움은 없다. 진 적이 없으니까."

✅ 천마 대사 (후반, 애정 성장 반영):
"잘 살아라, 위소운. 네가 내 생애 최고의 투자였다." (← 이준혁 어투 차용 = 우정)
"울지 마라. 이 몸의 마지막 모습이 눈물이면... 체면이 안 서니까." (← 속마음 드러남)
```

### 조칠 (충직한 부하)

| 항목 | 규칙 |
|------|------|
| 말투 | "두목님!" + 경상도 사투리 뉘앙스 |
| 특징 | 항상 열정적. 감탄사 많음. "우와!" "대단합니다!" |
| 감정 표현 | 숨기지 못함 → 얼굴에 다 드러남 |
| 금기 | 배신 불가. 의심 불가. 100% 충성 캐릭터 |

```
✅ 조칠 대사:
"두목님! 이거 진짜입니까?!"
"두목님, 제가 앞장서겠습니다!"
"두목님, 울지 마십시오. 놈들은 두목님을 위해 기꺼이..."
```

### 왕팔 (코믹→반전 캐릭터)

| 항목 | 규칙 |
|------|------|
| 말투 (전반) | 유쾌한 반말. "형~" "그거 아냐?" "히히" |
| 특징 (전반) | 주판 딸깍 효과음. 돈 이야기 = 눈 빛남 |
| 말투 (반전 후) | 냉정한 경어체. 완전히 다른 사람 |
| 금기 | 반전 전: 진지한 대사 금지. 반전 후: 코믹 대사 금지 |

```
✅ 왕팔 대사 (전반):
"형, 이거 얼마짜리인 줄 알아? 히히."
(주판 딸깍) "수익률 3%... 아, 아니 그냥 감이야, 형."
"돈 냄새가 나. 이 왕팔의 코는 안 속아!"

✅ 왕팔 대사 (반전 후, 219화~):
"미안하다, 형. 처음부터 말했어야 했는데."
(주판을 내려놓으며) "더 이상 연기할 필요 없다."
```

### 설화 (여주인공)

| 항목 | 규칙 |
|------|------|
| 말투 | 반말 (친한 사이), 존댓말 (공식 석상) |
| 특징 | 독립적. 의지가 강함. 의존적 대사 금지 |
| 감정 표현 | 직접적 ("좋아해"), 그러나 눈물은 잘 안 흘림 (강한 캐릭터) |
| 금기 | "오빠 살려줘" 타입 대사 절대 금지. 자기 전투는 자기가 함 |

```
✅ 설화 대사:
"나... 좋아해." (직접적)
"내가 할게. 네 뒤에 숨을 생각 없어." (독립적)
"바보야. 천하 따위 없어도 네 옆이면 됐는데." (솔직)

❌ 설화가 절대 하지 않는 대사:
"오빠, 무서워요..." (← 다른 소설임)
"제가 뭘 할 수 있겠어요..." (← 설화 캐릭터 위반)
```

## 📝 문체별 장면 템플릿

### 전투 장면 문체

```
[구조]
1. 짧은 문장 연속 (A급 15자 × 3~5문장)
2. 초식 이름 강조 (볼드체 또는 별도 줄)
3. 파괴 묘사 (중~대 스케일)
4. 관전자 리액션 (단역 배치)
5. 이준혁/천마 내면 해석

[예시]
  검이 빛났다.
  매화검법 제7초식.
  **낙화유수(落花流水).**
  
  바람이 갈라졌다.
  적의 검이 반으로 쪼개졌다.
  피가 튀었다.
  
  관중이 숨을 멈췄다.
  
  천마: "쓸 만하군."
  이준혁: '경쟁사 제거 완료. 시장 점유율 +8%.'
```

---

### 경영 장면 문체

```
[구조]
1. 상황 설명 (B급 25자)
2. 이준혁의 분석 (데이터 포함)
3. 위소운의 결단 (짧은 대사)
4. 결과 (숫자로 마무리)

[예시]
  위소운이 계약서를 펼쳤다.
  
  이준혁: '인수 대상 자산 평가 완료. 시가 12만냥.
           부채 비율 40%. 실질 가치 7만 2천냥.
           제안가: 5만냥. 할인율 30%.'
  
  위소운: "5만냥."
  상대: "그건 너무..."
  위소운: "4만 8천냥으로 내릴까요?"
  
  도장이 찍혔다.
  천보상단의 자산이 7만냥 늘었다.
```

---

### 감동 장면 문체

```
[구조]
1. 분위기 묘사 (자연, 빛, 바람)
2. 침묵 (행간으로 감정 전달)
3. 짧은 대사 (1~2문장)
4. 행동 묘사 (눈물, 떨림 등 → 직접 "슬펐다" 금지)

[예시]
  석양이 무덤을 비췄다.
  바람이 잡초꽃을 흔들었다.
  
  위소운은 무릎을 꿇었다.
  
  "스승님."
  
  그것뿐이었다.
  그것으로 충분했다.
  
  이준혁은 아무 말도 하지 않았다.
  천마도.
  
  바람만 불었다.
```

---

### 일상/개그 장면 문체

```
[구조]
1. 가벼운 묘사 (밝은 톤)
2. 캐릭터 간 대화 (빠른 주고받기)
3. 왕팔/조칠 중심 코믹 진행
4. 이준혁의 뜬금없는 분석 (코믹 요소)

[예시]
  객잔은 시끄러웠다.
  
  조칠이 닭다리를 뜯으며 외쳤다.
  "두목님! 이 닭다리 최고입니다!"
  
  왕팔이 눈을 빛냈다.
  "한 마리에 300문이야. 형, 2마리 시키면 550문에 깎아줄 수 있어."
  
  위소운: "...먹기나 해."
  
  이준혁: (내면) '식비 예산 초과. 그러나... 조직 사기 진작 효과를 고려하면 ROI 양호.'
  
  천마: (내면) '벌레들의 식사 시간이로군.'
```

## 📊 오감 묘사 규칙

### 시각 (가장 중요: 50%)
| 상황 | 묘사 대상 | 예시 |
|------|----------|------|
| 전투 | 검기, 파괴, 피, 눈빛 | "검기가 공기를 갈랐다. 푸른 빛이 번개처럼." |
| 감동 | 석양, 달빛, 눈물, 떨림 | "석양이 무덤을 비췄다." |
| 경영 | 서류, 도장, 금화, 건물 | "붉은 도장이 계약서에 찍혔다." |
| 일상 | 음식, 거리, 간판, 풍경 | "연기가 피어오르는 객잔 지붕." |

### 청각 (25%)
| 소리 유형 | 예시 |
|----------|------|
| 전투음 | "쩌억!" (충격음), "슈아아아" (검풍), "콰광!" (폭발) |
| 환경음 | "달깍" (주판), "찰칵" (도장), "딸깍" (잠금장치) |
| 인간음 | "껄껄" (호쾌한 웃음), "흡!" (놀람), "크읍" (고통) |
| 침묵 | "...정적이 흘렀다." (가장 강력한 소리) |

## 🎯 3인격 대사 샘플집 (상황별 20개)

> **목적**: 같은 상황에서 위소운/이준혁/천마가 **반드시 다르게** 반응한다는 것을 보여주는 실전 레퍼런스.
> 집필(Step 5) 시 이 표를 참조하면 인격이 섞이는 사고를 방지할 수 있다.
>
> **사용법**: 
> - 위소운 = **가슴(Heart)**. 감정과 공감이 먼저.
> - 이준혁 = **머리(Brain)**. 데이터와 시스템이 먼저.
> - 천마 = **주먹(Fist)**. 힘과 경험이 먼저.

---

**[상황 1] 이틀째 굶었다. 객잔에서 고기 굽는 냄새가 난다.**

| 인격 | 대사 (입 밖) | 내면 독백 |
|------|------------|----------|
| **위소운** | (침을 삼키며) "..." | *3년 동안 이런 냄새를 맡으면 다리에 힘이 풀렸다. 지금도 그렇다.* |
| **이준혁** | — (입 밖 대사 없음) | *저 객잔의 좌석 점유율은 80% 이상입니다. 번성하는 가게의 냄새는 다릅니다. 고기 기름의 연기량으로 볼 때 소고기... 아, 지금 그게 중요한 게 아니라 배가 고프다는 겁니다.* |
| **천마** | — (입 밖 대사 없음) | *...이 굴욕. 본좌가 고기 냄새에 침을 흘릴 뻔했다. 300년 만에 처음 느끼는 이 감각이... 이렇게 비참한 것이었느냐.* |

> **포인트**: 위소운은 '익숙한 고통', 이준혁은 '분석하다가 포기', 천마는 '치욕'. 같은 배고픔이 세 가지 색으로 갈린다.

**[상황 2] 드디어 밥을 먹었다. 싸구려 국수 한 그릇.**

| 인격 | 대사 (입 밖) | 내면 독백 |
|------|------------|----------|
| **위소운** | (한 젓가락 먹고 멈추며) "...맛있다." | *눈이 뜨거워졌다. 참아. 국수 먹으면서 우는 놈이 어딨어.* |

| **천마** | — | *...나쁘지 않군. (한 박자 뒤) ...이것이 네놈들이 매일 먹던 것이란 말이냐. 본좌는 300년 동안 이것조차 없었다.* |
