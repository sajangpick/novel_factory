## 🏢 개봉 세력 + 안씨표국 캐릭터 카드

`🔎 안씨표국, →천화표국, 천화련의, 뿌리, 우리, 안세진, 安世珍, 신사업, 총괄, 안씨표국주`

> 천화련 본사가 위치할 개봉의 기존 세력. 11화~부터 등장.
> **천화련 초기 스토리의 핵심 NPC들.** 이들과의 관계가 개봉 장악의 열쇠.

### 안씨표국(→천화표국) — 천화련의 뿌리 (우리 편)

> 안씨표국 → **천화표국 본사 겸 개봉지국**으로 리브랜딩.
> 무공 전수 정책 변경: 심법 5층 + 검법 12초식 전부 → 절정급 정예화.

### 천화련 핵심 인력 — 스카우트 인재

#### 손약령(孫若玲) — 재료 전문가 (천화 연구방 재료 총괄)

| 항목 | 내용 |

| **실력** | 약재·건어물·향신료의 산지·등급·계절을 **냄새와 손끝만으로 구별**. "이 표고는 절강 산지, 봄 채취, 2등급" |

| **스카우트** | 이준혁이 시장에서 재료를 사다가 → 손약령의 감별에 충격 → "이 분이 필요합니다" |
| **핵심 능력** | ① 산지별 품질 감별 ② 계절별 최적 조달 시기 ③ 약재+식재료 교차 지식 ④ 전국 약재상 인맥 |
| **역할** | 천화 연구방 재료 총괄. 양념 R&D에서 이준혁의 파트너. 해안 조달 루트 개척 |
| **대표 대사** | "이 다시마는 산동 래주산. 올 가을 채취. 짠맛이 덜하고 감칠맛이 강해요. 최상품." |
| **습관** | 재료를 살 때 반드시 냄새를 맡고 손으로 질감 확인. "눈으로만 사는 건 바보예요." |
| **등급** | D등급 → C등급 (사업 확장 후) |
| **역할 유형** | 세계관 확장자 + 정보 전달자 (재료 지식 = 세계관 깊이) |

| 3인격 반응 | |
|-----------|---|
| **이준혁** | "소믈리에가 와인을 감별하듯 재료를 감별합니다. 미슐랭에서도 이런 분은 귀합니다." |
| **천마** | "여자? ...흥, 실력이 있으면 상관없어." → (이후 감별 실력에 감탄하지만 인정 안 함) |
| **위소운** | "눈이 밝아요. 재료를 볼 때 눈빛이 달라져요." (→ 사교 서투른 위소운이 의외로 편하게 대화) |

> ⚠️ **스토리 기능**:
> - 소연화·당찬 외에 **사업 파트에서 활약하는 여성 캐릭터** = 다양성
> - 이준혁(셰프) × 손약령(재료) = **전문가 대 전문가 케미**
> - 시장에서 재료 감별하는 장면 = 세계관 깊이 + 재미
> - 전국 약재상 인맥 = 천화표국 물류 루트 개척에 직접 도움
> - 이준혁 "이런 맛을 내고 싶다" → 손약령 "이 재료를 쓰면 됩니다" = 자연스러운 파트너십

6. **무공비급이 불완전** — 비급이 온전하지 않아 익힐 수 없었음. 핵심 부분이 빠져 있었거나 손상되어 있었음.
7. **무인도에서 생을 마감** — 이미 나이가 들었고, 불완전한 비급을 익히지도 못한 채 한탄하다가 섬에서 삶을 마감.

**위소운과의 관계**:
- 어린 시절: 소백하(8세)가 **위소검**(9세)을 잘 챙김. 고아 출신 소검 사형에게 유일한 따뜻함
- 위소운 추락 후: 7년간 제삿날마다 절벽 앞에서 꽃을 놓음
- 곽진과의 혼인: 아버지(장문인)가 밀어붙이는 중. 소백하는 내키지 않지만 거부 이유를 말 못 함
- **위소운이 살아 돌아오면**: 감정 폭발 예정

**무림대회 관련**:
- 화산파 대표단의 일원으로 개봉에 옴
- 곽진 옆에 있음 = 위소운이 두 사람을 동시에 보게 되는 장면
- 소연화 vs 소백하 — 위소운을 둘러싼 미묘한 삼각관계 시작

**⭐ 핵심 떡밥 — 위소운이 소백하에게만 진실을 말한다**:
- 시점: 무림대회 이후, 둘이 단둘이 만나는 장면
- 위소운이 말하는 것: **"사매, 그때 나는 떨어진 게 아니야. 곽진 형이 밀었어."** — 이것만.
- 복잡한 설명 없음. 커버스토리도 아님. 그냥 사실 그대로.
- ⚠️ 위소운은 사매에게 거짓말을 할 수 없다 — 어릴 때 유일하게 따뜻했던 사람이니까
- **소백하 반응**: 7년간 "사고사"로 알고 있었음 → 곽진이 "살인 미수"였다는 충격 → 7년간 곽진 옆에서 웃었던 자신에 대한 분노+자괴감 → "내가...... 그 사람 옆에서......"
- **파장**: 소백하가 이 사실을 어떻게 처리하느냐 = 화산파 내부 폭풍의 시작
  - 아버지(장문인)에게 말하면 → 화산파 분열
  - 혼자 삼키면 → 소백하의 내면 갈등
  - 곽진에게 직접 추궁하면 → 곽진의 본성 폭발
- **3인격 반응**:
  - 이준혁: "위소운 님...... 이건 제가 뭐라 할 수 있는 게 아닙니다." (이번만은 경영이 아님)
  - 천마: "......" (침묵. 드물게 위소운의 선택을 존중)
