#### 춘향루 (春香樓) ⭐⭐⭐⭐⭐

**등급**: 최고급 객잔  
**위치**: 소주 시내 중심, 운하 옆  
**규모**: 5층 건물  
**특징**: 
- 강남 제일 객잔
- 연못과 정원 있음
- 1층 식당 + 2층 다실 + 3~4층 객실 + 5층 연회장

**주인**: 
- 이름: 진춘화 (陳春花)
- 성별: 여성, 40대
- 배경: 강남 상단 출신, 부유함
- 특징: 경영 수완 뛰어남

**객실 가격**:
- 상등방: 500문/박 (창문이 운하 향함)
- 중등방: 300문/박
- 하등방: 150문/박
- 귀빈실: 1000문/박 (5층, 전망 최고)

**대표 메뉴**:
| 메뉴 | 가격 | 특징 |
|------|------|------|
| 소룡포 | 50문/5개 | 춘향루 시그니처, 국물 가득 |
| 게찜 | 800문 | 대게 1마리, 가을 한정 |
| 송서계 | 400문 | 다람쥐 모양 생선 튀김 |
| 동파육 | 250문 | 소주식, 더 달콤함 |
| 벽라춘 (차) | 100문/호 | 소주 특산, 최고급 녹차 |
| 여아홍 (술) | 200문/병 | 15년 숙성, 소주 명주 |

**특색 서비스**:
- 뱃놀이 가능 (운하에서)
- 기생 공연 (5층, 저녁)
- 다실 (2층, 조용한 상담)

**이용 고객층**:
- 부유한 상인
- 정파 무인 (9대문파 제자)
- 귀족

**시나리오 예시**:
```

1층 대청은 화려했다.
홍목으로 만든 테이블 20개.
천장에는 붉은 등롱이 수십 개.

창밖으로 운하가 보였다.
배가 천천히 지나간다.

"손님, 어서 오십시오!"

40대 여주인 진춘화가 직접 나왔다.
비단 옷에 화려한 비녀.
부유함이 몸에 배어 있었다.

"상등방 하나. 운하가 보이는 방으로."

"예, 손님. 500문입니다."

- 이준혁: "1박 5만원. 합리적 가격입니다."
```
