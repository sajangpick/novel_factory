## 📍 1~25화: 서구진 (西口鎭) 세력도

### ⚫ 흑호문 (黑虎門) — 서구진 최대 세력 → 위소운에게 인수됨

```
문주: 마흑호 (절정 초입) — 25화 사망
├─ 좌호법: 오상철 "철갑귀" (일류 상위) — 25화 사망
├─ 우호법: 사독 "독안사" (일류 중위) — 24화 도주
├─ 순라당주: 석파 "쇄골두" (이류 상위) — 25화 항복
├─ 포구당주: 곽대봉 "흑봉" (이류 중위) — 25화 항복
├─ 객사당주: 호연수 "화면호" (이류 하위) — 25화 항복
├─ 이류급 문도:
│   ├─ 양삼 (돌격대장, 이류 중위) — 25화 사망
│   ├─ 묵이 (야간순찰 부대장, 이류 하위) — 23화 항복
│   ├─ 한용 (포구 하역반장, 이류 하위) — 25화 항복
│   └─ 두만 (도박장 관리인, 삼류) — 25화 즉시 항복
└─ 일반 문도: 약 400명 (삼류~이류)
```

**규모**: 문도 약 400명 | **자산**: 미상 (포구 하역권, 상점 보호비, 도박장)
**결말**: 25화 위소운에 의해 장악. 마흑호 사망, 핵심 간부 일부 사망/도주/항복.

### 🔵 위소운 세력 (1~25화 종료 시점)

```
지도자: 위소운 (절정 — 심상전장 1000일 + 실전 6개월)
├─ 조칠 (일류) — 충성 근위
├─ 왕팔 (삼류) — 재무
├─ 강무혁 (삼류 상) — 전투 선봉
├─ 석파 "쇄골두" (이류 상위) — 항복 합류
├─ 곽대봉 "흑봉" (이류 중위) — 항복 합류
├─ 호연수 "화면호" (이류 하위) — 항복 합류
├─ 묵이 (이류 하위) — 항복 합류 (야간 정보)
├─ 한용 (이류 하위) — 항복 합류 (포구 하역)
├─ 두만 (삼류) — 항복 합류 (도박장/정보)
└─ 흑호문 잔여 문도 — 약 200~250명 (사상자/이탈 제외)
```

**26화 이후**: 흑호문을 기반으로 **천화련(天火聯)** 재편 예정
