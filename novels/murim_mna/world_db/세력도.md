## 📍 1~25화: 서구진 (西口鎭) 세력도

`🔎 흑호문, 黑虎門, 서구진, 최대, 세력, 위소운, 25화, 이후, 일반`

### ⚫ 흑호문 (黑虎門) — 서구진 최대 세력

| 항목 | 내용 |
|------|------|
| **규모** | 문도 약 400명 (삼류~이류) |
| **문주** | 마흑호 (절정 초입, 42세) — 사천 무인 가문 서자 출신, 22세에 서구진 정착 후 20년간 군림 |
| **자산** | 포구 하역권, 상점 보호비, 도박장 3곳, 객잔 2곳 |
| **수입원** | 보호비(월 200냥), 하역비(월 150냥), 도박장(월 100냥), 객잔(월 50냥) ≈ **월 500냥** |
| **관아 유착** | 서구진 관아에 매달 보호비 상납 → 관아는 흑호문 활동 묵인 |
| **핵심 간부** | 오상철(좌호법, 일류 상위), 사독(우호법, 일류 중위), 석파(순라당주, 이류 상위), 곽대봉(포구당주, 이류 중위), 호연수(객사당주, 이류 하위) |
| **결말** | 25화 위소운에게 장악됨. 마흑호+오상철+양삼 사망, 사독 도주, 나머지 항복 |

### 🔵 위소운 세력 — 25화 이후

| 항목 | 내용 |
|------|------|
| **규모** | 흑호문 잔여 200~250명 + 핵심 측근 3명 (조칠, 왕팔, 강무혁) |
| **지도자** | 위소운 (절정급, 16세) |
| **자산** | 흑호문 자산 계승 (단, "파산한 문파" 수준 — 부채, 보수 비용, 이탈자 처리 필요) |
| **당면 과제** | ① 흑호문 잔당 통합 ② 서구진 민심 수습 ③ 수익 구조 개편 ④ 사독(도주) 추적 ⑤ 외부 세력 경계 |
| **26화~ 목표** | 흑호문 → **천화련(天火聯)** 재편. 이준혁의 경영 전략 + 천마의 무력 기반 |

### 🟤 서구진 일반 세력

| 세력 | 규모 | 특징 | 관계 |
|------|------|------|------|
| **서구진 관아** | 관리 10명, 포졸 30명 | 흑호문과 유착, 실질적 치안 능력 부족 | 중립 → 위소운 체제에도 유착 유지 예상 |
| **포구 상인들** | 상점 40여 개 | 흑호문에 보호비 납부, 억압당하는 중 | 위소운 장악 후 기대+경계 |
| **어부/짐꾼** | 약 200명 | 서구진 하층민, 흑호문 착취 대상 | 위소운의 과거(짐꾼) = 공감 기반 |

## 📍 7~10화: 개봉(開封) — 무림대회 + 천화련 기반

`🔎 개봉, 세력, 지도, 확정, 천화련, 진입, 전략, 이준혁의, 3단, 잠금`

> 개봉은 무림의 수도. 천화련 본사 소재지.
> 최상위 세력(무림맹, 황실)은 감시자이고, 실질적 경쟁은 중간 세력에서 일어난다.

### 개봉 세력 지도 (확정)

```
[최상위] 무림맹(武林盟) — 무림의 정부. 개봉성 북쪽 무림맹 본부.
[최상위] 황실/금의위/육선문 — 정치적 영향력. 226화+ 본격.
[상위]   천하제일상단 — 개봉 상권 40%. 최종 M&A 타겟(276~300화).
─────────────────────────────────────────
[중간]   개봉상회(開封商會) — 상인 길드. 회원 300+. 회주 진만복(62세).
[중간]   진무관(鎭武館) — 최대 무관. 제자 200~300명. 관주 풍만장(52세, 절정).
[중간]   만리표국(萬里鏢局) — 최대 표국. 표사 500+. 총표두 곽대용(48세, 절정).
─────────────────────────────────────────
[하위→성장] 안씨표국→**천화표국 본사 겸 개봉지국** — ⭐ 우리 편. 300년 토박이 → 천화련 본사로 확장. 심법5층+검법12초식 전수 후 절정급 정예. 지국장 안세진(45세).
[하위]   기타 소규모 문파/무관/상점
```

### 천화련 개봉 진입 전략 (이준혁의 3단 잠금 해제)

| 단계 | 대상 | 방법 | 담당 | 결과 |
|------|------|------|------|------|
| **1. 합법성** | 개봉상회 | 상회 가입 + 규칙 준수 | 이준혁 | 합법적 상업 활동 |
| **2. 무력 인정** | 진무관 | 위소운 비무 → 풍만장 압도 | 위소운 | 개봉 무림 인정 |
| **3. 물류 확보** | 만리표국 | 분업 제안 (장거리=만리, 단거리+식품=천화) | 이준혁 | 전국 배송 파트너 |

### 안씨표국 → 천화표국 본사 겸 개봉지국 (리브랜딩 후)

| 항목 | 변경 전 (안씨표국) | 변경 후 (천화표국 개봉지국) |
|------|-----------------|----------------------|
| 간판 | 안씨표국 | 천화표국 개봉지국 |
| 무공 | 심법4층 + 검법6초식 (일류) | 심법5층 + 검법12초식 (절정) |
| 규모 | 30~50명 | 100~200명 |
| 사업 | 소규모 호위 | 천화루/천화상회 물류 + 호위 + 정보 |

**내부 세력:**

| 유형 | 비율 | 대표 | 처리 |
|------|------|------|------|
| **환영파** | ~40% | 안세진(지국장) | 핵심 인력 등용 |
| **관망파** | ~40% | 유경(부표두) 등 | 성과로 증명 → 합류 |
| **불만파** | ~20% | 안효림(장남) 등 | 🗡️ 무력 제압 → 💼 천화루 지점장 파견 |

### 천화련 3사업부 개봉 배치

```
🗡️ 천화표국 본사 = 옛 안씨표국 건물
🍜 천화루 본점 = 안씨표국 근처 객잔 인수/신축
   └─ 1층: 식당+주점 / 2~3층: 객실(숙박) ← 한 건물 통합
🏪 천화상회 본점 = 천화루 옆 매장 (증류주·비누·생활용품)
🏭 양조장 = 안씨표국 근처 별채 (안씨세가 가족만 출입)
```
