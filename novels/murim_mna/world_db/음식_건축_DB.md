#### ⭐ 이준혁의 "천연 감칠맛 과학" — 3종 겹치기 원칙

> **미슐랭에서 배운 핵심: 감칠맛은 "겹치면 폭발한다."**
> 글루탐산(버섯·다시마) + 이노신산(멸치·건새우) + 구아닐산(표고) = **감칠맛 10배 상승**
> 1가지=10, 2가지=50, 3가지=**100**. 이것이 천연 MSG의 비밀.

```
[이준혁의 감칠맛 공식]

① 글루탐산 (식물성)     ② 이노신산 (동물성)     ③ 구아닐산 (균류)
   다시마, 간장, 두시       멸치, 건새우, 뼈육수      건표고버섯
        \                    |                    /
         \                   |                   /
          -----→  3종 합체  ←-----
                     ↓
            ⭐ 감칠맛 10배 상승
            = "왜 이렇게 맛있지?"
```

> 이준혁: "이 시대에 MSG가 없는 게 문제가 아닙니다. MSG의 **원료**는 다 있어요.
> 표고버섯, 다시마, 멸치... 이걸 말려서 가루로 만들면 **천연 MSG**입니다.
> 미슐랭 3스타에서도 MSG 안 씁니다. 천연 재료로 감칠맛을 만드는 게 진짜 실력이에요."
> 천마: "그래서 이 가루를 넣으면?"
> 이준혁: "한 꼬집이면 됩니다. 음식이 '살아납니다.'"

> - 이준혁: "라면 스프처럼... 아, 그러니까, 가루만 넣으면 맛있는 음식이 되는 겁니다."

#### 재료 조달 핵심 — "해산물 루트"

> ⚠️ **최대 과제**: 멸치·다시마·건새우·건굴 = 전부 **해안에서 와야 함**.
> 개봉은 내륙 도시. 해안에서 재료를 조달하려면 **물류**가 필수.
> → 천화표국의 물류 역할이 여기서 빛남.
> → 강소·산동 해안 지사 설립 시 원재료 조달도 겸함.

| 재료군 | 조달처 | 운반 | 비고 |
|--------|--------|------|------|
| 버섯류 | 개봉 인근 산지 + 절강 | 육로 (가까움) | ✅ 쉬움 |
| 해산물류 | 산동·강소 해안 | **천화표국 호송** | ⚠️ 핵심 물류 |
| 곡물·콩류 | 개봉 곡물시장 | 현지 조달 | ✅ 쉬움 |
| 향신료 | 사천(화자오), 서역(후추) | 장거리 교역 | ⚠️ 비용 주의 |

> 이준혁: "해산물 재료가 핵심인데, 해안에서 개봉까지 운반이 문제입니다.
> 천화표국이 호위하면서 동시에 원재료를 실어 오는 겁니다. **일석이조.**"
> 천마: "표국이 짐도 지키고 재료도 가져오고?"
> 이준혁: "물류 효율화입니다. 빈 수레로 돌아오는 건 낭비니까요."

| **결과** | 입자 크기가 맷돌의 1/10 이하. 혀에 올리면 녹는 수준의 초미세 분말. 이 곱기가 감칠맛의 폭발적 방출을 만든다. |
| **무공 기반** | 중원제일인의 장력(掌力) 응용 — 본래 적의 내부 장기를 진동시키는 기술. 위소운이 "파괴"가 아닌 "분해"로 역전환. |
| **소모** | 내공 소모 중간. 1회 가공량 = 양손에 쥘 수 있는 분량(약 1근). 하루 최대 10~15근 가공 가능. |
| **천마 반응** | "경맥을 진동시켜 적의 오장을 터뜨리는 공력을... 표고버섯에? ......나를 죽일 셈이야." |
| **이준혁 반응** | "볼밀(Ball Mill)이 하는 일을 내공으로 하는 겁니다. 아니, **입자 크기 제어**가 가능하니까 기계보다 낫습니다." |
| **한계** | ⚠️ 내공이 부족하면 균일하지 않음. 위소운의 내공이 성장할수록 분말 품질도 향상 = **성장 서사와 직결** |

**장면 예시 — 진기분쇄 첫 시도:**
> 위소운이 건표고 한 움큼을 양손에 쥐었다. 이준혁의 목소리가 머릿속에 울렸다.
> '아주 천천히. 진기를 손바닥에서 손가락 끝으로, 다시 안으로. 진동이에요.'
> 내공을 운용했다. 가느다란 떨림이 손바닥에서 시작되어 건표고의 결을 따라 스며들었다. 파직, 하는 소리가 손 안에서 났다. 건표고의 섬유질이 안에서부터 허물어지는 감촉. 마치 — 모래성이 저절로 무너지듯.
> 손을 펴자, 갈색 가루가 공기 중에 피어올랐다.
> '......이건.' 천마가 말을 잃었다. 맷돌로 빻은 것과는 차원이 달랐다. 밀가루보다 고운 가루. 숨을 들이쉬자 코끝으로 표고의 향이 올라왔다. 향까지 살아 있었다.
> '미슐랭에서 프랑스산 트뤼플을 가루 낼 때 이런 장비가 있었습니다. 300만 원짜리. 위소운 님 손이 그 기계입니다.'
> 천마가 한참 침묵하더니, 낮게 말했다.
> '......다시 해봐.'

**맷돌 vs 진기분쇄 비교:**

| 비교 | 맷돌 분쇄 | 진기분쇄 |

| **재현 가능성** | ✅ 누구나 가능 | ❌ **내공 없으면 불가능** |

#### ⭐ 경쟁력 분석 — 왜 절대 흉내 못 내는가

```
[기존 기술 보호 — 2중 보안]
① 양념 배합 비율 = 비밀 (이준혁+손약령만 앎)
② 재료 산지·품질 = 전문 네트워크 (손약령 인맥)

[무공 가공 추가 — 3중 보안 → 절대 보안]
③ 가공 품질 = 내공 없으면 재현 불가능

3가지가 합쳐야 천화선분:
  배합 비율 (뭘 섞는지) + 재료 품질 (좋은 재료) + 가공 기술 (초미세 분말)
  ❌ 하나라도 빠지면 → "비슷하긴 한데... 그 맛은 아닌데."

경쟁자 시나리오:
├── 배합 비율을 훔침 → 가공 품질이 안 따라옴 → 실패
├── 재료를 같은 거 삼 → 맷돌 분쇄 = 거친 가루 → 맛이 다름
├── 요리사를 빼감 → 양념이 없음 → 실패
└── 셋 다 빼가려면 → 위소운급 고수 + 이준혁급 지식 + 손약령급 전문가
                    → 이런 조합이 세상에 또 있을 리 없음 = 절대 모방 불가
```

> 이준혁: "증류주는 기술을 숨기면 되고, 양념은 배합을 숨기면 됩니다.
> 하지만 천화선분은... **기계가 아니라 사람이 만듭니다.**
> 같은 기계를 사면 되는 현대와 달리, 여기서는 **위소운 님 자체가 기계입니다.**
> 위소운 님을 복제할 수 없으니, 천화선분도 복제할 수 없습니다."

> 천마: "또 그 이상한 이름."
> 위소운: (미소) "좋은 거래 아닌가요?"

#### ⭐ 합원진(合源陣) — "7인의 힘을 하나로"

> **이름**: 합원진(合源陣) — "근원을 합치다". 청원심법의 "청원(淸源)"에서 따온 이름.
> **원리**: 같은 심법(청원심법)을 익힌 수련자들이 특정 대형(陣形)으로 서서,
> 진기를 중앙의 실행자에게 모아준다. 실행자 1명이 합쳐진 내공으로 가공을 수행.
>
> 무협에서 흔한 "합기(合氣)" 기술이지만, **같은 심법을 쓰는 사람들끼리만 가능**하다.
> 다른 심법 수련자의 기는 충돌한다. 청원심법 수련자만 할 수 있다 = **안씨세가 독점**.

```
[합원진 배치도 — 7인 진형]

         ② 공급자
           ○
    ⑥ ○         ○ ③
         \     /
   ⑤ ○ ── ★ ── ○ ④     ★ = 중앙 실행자 (가공 담당)
         /     \         ○ = 내공 공급자 6명
    ⑦ ○         (빈자리 가능)   
                          
   진기 흐름: ○ → ★ (6명이 중앙으로 내공 집중)
   실행자: 합쳐진 내공으로 진기분쇄·내공건조 등 수행
```

| 항목 | 내용 |
|------|------|
| **최적 인원** | **7명** (실행자 1 + 공급자 6). 최소 5명, 최대 10명 |
| **진형** | 육각형 배치. 중앙에 실행자. 청원심법의 기 순환 경로를 따른 배치 |
| **필수 조건** | ⚠️ 전원 청원심법 수련자. 다른 심법은 기가 충돌하여 불가능 |
| **실행자 조건** | 심법 최소 3층 이상 + 무공 가공 기법 훈련 완료 (위소운이 직접 전수) |
| **공급자 조건** | 심법 최소 2층 이상. 내공만 안정적으로 보내면 됨 |
| **합산 내공** | 2~3층 × 7명 ≈ 초절정(6층) 1명의 **70~80%** |
| **가공 품질** | 위소운 단독 = ⭐⭐⭐ (최상) / 합원진 = ⭐⭐ (상) / 맷돌 = ☆ (하) |
| **내공 소모** | 1인당 소모는 적음. 7명이 나눠 부담 → 하루 2~3회 가공 가능 |
| **훈련 기간** | 진형 호흡 맞추기: 약 1~2개월. 위소운이 직접 지도 |
| **창안자** | 천마(진법 원리 제공) + 이준혁(생산 시스템 설계) + 위소운(실행 기법 전수) |

**3인격 반응:**

#### 합원진의 스토리 가치

**① 안씨세가의 존재 가치 상승**

**② 천마의 유산이 살아남**

- 코미디 + 감동 + 떡밥 회수의 삼중 효과

**③ 합공 훈련 = 팀워크 형성**

- 합원진 성공 순간 = "우리가 하나로 움직였다!" = 단결의 명장면
- 위소운이 가르치고 → 안씨세가가 성장 = 주인과 가신의 유대 강화

**④ 확장 가능성**

- 천화선분의 지역 생산 가능 = 물류 비용 절감 + 신선도 유지

**⑤ 전투 전용 합원진 — "양념에서 전투로"**

**⑤ 무림대회 연결**

### ⭐ 이준혁의 매운맛 공식 — "고추 없이 매운맛 만들기"

#### 소설 활용 포인트

**1. 감성 드라마**
- 이준혁의 그리움 → 인간적인 면 노출
- 구내식당 = 가족 → 조직이 아니라 "식구"가 됨
- "같이 밥 먹는 사이" = 무협에서 가장 강한 유대

**2. 코미디**
- 천마 검으로 삼겹살 3mm 썰기
- 된장 vs 두반장 논쟁
- 왕팔 줄서기 소동
- "형님 오늘 메뉴 뭐예요?!" (매일 물어보는 부하들)

**3. 무협 활용**
- 60도 천화주 화공(火攻) 전술
- 70도 주정 → 상처 소독 → 부하 사망률 급감
- 접대 음식으로 동맹 확보 → "밥 한 끼"의 힘

**4. 경영 드라마 (51화 이후)**
- 구내식당 → 독고장원 전환
- 요리 배운 부하 → 주방장 발탁 (인재 양성)
- 기존 주류/객잔 세력 반발 → 상단 전쟁
- 레시피 보호 → 천마의 무력이 곧 지식재산권

**술잔 단위**:
- 1병 (壺): 약 500ml, 2~3명 분량
- 1잔 (盞): 약 50ml

**표현**:
- "점소이! 죽엽청 한 병 가져오게!"
- "이 술은 10년 묵은 것이오. 향이 그윽하지."

## 🍽️ 객잔 메뉴판 (가격표)

### 흑호객잔 메뉴 (서구진)

#### 술
- 죽엽청: 100문/병
- 소흥주: 80문/병
- 화조주: 120문/병

#### 요리
- 우육면: 80문
- 동파육: 200문
- 편육: 100문
- 교자: 30문/10개
- 포자: 20문/개

#### 밥/면
- 백미: 10문
- 죽: 5문
- 면: 30문

#### 차
- 용정차: 50문/호
- 보이차: 40문/호

#### 객실
- 상등방: 200문/박
- 중등방: 100문/박
- 하등방: 50문/박

**세트 메뉴**:
- **1인 기본**: 밥 + 교자 + 차 = 70문
- **2인 표준**: 술 1병 + 우육면 2그릇 + 편육 = 360문
- **4인 호화**: 술 2병 + 동파육 + 편육 + 교자 = 800문

## 💰 화폐 가치 체계

### 화폐 단위

| 단위 | 환율 | 현대 가치 |
|------|------|----------|
| 1냥 (兩) | = 1000문 | 약 10만원 |
| 1관 (貫) | = 1000문 = 1냥 | 10만원 |
| 1문 (文) | - | 약 100원 |

**예시**:
- 객실 1박 (200문) = 2만원
- 술 1병 (100문) = 1만원
- 식사 1끼 (50문) = 5천원
- 월급 10냥 = 100만원

## 🎭 객잔 전형적인 장면 연출

### 입장 장면

```
"점소이! 여기 있소!"

위소운이 객잔 문을 밀고 들어서자, 
1층 대청은 이미 술손님들로 가득했다.
기름기가 흥건한 나무 테이블 15개.
술 냄새, 땀 냉새, 고기 굽는 냄새가 뒤섞였다.

"손님, 어서 오십시오!"

앞치마를 두른 점소이가 달려왔다.
40대 중반, 얼굴에 기름때가 낀 사내.

"객실 하나 주시오. 상등방으로."

"예, 손님. 상등방은 200문입니다."

위소운이 은자 1냥을 꺼내 던졌다.

"술과 안주도 준비하시오. 
죽엽청 한 병, 우육면 한 그릇, 편육 한 접시."

"알겠습니다! 잠시만 기다리십시오!"

점소이가 주방으로 뛰어갔다.
곧 따끈한 술 향이 퍼져나왔다.
```

### 식사 장면

```
탁자 위에 요리가 올라왔다.

죽엽청 한 병. 푸른 병에 담긴 투명한 술.
코를 찌르는 독한 향. 45도는 될 것이다.

우육면 한 그릇. 붉은 국물에 소고기 편육이 
두둥실 떠 있다. 면발은 굵고 쫄깃해 보였다.

편육 한 접시. 얇게 썬 돼지고기가 
파와 마늘 소스에 절여져 있다.

위소운이 술잔을 들어 입에 댔다.

쏴아아.

독한 술이 목구멍을 타고 내려갔다.
속이 화끈거렸다. 5년 만에 마시는 술.
눈물이 날 것 같았다.

"...맛있군."

천마의 목소리가 들렸다.

- 천마: "흥, 300년 전에 마시던 술보다 못하지만, 
        그래도 먹을 만하구나."

- 이준혁: "알코올 45도. 간에 무리가 갑니다. 
          적정량만 섭취하세요."

위소운은 쓴웃음을 지으며 면을 젓가락으로 집었다.
```
