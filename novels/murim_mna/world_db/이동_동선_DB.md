# 무림 M&A - 이동 동선 데이터베이스 (화산귀환 스타일)

> ⚠️ **주의**: 이 파일의 "N화" 표기는 **초기 기획 시점의 대략적 배치**이며, 실제 집필 시 화수가 달라질 수 있다.
> 정확한 화수 배정은 `master_story_bible.md`를 기준으로 한다.
> 이 파일의 핵심 가치는 **화수가 아니라 거리·소요시간·경유지 데이터**이다.

## 🗺️ 중원 주요 지역 거리 맵

> **리얼리티의 핵심**: 모든 이동에는 시간과 거리가 있다!

---

## 📏 거리 단위 및 속도

### 기본 단위
- **1리 (里)**: 약 500m
- **10리**: 약 5km
- **100리**: 약 50km

### 이동 속도
| 방법 | 속도 | 하루 이동 거리 | 특징 |
|------|------|----------------|------|
| 도보 | 10리/시간 | 80~100리 | 평민, 일반인 |
| 말 (보통) | 20리/시간 | 150~200리 | 상인, 무인 |
| 말 (전력질주) | 40리/시간 | 300~400리 | 긴급 상황 (1일만 가능) |
| 경공 (일류급) | 30리/시간 | 200~250리 | 무인, 지치면 느려짐 |
| 경공 (화경급) | 50리/시간 | 400~500리 | 고수, 장거리 가능 |

### 휴식 규칙
- **아침 출발**: 6시 출발 → 12시 점심 (6시간)
- **점심 휴식**: 12~14시 (2시간)
- **오후 이동**: 14~20시 (6시간)
- **저녁 숙박**: 20시 도착 → 객잔 투숙

**총 이동 시간**: 12시간/일

## 📍 주요 지역 간 거리 맵

### ⭐ 서구진 → 개봉 (도보 7일 여정) — 2~8화 무림대회 루트

> **화수 대응 (실제 집필 기준)**: 1화=서구진 천향루(이동 없음), 2화=1일차(서구진→패산), 3화=2일차(패산→의흥), 4화=3~4일차(의흥→진강→양주), 5화=6일차(초양→진류), 6화=7일차(개봉 입성)
> ⚠️ 아래 상세 테이블은 **원래 계획**이며, 실제 집필과 일부 다를 수 있음. 실제 화수 기준은 `master_story_bible.md` 참조.

> **구간**: 서구진(태호 서쪽) → 개봉(무림맹 본부)
> **총 거리**: 약 700리 (350km)
> **인원**: 위소운, 당찬, 소연화, 장위, 마현 (5명, 도보)
> **시기**: 봄, 날씨 맑음~흐림
> **경로**: 서구진 → 패산 → 의흥 → 진강 → (장강 도하) → 양주 → 초양 → 개봉

#### 0일차(1화): 서구진 천향루 — 귀환

> **1화는 이동 없음.** 서구진 천향루(天香樓)에서 하루 동안 벌어지는 이야기.

**1화 사건 요약**:
- 점심: 위소운이 천향루 2층 창가석에 착석. 3인격 소개 (이준혁·천마)
- 점심: 3년 만의 식사. 돼지고기 볶음(마늘 감동), 소고기 탕, 만두, 볶음면, 술. 이준혁 "마늘이다" (목소리 떨림)
- 점심: 흑호문 졸개 4명 등장 → "야, 이 새끼" → 위소운 젓가락 전투 3초 제압 → "젓가락 하나 더 주세요"
- 점심: 졸개 선두가 "서구진의 바보?"를 떠올림 → 흑호문 두목에게 보고 예고
- 오후: 소연화 합석 제안 → 4인(당찬·소연화·장위·마현) 소개 → 위소운 "14년 섬에서 수련" 커버스토리
- 오후: 무림대회 정보 획득 → 소연화 "같이 가시겠습니까?" → 위소운 "좋습니다"
- 석양: 서구진 거리 엔딩. "돌아왔구나 싶어서요."

**등장인물**: 위소운, 이준혁(내면), 천마(내면), 당찬, 소연화, 장위, 마현, 흑호문 졸개 4명, 점소이
**2층 창가석**: 당찬·소연화·장위·마현이 먼저 앉아 있었고, 위소운은 반대편 창가석에 혼자 앉음 → 합석

#### 1일차: 서구진 → 패산객잔 (覇山客棧) — 약 100리

**구간**: 서구진 남문 → 관도 → 패산 산길 → 패산 기슭 삼거리
**지형**: 전반 평지(관도) → 후반 소나무 산길(오르막)
**소요**: 12시간 (아침 출발 → 해질 무렵 도착)

**패산(覇山)**:
- 서구진 남쪽의 낮은 산. 해발 300m 정도. 소나무 숲이 빽빽함
- 관도에서 산길로 빠지는 구간은 도적 출몰 주의 지역
- 산길을 넘으면 관도와 다시 만나는 삼거리에 객잔 1곳

**패산객잔 (覇山客棧)** ⭐:
- **등급**: 하급 객잔 (관도변 허름한 1층 건물)
- **주인**: 손파(孫婆), 60대 할머니. 마른 몸, 매서운 눈. 남편 사후 혼자 운영
- **객실**: 방 3개 (남자방 2, 여자방 1). 짚 깔린 바닥+얇은 이불
- **가격**: 숙박 20문/인, 식사 30문/인
- **메뉴**: 볶음밥, 두부탕, 만두 (맛은 보통. 이준혁 평: "기본이 안 돼 있어요")
- **특징**: 문 앞 화로에서 고구마 구움. 간판 기울어짐. 거미줄. 정보통(관도 소문 빠름)
- **분위기**: 허름하지만 따뜻함. 등불 오렌지색. 좁은 탁자에 옹기종기

**2화(1일차) 사건**: 
- 오전: 서구진 남문 출발, 5명 도보. 소연화-위소운 대화 ("외롭지 않으셨어요?")
- 오후: 패산 산길에서 도적 곽철 일당(10여 명) 조우 → 위소운 4초 제압(검, 한 명도 안 죽임) → 곽철 "기억하지" 퇴장
- 오후: 진삼덕·진소아 부녀 합류 (포목상인, 개봉 가는 길. 7명으로 증가)
- 저녁: 패산객잔 도착. 손파 할머니 "밥값 먼저!" → 7명 식사
- 밤: **곽진(霍進) 소문 최초 투하** — 손파 할머니: "화산파에서 천재가 나온다나 봐. 곽진? 스물넷에 일류 고수래." → 위소운 손가락 하얀 반응 → 소연화 "비밀이 많으시네요"
- 밤: 위소운·이준혁·천마 3인격 밤 대화. "곽진...... 어떻게 할 거야?" → "아직 모르겠어요." → 출전 결심 온도: 🔥 10%

#### 2일차: 패산객잔 → 의흥현 (宜興縣) — 약 100리

**구간**: 패산 기슭 → 관도 남하 → 의흥현 성문
**지형**: 완만한 구릉지대 → 차밭 → 도자기 가마 연기 보임 → 현성(縣城)

**의흥현(宜興縣)**:
- 태호 남서쪽의 현(縣). 인구 약 3만. 성벽은 낮지만 성문 있음
- **특산**: 자사호(紫砂壺, 보라색 찻주전자). 차(茶). 도자기. 대나무
- **분위기**: 차향이 거리에 떠돌음. 도자기 가마에서 올라오는 연기가 마을 전체를 안개처럼 감쌈. 장인(匠人)이 많아 손재주 좋은 사람이 존경받는 동네
- **시장**: 차시장이 유명. 벽라춘(碧螺春) 차잎 거래. 도자기 노점
- **주요 거리**: 도자거리(窯街) — 양쪽에 가마와 도자기 가게가 줄지어 있음

**의흥 객잔 — 자사루(紫砂樓)** ⭐⭐⭐:
- **등급**: 중급 객잔. 2층 건물. 1층 식당+다실, 2층 객실
- **주인**: 도장인 출신 50대 남자. 객잔 곳곳에 자사호 장식
- **객실**: 상등방 100문, 중등방 60문, 하등방 30문
- **대표 메뉴**: 의흥 죽순볶음(50문), 태호 은어구이(80문), 벽라춘 차(30문/호)
- **특징**: 찻잔이 전부 자사호(보라색 도자기). 차 맛이 다른 객잔과 다름

**3화(2일차) 사건 예정**:
- 아침: 패산객잔 출발. 진삼덕·진소아 부녀와 이별 (진삼덕은 의흥에서 장사 → 재등장 복선)
- 오전: 이동 중 3인격 코미디. 소연화 위소운 옆에 자연스럽게 붙음. 당찬 뒤에서 봄
- 낮: 의흥현 도착. 도자기 가마 연기+차향. 자사루 투숙
- 낮: 이준혁 팬텀 습관 — 자사호 가격 환산 "현대에서 수백만 원인데..."
- 저녁: **소연화가 위소운에게 차를 따라줌** (관계 발전, 물리적 거리 좁아짐)
- 저녁: **"매화검법 후계자" 소문** (옆 탁자 무림인에게서 들음) → 위소운 반응 → 천마 "너도 썼잖아" → **위소운 최초 직접 고백: "사형이에요"**
- 밤: 달빛 아래 검 손질. 당찬이 검 다루는 태도 관찰 → 내심 감탄
- 밤: **소연화가 여자방에서 달빛 아래 검 든 위소운 실루엣 목격**
- 절단신공: "개봉까지 5일. 곽진. ......보고 결정하자."
- 출전 결심 온도: 🔥 25%

#### 3일차: 의흥 → 진강부 (鎭江府) — 약 120리

**구간**: 의흥 → 관도 서북 → 구릉지대 → 진강부 성문
**지형**: 구릉 → 평야 → 장강(長江)이 보이기 시작 → 대도시

**진강부(鎭江府)**:
- 장강 남안(南岸)의 대도시. 인구 약 10만. 장강 도하의 핵심 나루터
- **핵심**: 장강을 건너야 개봉으로 갈 수 있음. 나루터(渡口)가 3곳
- **분위기**: 장강의 위엄. 강폭이 수 리(1~2km). 강바람이 거세고 물결이 높음. 부두에 대형 범선과 나룻배. 짐꾼과 선원, 장사꾼으로 붐빔
- **명소**: 금산사(金山寺) — 장강 위 섬에 있는 절. 멀리서도 보임
- **시장**: 나루터 앞 시장. 남방 물산(비단, 차, 도자기)과 북방 물산(가죽, 약재, 말)이 교차
- **주요 거리**: 나루거리(渡口街) — 여관·주막·환전상이 밀집
- **주의**: 나루터 주변에 소매치기와 사기꾼 많음. 도강비(渡江費)를 터무니없이 부르는 뱃사공

**진강 객잔 — 강월루(江月樓)** ⭐⭐⭐⭐:
- **등급**: 상급 객잔. 3층 건물. 장강이 보이는 전망
- **주인**: 전직 선원 출신 60대. 한쪽 다리가 나무의족. 장강 물길을 꿰뚫음
- **객실**: 상등방 200문(강 전망), 중등방 120문, 하등방 50문
- **대표 메뉴**: 장강 쏘가리찜(150문), 진강 식초 냉채(60문), 진강 향초(30문/호)
- **특징**: 3층 테라스에서 장강 야경이 보임. 밤에 강 위 어선 등불이 별처럼 떠다님

**스토리 포인트**:
- **장강 첫 대면**: 위소운+이준혁+천마 모두 장강의 위엄에 반응
  - 위소운: "......넓다." (태호보다 훨씬 큼)
  - 이준혁: "한강의 3배는 되는 것 같은데요." (현대 비교)
  - 천마: "300년 전에도 이 강은 이랬어." (감회)
- 도강비 흥정 → 이준혁 협상 본능 발동
- 금산사가 보이는 야경 → 감성 장면

#### 4일차: 진강 → 장강 도하 → 양주부 (揚州府) — 약 80리

**구간**: 진강 나루터 → 장강 도하(배, 1~2시간) → 북안 과보진 → 양주부 성문
**지형**: 장강 수면 → 북안 평야 → 양주부 대도시

**장강 도하**:
- 나룻배: 약 1~2시간. 도강비 1인당 50문 (이준혁이 30문으로 깎음)
- 강폭 넓고 물결 있음. 배가 흔들림
- 이준혁: 배멀미 안 하려고 안간힘 (감각 공유이므로 셋 다 느낌)

**양주부(揚州府)**:
- **장강 북안 최대 도시**. 인구 약 30만. 이 세계 3대 도시 중 하나
- **핵심**: 소금 무역의 중심. 염상(鹽商)들이 천하의 부를 쥐고 있음. 부자 도시
- **분위기**: 화려함. 운하(대운하)가 도시를 관통. 버들나무가 운하변에 늘어짐. 야시장이 밤새 열림. 기생집·찻집·음식점이 즐비. "돈 냄새가 나는 도시"
- **명소**: 수서호(瘦西湖) — 서호를 본떠 만든 인공 호수. 버들+다리+정자. 달빛 명소
- **시장**: 동관가(東關街) — 강북 최대 상업 거리. 소금·비단·약재·보석 거래
- **무림**: 양주에는 크고 작은 상단이 40개 이상. 상단 호위 무사 수요가 높아 무인이 많음. 무림맹 양주 분타(分舵)가 있음
- **음식**: 양주볶음밥(원조), 대자두부(大煮干絲), 사자머리(獅子頭, 대형 완자), 양주 3칼(三刀: 요리칼, 이발칼, 발칼)

**양주 객잔 — 취선루(醉仙樓)** ⭐⭐⭐⭐:
- **등급**: 상급 객잔. 3층. 운하변 위치. 버들나무 그늘
- **주인**: 전직 염상 부인. 40대 여성. 화려한 옷. 장사 수완 뛰어남
- **객실**: 상등방 300문(운하 전망), 중등방 150문, 하등방 60문
- **대표 메뉴**: 양주볶음밥(40문), 사자머리(120문), 대자두부(80문), 양주 소흥주(60문/병)
- **특징**: 2층 난간에서 운하 뱃놀이가 보임. 밤에 등불 배가 떠다님

**스토리 포인트**:
- **양주의 부(富)에 충격**: 서구진과 차원이 다른 규모
  - 이준혁: "여기 GDP가 서구진의 100배는..." (경영 본능 폭발)

#### [떡밥 라인 A] 화산파 — 구경에서 출전으로, 위소운의 결심이 익어간다

> **핵심**: 처음에는 "구경만" 하려 했던 무림대회. 7일 동안 곽진 소문이 점점 커지면서 — 위소운은 구경자에서 참가자로 바뀐다. 갑자기 결심하는 게 아니라, **7일에 걸쳐 천천히 익어가는 결심**.

| 일차 | 화수 | 화산파 떡밥 | 위소운 반응 | 3인격 반응 | 출전 결심 온도 |

> **⚠️ 위소운은 신분을 안 밝힌다**: "화산파"라고 안 적음. 소속: 없음. 이름만 적음. → 곽진은 "위소운"이라는 이름을 보고도 못 알아봄 (7년 전 14살 꼬마 → 지금 21살 청년. 죽은 줄 알았으니까)
> **⚠️ 곽진이 알아보는 순간**: 대회장에서 직접 만났을 때. 얼굴을 보는 순간. → 300화 급 명장면

#### [떡밥 라인 C] 당찬 — "질투에서 인정으로, 그리고 충격"

| 일차 | 화수 | 당찬의 변화 |
|------|------|-----------|
| **1일 오전** | **2화** | 앞에서 걸음. 등이 곧고 보폭 넓음. 수석제자의 걸음. 소연화가 위소운 옆에 가는 걸 봄 → 기분 묘함 |
| **1일 오후** | **2화** | 도적 제압을 봄. 자기가 검 뽑기 전에 끝남. 할 말 잃음. 저녁에 "일류 고수면 대단하군" (경쟁심+위소운과 비교 시작) |

#### [떡밥 라인 D] 3인격 — "구경" → "출전"으로 가는 머릿속 전쟁

### 곽진과의 16강전 — 상세 설계

> ⚠️ **이 경기가 전반부 최대 카타르시스.**
> 곽진은 **16강에서 위소운에게 압도적으로 패배**한다. 일찍 정체를 공개하여 나머지 대회 동안 파장이 퍼짐.

| 단계 | 내용 |
|------|------|
| 대진 발표 | 곽진은 위소운의 이름을 모름 (무소속, 처음 보는 이름). 무관심 |
| 경기 시작 | 위소운이 검을 뽑음. 곽진이 자세를 봄. "......이 검법은?" 매화검법의 기초 동작이 섞여 있음 |
| 긴가민가 | 곽진의 눈썹이 미세하게 떨림. 얼굴을 봄. "어디서......" 이름은 "위소운"이지 "위소검"이 아님. 그러나 위(魏)씨 + 눈매 + 매화검법 → 불안 |
| 동요 | 싸우면서 점점 확신이 쌓임. 매화검법 기초 동작 + 눈매 + 같은 성(魏) → 퍼즐. 손에 땀 |
| 전투 | 위소운이 압도. 곽진은 "설마"라는 생각이 머리를 점령하면서 기교가 흐트러짐. 일류 최상위 실력이 심리 붕괴로 반도 못 나옴 |
| 결말 | 위소운이 마지막 일격을 멈춤. 곽진의 귀에만 들리게: **"형, 오랜만이야."** → 곽진의 눈이 최대한 벌어짐. 이제 확신. 공포 |
| 3인격 | 천마: "죽여." 이준혁: "관중 앞에서 죽이면 끝입니다." 위소운: "안 죽여. 죽이면 나도 그놈하고 같아져." |
| 파장 | 16강에서 일찍 터지면서 → 나머지 대회 동안 "저 놈이 위소검?" 소문 확산. 화산파 동요. |
| ⚠️ 핵심 | 곽진이 밀어 떨어뜨린 건 **"위소검"**. 대회에 나온 건 **"위소운"**. 이름부터 다르고 + 7년 세월 + 14세→21세 외모 변화. 바로 알아볼 수 없음. **"형, 오랜만이야"를 듣고서야 확신** |
