### 천화련(天火聯) — 조직 구조 (무림대회 이후)

> ⚠️ **천화련 = 지주회사(그룹). 산하에 3개 사업부.**
> 현대로 치면 **삼성그룹** = 천화련, 산하에 삼성물산+CJ푸드빌+신세계.

#### 전체 구조

```
천화련(天火聯) ← 그룹 전체 / 위소운 총수, 이준혁 경영, 천마 무력 총괄
│
├── 🗡️ 천화표국(天火鏢局) — 무력 + 물류 + 호위
│   ├── 본사 겸 개봉지국 = 옛 안씨표국 (300년 뿌리)
│   ├── 소주지국, 항주지국... (각 성 1~2개)
│   ├── 물류 호위 (천화루/천화상회 상품 배송 보호)
│   └── 정보 수집 (각 지역 동향 파악)
│
├── 🍜 천화루(天火樓) — 객잔+식당 통합 (중국식 주루 모델)
│   ├── ⭐ 1층 = 식당/주점 (이준혁 레시피), 2~3층 = 객실(숙박)
│   ├── 개봉 본점, 소주점, 항주점... (각 성 체인)
│   └── "먹고 자고"를 한 건물에서 = 객잔이 곧 천화루
│
└── 🏪 천화상회(天火商會) — 유통/소매 (백화점형 매장)
    ├── 천화루 옆 또는 1층 한편에 매장
    ├── 이준혁 발명품 주력: 증류주, 비누, 양념, 생활용품
    └── "귀족만 쓰던 물건을 모두에게" = 시장 혁명
```

> ⚠️ **천화루 = 객잔+식당 통합**: 중국식 주루(酒樓)/객잔(客棧)은 원래 **한 건물**.
> 1층에서 먹고 마시고, 2~3층에서 잔다. 객잔과 식당을 따로 분리하는 건 비현실적.
> 천화루 하나가 "숙박+식사+주점"을 전부 해결. 천화상회는 바로 옆에 배치.
>
> ⚠️ **지역 거점 구성**: 각 성마다 **천화루(숙식) + 천화상회(쇼핑) + 천화표국 지국(호위/물류)**
> 여행자 동선: 천화루에서 묵고+먹고 → 옆 천화상회에서 쇼핑 → 짐은 천화표국이 배송
> 이준혁: "고속도로 휴게소 모델입니다. 한곳에서 다 해결."
> 천마: "그게 뭔데."

#### 천화상회(天火商會) — 이준혁의 핵심 사업

> ⭐ **"귀족만 쓰던 것을 모두에게."** 이것이 천화상회의 슬로건.
> 이 시대에는 시계(해시계 정도)도 없고, 비누는 귀족 전용, 증류주는 존재하지 않음.
> 이준혁이 현대 지식으로 만든 상품을 유통하는 **백화점형 체인 매장**.

| 상품 카테고리 | 주력 상품 | 이 시대 상황 | 이준혁의 혁신 |
|-------------|----------|-----------|------------|
| **주류** | 천화소주(증류주) | 막걸리/약주 수준(15도 이하) | 30~40도 증류주. "술이 불처럼 타오른다" |
| **위생** | 천화비누 | 귀족만 향료 사용. 일반인은 물로만 | 저가 비누 대량생산. "모든 사람이 깨끗해질 수 있다" |
| **양념** | 천화장(간장류), 천화유(참기름) | 소금+된장 위주 | 다양한 양념 = 천화루 맛의 비밀 |
| **생활** | 개량 등잔, 보온 용기 | 기름등잔 어두움, 음식 금방 식음 | 이준혁의 "작은 불편 해결" 철학 |
| **계절** | 냉면(여름), 화로구이 세트(겨울) | 계절 특화 상품 없음 | 계절마다 신상품 = 반복 매출 |

> 이준혁: "비누 하나에 5문이면 됩니다. 귀족 향수는 500문. 100배 차이인데, 세정 효과는 같아요."
> 천마: "그래서 100배 더 많이 팔린다는 거야?"
> 이준혁: "정확합니다. 이게 **대중 시장**입니다."

#### 안씨표국 → 천화표국 본사 겸 개봉지국

> ⭐ **안씨표국이 천화표국으로 리브랜딩.** 새로 만든 게 아니라 300년 기존 조직이 확장.
> 안씨표국 간판 → 천화표국 간판으로 교체. 사람·건물·인맥은 그대로.
> "주인이 돌아와서 가게 이름을 바꾼 것"일 뿐.
>
> ⭐ **거점 배치 (개봉 안씨표국 일대)**:
> ```
> [안씨표국 본건물] ← 천화표국 개봉지점 (안효림 국주)
>    │                  무력/물류/호위/정보
>    │
> [옆 대형 건물 인수] ← ⭐ 안씨표국의 5배 규모!
>    1층: 증류주 공장 + 식당(천화루 1호점)
>    2층: 객잔(숙박) + 주점 + VIP실
>    3층: 객잔(숙박) + 연회장
>    별채: 천화 연구방 (양념 R&D) + 천화상회 매장
>    관리: 안세진(총괄) + 안효연(실무)
> ```
> 이준혁: "본사 건물보다 5배 큰 객잔이 바로 옆에 있다니, 이건 입지의 기적입니다."

| 항목 | 안씨표국 (변경 전) | 천화표국 개봉지국 (변경 후) |
|------|-----------------|----------------------|
| 규모 | 30~50명 | 100~200명 (무림대회 인연 합류) |
| 업무 | 소규모 호위 | 천화루/천화상회 물류 + 호위 + 정보 |
| 무공 | 심법 4층 + 검법 6초식 (일류) | ⭐ **심법 5층 + 검법 12초식 (절정)** |
| 간판 | 안씨표국 | 천화표국 개봉지국 |
| 위상 | 개봉 소규모 표국 | 천화련 본사 직할 + 개봉 지역 표국 |
| **국주** | 안세진 (표국주) | ⭐ **안효림** (장남. 아버지는 신사업 전념) |

#### 무공 전수 정책 — "절정급 정예 부대"

> ⭐ **변경**: 천화표국 소속 무인에게는 **심법 5층 + 검법 12초식 전부** 전수.
> 이전의 "반만 주는 잠금장치"에서 → "절정까지 키워서 전력화"로 정책 변경.

| 구분 | 전수 범위 | 도달 경지 | 비고 |
|------|----------|----------|------|
| **천화표국 일반 무인** | 심법 1~5층 + 검법 12초식 | **절정** | 대부분의 표사/호위 |
| **천재급 인재** | 동일 + 자력 깨달음 | **초절정 가능** | 안효림 등 재능 있는 자 |
| **위소운만** | 9층 전부 | 현재 6층(초절정), 이후 화경→현경 | 최고 권력은 주인만 |

> 이준혁: "직원을 강하게 키워야 회사가 큽니다. 단, CEO보다 강해지면 안 됩니다."
> 천마: "5층이면 충분해. 6층 이상은... 그 놈들이 감당할 수준이 아니야."
> **핵심**: 6층 이상(초절정~현경)은 위소운 독점 = 조직 내 절대 권위 유지

#### 지역별 지사+분타 시스템 — "당근과 채찍"

> 이준혁: "프랜차이즈는 현지 파트너가 핵심입니다. 협력하면 같이 커지고..."
> 천마: "안 하면?"
> 이준혁: "...그때는 천마 님 차례입니다."
> 위소운: "최대한 안 다치게. 하지만 안 되면 안 되는 거예요."

#### ⚔️ 적대적 장악 ① — 호남(湖南) 장사: 흑사련(黑蛇聯) 제압

> ⭐ **"첫 번째 본보기."** 이 사건 이후 다른 지역 토호들이 천화련과 싸우면 어떻게 되는지 깨닫는다.

| 항목 | 내용 |
|------|------|
| **시기** | 76~100화경 (승(承) 초반) |
| **적대 세력** | **흑사련(黑蛇聯)** — 호남 장사를 지배하는 악질 무력 조직 |
| **흑사련 실체** | 상인에게 "보호세" 30% 징수. 외부 상단 진입 차단. 관아와 결탁. 절정급 수장 + 일류 고수 5~6명 |
| **파트너십 시도** | 이준혁이 먼저 협력 제안 → 흑사련: "우리 땅에 감히? 보호세 50% 내고 장사해라." 거부 |
| **3인격 반응** | 이준혁: "협상이 불가합니다." / 천마: (미소) "드디어 내 방식이야." / 위소운: "...합시다." |
| **장악 과정** | 위소운 + 조칠 + 철무광 등 정예 투입. 흑사련 본거지 급습. 수장 제압. 항복 받음. |
| **결과** | 흑사련 해체. 보호세 폐지. 장사 상인들 환호. 천화루 장사 지사 설립 (직영). |
| **파장** | ⭐ **"호남의 뱀이 밟혔다"** 소문이 전국에 퍼짐. 다른 지역 토호들 긴장. |

**스토리 효과:**
- ⚡ **천마 활약**: 작전 지휘. "300년 전에 이런 벌레는 하루에 세 무리씩 밟았어."
- 😂 **이준혁 코미디**: 전투 직후 "피해 보상금 산정이..." / "인수 후 리브랜딩 비용이..."
- 💪 **위소운 성장**: "밥값"이 아니라 "정의"로 싸운 첫 대규모 전투
- 📢 **경각심**: 파트너십 제안 → 거부 → 장악. "순서를 기억하세요." (이준혁)
- 🏪 **사업 효과**: 보호세 없는 장사 → 상인들이 천화루에 몰림 → 폭발적 매출

#### ⚔️ 적대적 장악 ② — 산서(山西) 태원: 철기방(鐵騎幇) 제압

> ⭐ **"두 번째이자 마지막 본보기."** 이 이후로는 아무도 천화련에 정면으로 맞서지 않는다.

| 항목 | 내용 |
|------|------|
| **시기** | 151~175화경 (전(轉) 초반) |
| **적대 세력** | **철기방(鐵騎幇)** — 산서 태원을 거점으로 한 북방 무력 방파. 거칠고 잔인함 |
| **철기방 실체** | 북방 상로(商路)를 장악. 통행세 징수. 무역 독점. 화경급 방주 + 절정급 부방주 2명 + 정예 500명 |
| **차이점** | 흑사련보다 훨씬 강함. 천화련이 성장한 이후의 "진짜 전쟁" |
| **파트너십 시도** | 이준혁 제안 → 철기방: "남쪽 장사꾼이 여기까지? 꺼져라." 위소운 사절단 습격 |
| **3인격 반응** | 이준혁: "사절단을 공격했으니 이건 선전포고입니다." / 천마: "북방 놈들은 맞아야 알아듣는다." / 위소운: "우리 사람을 건드렸어." (분노) |
| **장악 과정** | 대규모 원정. 위소운이 화경급 방주와 직접 대결. 천화련의 전력을 총동원한 전면전. |
| **결과** | 철기방 방주 패배. 항복하지 않으면 해체. 태원 직영 지사 설립. |
| **파장** | ⭐ **"태원의 철기마저 꺾었다"** → 이후 남은 지역(복건, 광동)은 천화련이 오면 자발적 파트너십 요청 |

**스토리 효과:**
- ⚡ **위소운 화경 실력 공개**: 화경급 방주를 직접 상대 = 천하에 실력 증명
- 🔥 **천화련 위상**: "사업 조직인 줄 알았는데 문파도 이렇게 강해?"
- 📢 **결정적 경각심**: 흑사련(절정급) 제압은 "강한 편"이라 생각했는데, 철기방(화경급)도 밟힘 → "아무도 못 이긴다"
- 🤝 **이후 파트너십 가속**: 복건/광동 지역 세력이 먼저 연락 "저희가 천화루 유치하고 싶습니다"

#### 당근과 채찍의 효과 — 소문의 힘

```
[1차] 호남 흑사련 제압 (76~100화)
  → 소문: "보호세 뜯다가 밟혔대" → 섬서/강서 파트너십 순조로움

[2차] 산서 철기방 제압 (151~175화)  
  → 소문: "화경급도 졌대" → 복건/광동이 먼저 연락

[결과] 12개 성 중 10개는 평화적 파트너십, 2개만 무력 장악
  → 이준혁: "전체 확장 비용의 80%를 이 2곳에 쓴 겁니다. 하지만 나머지 10곳의 협상 비용이 0이었으니..."
  → 천마: "그래서 남는 장사라는 거야?"
  → 이준혁: "...ROI로 보면 역대급이죠."
```

> ⚠️ **3인격 갈등 포인트**: 적대적 장악 시 항상 의견 충돌
> - 천마: "전부 밟으면 빠르잖아." (무력 선호)
> - 이준혁: "전쟁은 비용이에요. 파트너십이 싸요." (효율 선호)
> - 위소운: "대의명분이 있으면 싸운다. 없으면 안 싸운다." (중재)
> - 흑사련/철기방은 **명확한 악당**이므로 위소운도 동의 = 대의명분 충족

---

> ⚠️ **파트너십 모델** = 프랜차이즈 (10개 성):
> - 천화련이 주는 것: **천화루 간판+레시피** + **천화상회 상품** + **천화표국 호위** = 3종 세트
> - 파트너가 주는 것: 현지 인맥, 운영 인력, 지역 정보, 기존 거점
> - 수익 분배: 매출의 일정 %를 본사에 상납 (로열티)
> - **파트너가 협력하는 이유**: "같이 가면 3가지 사업이 동시에 돈 된다" + "안 가면 호남/산서 꼴 난다"

#### 인력 확보 3단계

| 단계 | 시기 | 출처 | 핵심 인물 |
|------|------|------|----------|
| **1. 대회 인연** | 10~25화 | 무림대회 패배자 중 감복한 자 | 철무광, 엄표, 진소산 등 |
| **2. 현지 영입** | 26~100화 | 각 지역 확장 시 현지인 합류 | 흑호문 인수, 지역 무인/상인 |
| **3. 명성 흡인** | 100화+ | 천화련 명성 → 지원자 쇄도 | 선발 시스템 필요 |

#### 3사업부 시너지 — "삼위일체"

| 천화표국 → | 천화루 → | 천화상회 → |
|-----------|---------|-----------|
| 물류 호위 (상품 배송) | 먹거리 (표사 급식) | 생활용품 (표사 비누/장비) |
| 정보 수집 (각 지역 동향) | 고객 접점 (소문/평판) | 매출 (자금 = 무기/수련장) |
| 무력 보호 (매장 치안) | 레시피 독점 (경쟁력) | 대중 시장 (브랜드 인지도) |

> **순환 구조**: 표국이 지키고 → 루에서 먹이고 → 상회에서 팔고 → 돈으로 표국 키우고 → 반복
> 이준혁: "세 바퀴가 동시에 돌면, 아무도 못 멈춥니다."
