```
곤륜산 (곤륜파) [3일]
  ↓ 2200리
화산 (화산파) [2일]
  ↓ 800리
개봉 (귀환)
```

**이동**: 말 + 경공술 혼용  
**수행원**: 조칠, 왕팔, 설화 + 호위 100명

## 📋 이동 계산 공식 (작가용)

`🔎 기본, 공식, 예시, 계산`

### 기본 공식

```
1. 거리 계산
   km = 리 ÷ 2
   리 = km × 2

2. 도보 시간
   일수 = 총 거리(리) ÷ 50

3. 말 시간 (교체 고려)
   일수 = 총 거리(리) ÷ 150
   말 교체 횟수 = 일수 ÷ 3 (3일마다 교체)

4. 경공술 시간 (일류급)
   시간 = 총 거리(km) ÷ 40
   일수 = 총 거리(리) ÷ 400 (휴식 포함)

5. 경공술 시간 (화경급)
   시간 = 총 거리(km) ÷ 60
   일수 = 총 거리(리) ÷ 600 (휴식 포함)
```

### 예시 계산

**소주 → 개봉 (1600리)**
- 도보: 1600 ÷ 50 = **32일**
- 말: 1600 ÷ 150 = **11일** (말 3회 교체)
- 경공 (일류): 1600 ÷ 400 = **4일**
- 경공 (화경): 1600 ÷ 600 = **2.7일** (3일)

#### 0일차(1화): 서구진 천향루 — 귀환

> **1화는 이동 없음.** 서구진 천향루(天香樓)에서 하루 동안 벌어지는 이야기.

**1화 사건 요약**:
- 점심: 위소운이 천향루 2층 창가석에 착석. 3인격 소개 (이준혁·천마)

- 오후: 소연화 합석 제안 → 4인(당찬·소연화·장위·마현) 소개 → 위소운 "14년 섬에서 수련" 커버스토리
- 오후: 무림대회 정보 획득 → 소연화 "같이 가시겠습니까?" → 위소운 "좋습니다"
- 석양: 서구진 거리 엔딩. "돌아왔구나 싶어서요."

**등장인물**: 위소운, 이준혁(내면), 천마(내면), 당찬, 소연화, 장위, 마현, 흑호문 졸개 4명, 점소이
**2층 창가석**: 당찬·소연화·장위·마현이 먼저 앉아 있었고, 위소운은 반대편 창가석에 혼자 앉음 → 합석

**패산객잔 (覇山客棧)** ⭐:

**의흥현(宜興縣)**:

**의흥 객잔 — 자사루(紫砂樓)** ⭐⭐⭐:
- **등급**: 중급 객잔. 2층 건물. 1층 식당+다실, 2층 객실

#### 3일차: 의흥 → 진강부 (鎭江府) — 약 120리

**구간**: 의흥 → 관도 서북 → 구릉지대 → 진강부 성문
**지형**: 구릉 → 평야 → 장강(長江)이 보이기 시작 → 대도시

**진강부(鎭江府)**:
- 장강 남안(南岸)의 대도시. 인구 약 10만. 장강 도하의 핵심 나루터
- **핵심**: 장강을 건너야 개봉으로 갈 수 있음. 나루터(渡口)가 3곳
- **분위기**: 장강의 위엄. 강폭이 수 리(1~2km). 강바람이 거세고 물결이 높음. 부두에 대형 범선과 나룻배. 짐꾼과 선원, 장사꾼으로 붐빔
- **명소**: 금산사(金山寺) — 장강 위 섬에 있는 절. 멀리서도 보임
- **시장**: 나루터 앞 시장. 남방 물산(비단, 차, 도자기)과 북방 물산(가죽, 약재, 말)이 교차
- **주요 거리**: 나루거리(渡口街) — 여관·주막·환전상이 밀집
- **주의**: 나루터 주변에 소매치기와 사기꾼 많음. 도강비(渡江費)를 터무니없이 부르는 뱃사공

**진강 객잔 — 강월루(江月樓)** ⭐⭐⭐⭐:
- **등급**: 상급 객잔. 3층 건물. 장강이 보이는 전망
- **주인**: 전직 선원 출신 60대. 한쪽 다리가 나무의족. 장강 물길을 꿰뚫음
- **객실**: 상등방 200문(강 전망), 중등방 120문, 하등방 50문
- **대표 메뉴**: 장강 쏘가리찜(150문), 진강 식초 냉채(60문), 진강 향초(30문/호)

**스토리 포인트**:
- **장강 첫 대면**: 위소운+이준혁+천마 모두 장강의 위엄에 반응

### 곽진과의 16강전 — 상세 설계

> ⚠️ **이 경기가 전반부 최대 카타르시스.**
> 곽진은 **16강에서 위소운에게 압도적으로 패배**한다. 일찍 정체를 공개하여 나머지 대회 동안 파장이 퍼짐.

| 단계 | 내용 |
|------|------|
| 대진 발표 | 곽진은 위소운의 이름을 모름 (무소속, 처음 보는 이름). 무관심 |
| 경기 시작 | 위소운이 검을 뽑음. 곽진이 자세를 봄. "......이 검법은?" 매화검법의 기초 동작이 섞여 있음 |
| 긴가민가 | 곽진의 눈썹이 미세하게 떨림. 얼굴을 봄. "어디서......" 이름은 "위소운"이지 "위소검"이 아님. 그러나 위(魏)씨 + 눈매 + 매화검법 → 불안 |
| 동요 | 싸우면서 점점 확신이 쌓임. 매화검법 기초 동작 + 눈매 + 같은 성(魏) → 퍼즐. 손에 땀 |
| 전투 | 위소운이 압도. 곽진은 "설마"라는 생각이 머리를 점령하면서 기교가 흐트러짐. 일류 최상위 실력이 심리 붕괴로 반도 못 나옴 |
| 결말 | 위소운이 마지막 일격을 멈춤. 곽진의 귀에만 들리게: **"형, 오랜만이야."** → 곽진의 눈이 최대한 벌어짐. 이제 확신. 공포 |
| 3인격 | 천마: "죽여." 이준혁: "관중 앞에서 죽이면 끝입니다." 위소운: "안 죽여. 죽이면 나도 그놈하고 같아져." |
| 파장 | 16강에서 일찍 터지면서 → 나머지 대회 동안 "저 놈이 위소검?" 소문 확산. 화산파 동요. |
| ⚠️ 핵심 | 곽진이 밀어 떨어뜨린 건 **"위소검"**. 대회에 나온 건 **"위소운"**. 이름부터 다르고 + 7년 세월 + 14세→21세 외모 변화. 바로 알아볼 수 없음. **"형, 오랜만이야"를 듣고서야 확신** |
