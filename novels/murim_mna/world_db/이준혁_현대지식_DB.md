## 🔥 이준혁의 욕구불만 시스템 (Frustration-Driven Innovation)

`🔎 이준혁이, 미치도록, 그리운, TOP, 욕구불만, 신라면이, 먹고, 싶다, 매운, 국물면`

> **핵심 공식**: 불편/그리움 → 해결 시도 → 실패 → 재시도 → 성공 → 사업화
> **독자 공감 포인트**: "나도 1100년에 가면 라면 먹고 싶을 거야!"
> **코미디 엔진**: 미슐랭 셰프 출신 천재가 "재료가 없다! 후추 한 줌이 금값이다!"고 좌절하는 갭

### 이준혁이 미치도록 그리운 것 TOP 15

| 순위 | 그리운 것 | 그리움 강도 | 1100년 상태 | 해결 가능? |
|------|----------|-----------|-----------|----------|
| 1 | **신라면 (매운 국물면)** | ★★★★★ | 면은 있으나 고추 없음 | △ 유사품 가능 |
| 2 | **커피** | ★★★★★ | 존재하지 않음 | ✕ 불가 → 차(茶)로 대체 |
| 3 | **화장실 (수세식)** | ★★★★★ | 구덩이 변소 | △ 개선 가능 |
| 4 | **샤워 / 비누** | ★★★★ | 목욕탕은 있으나 비누 없음 | ✅ 비누 제조 가능 |
| 5 | **치킨 (양념/후라이드)** | ★★★★ | 닭은 있으나 튀김 기술 부족 | ✅ 가능! |
| 6 | **삼겹살 + 소주** | ★★★★ | 돼지는 있으나 소주 없음 | ✅ 증류주로 해결 |
| 7 | **김치 (빨간 김치)** | ★★★★ | 고추 없음 → 백김치만 | △ 유사품 가능 |
| 8 | **된장찌개** | ★★★ | 두장(豆醬) 존재 → 비슷하게 가능 | ✅ 가능! |
| 9 | **치약 / 칫솔** | ★★★ | 없음 → 소금+버드나무 가지 | △ 개선 가능 |
| 10 | **따뜻한 온돌방** | ★★★ | 중국은 화로/난로 → 추움 | △ 온돌 설치 가능 |
| 11 | **스마트폰** | ★★★★★ | 존재하지 않음 | ✕ 절대 불가 |
| 12 | **시계** | ★★★ | 해시계/물시계만 | △ 기계식 가능? |
| 13 | **안경** | ★★ | 없음 (유리 자체가 귀함) | △ 수정 렌즈? |

> 꿈에서 깨어난 후 "비슷하게 만들 수 있을까?" → 사업 아이디어로 연결.

### 🍜 욕구불만 #1: "신라면이 먹고 싶다" — 매운 국물면 프로젝트

#### 문제 분석
```
신라면 재료:
- 밀가루 면 → ✅ 1100년에 있음 (국수/탕면)
- 고춧가루 → ❌ 없음! (고추 = 16세기 아메리카에서 전래)
- MSG → ❌ 없음
- 쇠고기 육수 → ✅ 있음
- 파, 마늘 → ✅ 있음
- 간장 → ✅ 있음 (간장은 중국이 원조)

결론: "빨간 매운맛"은 불가. 하지만 "다른 종류의 매운맛"은 가능!
```

#### 1100년에 가능한 "매운맛" 재료

| 재료 | 한자 | 매운맛 종류 | 강도 | 구할 수 있나? |
|------|------|-----------|------|------------|
| **화자오 (사천 후추)** | 花椒 | 마비 + 얼얼한 매움 (마라!) | ★★★★ | ✅ 사천 특산 |
| **겨자** | 芥子 | 코를 찌르는 매움 | ★★★ | ✅ 중국 전역 |
| **생강** | 生薑 | 알싸한 매움 | ★★ | ✅ 흔함 |
| **후추 (흑/백)** | 胡椒 | 톡 쏘는 매움 | ★★★ | △ 비쌈 (수입품) |
| **산초** | 山椒 | 화자오와 비슷 | ★★★ | ✅ 산지 채취 |
| **마늘** | 大蒜 | 알싸함 | ★★ | ✅ 흔함 |
| **부추** | 韭菜 | 매콤한 향 | ★ | ✅ 흔함 |

#### "이준혁 특제 마라탕면" 레시피

```
[1100년 버전 신라면]

이름: 천화면(天火麵) — "하늘의 불 같은 면"

베이스: 소뼈 + 닭뼈 우린 육수 (12시간)
매운맛: 화자오 갈은 가루 + 겨자즙 + 생강 + 후추
         → 고추 매운맛은 못 내지만, "마라(麻辣)" 매운맛 가능!
면: 밀가루 수타면 (기름에 살짝 튀긴 후 육수에)
토핑: 파, 마늘, 건두부, 고수
간: 간장 + 소금 + 두반장(고추 없는 버전)

→ 결과: 신라면은 아니지만, "매운 국물면"은 완성
```

### 🍗 욕구불만 #4: "치킨이 먹고 싶다" — 튀김 프로젝트

| 재료 | 1100년 상태 | 대체 |
|------|-----------|------|
| 닭 | ✅ 있음 | - |
| 튀김 기름 | △ 참기름/돼지기름 | 충분. 대량 튀김 가능 |

```
이준혁이 주방에서 지시했다.
기름을 가득 채운 솥.
닭에 밀가루를 입히고, 달걀물을 바르고, 다시 밀가루.

"넣으십시오."

치지지지직——

기름 위로 황금빛 거품이 올라왔다.
향이 퍼졌다.

조칠: "이...이 냄새는 뭡니까?!"
왕팔: (침 꿀꺽)

10분 후.
황금빛으로 튀겨진 닭.
간장, 꿀, 마늘을 섞은 양념을 바른다.

위소운이 한 입 베어 물었다.
바삭. 
육즙이 터졌다.

위소운: "..."
이준혁: "어떠십니까?"
위소운: "...이건 뭐야?"
이준혁: "닭입니다."
위소운: "닭이 이런 맛이 날 수 있어?"
이준혁: (미소) "기름의 마법입니다."

천마: (내면) '...이 몸이 300년을 살았지만, 
       이런 닭은 처음이다. 
       인정하기 싫지만... 맛있다.'

조칠: (입에 닭다리 물고) "두목님! 매일 이거 먹읍시다!"
왕팔: "이거 팔자. 형, 이거 팔면 대박이야."
이준혁: (눈 빛남) "...그것도 좋은 생각입니다."
```

**사업화**: "천화루(天火樓)" — 위소운 상단의 고급 요식 브랜드
- 메뉴: 천화면(마라탕면) + 천화계(양념치킨) + 천화주(증류주)
- "천화(天火)" = 하늘의 불 = 매운맛+강한 술+튀김의 불

### ☕ 욕구불만 #5: "커피가 마시고 싶다" — 해결 불가의 슬픔

```
[20화쯤]

이준혁: (내면) '아메리카노.'
         길을 걸을 때마다 생각난다.
         아침에 눈을 뜨면 제일 먼저 찾았던 것.

         이 세상에 커피는 없다.
         커피콩은 아프리카에 있다.
         여기서는 갈 수가 없다.
         
         '...차라도 마시자.'
         
이준혁이 진한 녹차를 찾기 시작했다.
가장 쓰고, 가장 진한 차.

차관에서 물었다.
"가장 쓴 차는 무엇입니까?"
"고정차(苦丁茶)요. 아무도 안 마시는데..."

이준혁: "그걸 주십시오."
```

### "왜 이걸 아는가?" — 소설 속 자연스러운 설명 대사 패턴

#### 패턴 A: "프로젝트에서 배웠다" — 경영 지식 (맥킨지 출처)
```
왕팔: "형, 술을 더 독하게 만들 수 있다고? 어떻게?"
이준혁: "...전에 주류 회사 일을 한 적이 있습니다.

         그 증기를 모으면 더 강한 술이 됩니다."
왕팔: "직접 만들 수 있어?"
이준혁: "원리는 압니다. 다만 장비가 문제입니다.
         이 시대 도구로 증류기를 만들어야 합니다."
```

#### 패턴 A-2: "직접 해봤다" — 요리 지식 (미슐랭 출처) ⭐ 핵심 차별화
```
조칠: "이 닭을 어떻게 하면 더 맛있어질까?"
이준혁: (소매를 걷으며) "...제가 직접 하겠습니다."
         (닭을 칼로 발골하며) "먼저 뼈를 분리합니다.
         이 방식이 가장 기름이 균일하게 튀겨집니다."
조칠: "...형, 칼 쓰는 거 누가 가르쳐준 거야?"
이준혁: "...세상에서 가장 좋은 식당에서 2년간 배웠습니다."
```
> ⭐ **핵심**: 경영 지식은 "배웠다"(간접), 요리는 "직접 한다"(직접). 이 차이가 독자에게 신선함.

## 💰 분야 1: 금융/회계 혁신 (이준혁의 최강 무기)

`🔎 복식부기, Double, Entry, Bookkeeping, 보험, Insurance, 선물거래, Futures, Contract, 복리`

### 1-1. 복식부기 (Double-Entry Bookkeeping)

**역사**: 1494년 이탈리아 루카 파치올리가 체계화. 무림 세계에는 없음.
**무림 현재**: 장부 = "들어온 돈, 나간 돈" 단순 기록 (단식부기)

| 무림 현재 방식 | 이준혁이 도입하는 방식 |
|-------------|-------------------|
| "오늘 비단 팔아서 50냥 벌었다" | 차변: 현금 +50냥 / 대변: 매출 +50냥 |
| 이익인지 손해인지 연말에야 앎 | 실시간으로 자산/부채/이익 파악 |
| 횡령 발견 불가 | 차변≠대변 → 즉시 이상 감지 |

**소설에서의 활용**:
- **15화**: 왕팔 합류 후, 이준혁이 "새로운 장부 체계"를 도입
- 이준혁: "대표님, 이것을 '복식장부'라 합니다. 돈이 들어오면 반드시 어디서 왔는지, 나가면 어디로 갔는지 두 줄로 기록합니다."
- 왕팔: (눈 빛남) "이거... 이러면 누가 돈을 빼돌려도 바로 보이잖아?!"
- **효과**: 다른 상단 대비 압도적 재무 투명성 → 투자자 신뢰 → 성장 가속

### 1-2. 보험 (Insurance)

**역사**: 1686년 런던 로이드 커피하우스에서 시작. 무림에 없음.
**무림 현재**: 상단이 도적에게 털리면 → 그냥 손해

| 무림 현재 | 이준혁이 만드는 보험 |
|----------|-------------------|
| 도적 피해 = 전액 손실 | 매달 소액 납부 → 피해 시 보상금 지급 |
| 상인이 위험 감수 | 위험 분산 → 더 멀리 교역 가능 |
| 없음 | "무림 최초 상호보험" |

**소설에서의 활용**:
- **60화**: 이준혁이 소주 상인들에게 "상호보험" 제안
- 이준혁: "매달 거래액의 2%를 공동 기금에 넣으십시오. 도적 피해, 화재, 수해 발생 시 기금에서 80% 보상합니다."
- 상인A: "그런 게 가능해?"
- 이준혁: "혼자 지면 망하지만, 100명이 나누면 아무도 안 망합니다."
- **효과**: 소주 상인 100명이 자발적으로 위소운 진영에 합류 → 상인 네트워크 확보

### 1-3. 선물거래 (Futures Contract)

**역사**: 1697년 오사카 쌀 거래소가 최초. 무림에 없음.
**무림 현재**: 수확 후 그때그때 거래 → 가격 변동에 무방비

| 무림 현재 | 이준혁의 선물거래 |
|----------|----------------|
| 수확 후 판매 (가격 불확실) | 수확 전에 가격 확정 계약 |
| 풍년 = 가격 폭락 → 농가 손해 | 미리 계약 → 안정 수익 보장 |
| 흉년 = 가격 폭등 → 매입자 손해 | 미리 계약 → 안정 매입가 보장 |

**소설에서의 활용**:
- **90화**: 소주 곡물 시장에서 선물거래 도입
- 이준혁: "봄에 가을 쌀 가격을 미리 정하는 겁니다. 농민은 안정, 우리는 싸게 삽니다."
- 왕팔: "형, 천재야?! 풍년이면 우리가 이득, 흉년이면 농민이 이득... 아, 둘 다 이득?!"
- **효과**: 곡물 시장 지배력 → 식량 = 군사력 = 정치력

### 1-4. 복리 (Compound Interest)

**역사**: 개념은 고대부터 있지만, 체계적 활용은 17세기 이후
**무림 현재**: 고리대금 = 단리. "월 10% 이자" (폭리)

| 무림 고리대금 | 이준혁의 복리 전략 |
|-------------|----------------|
| 단리: 100냥 × 10% × 12개월 = 120냥 이자 | 복리: 100냥 → 5년 후 약 1,600냥 (연 70%) |
| 채무자 파산 → 회수 불가 | 낮은 이율 + 장기 투자 → 안정 수익 |
| "고리대금업자 = 악인" 이미지 | "투자자 = 파트너" 이미지 |

**소설에서의 활용**:
- **40화**: 서구진 상인들에게 저리 대출 시작
- 이준혁: "월 2%입니다. 다른 전당포의 1/5입니다."
- 상인: "이렇게 싸게 빌려주면 손해 아닙니까?"
- 이준혁: (미소) "단기는 손해입니다. 장기는... 복리라는 것이 있습니다."
- **효과**: 낮은 이율로 고객 확보 → 장기 복리 수익 → 경쟁 전당포 도태

### 1-5. 주식회사 / 합자 투자

**역사**: 1602년 네덜란드 동인도회사가 최초 주식회사. 무림에 없음.
**무림 현재**: 사업 = 1인 소유. 자금 부족 = 포기

| 무림 현재 | 이준혁의 합자 구조 |
|----------|-----------------|
| 상단 = 1인 소유주 | 여러 투자자가 지분 투자 |
| 자금 부족 → 성장 한계 | 소액 다수 투자 → 대형 사업 가능 |
| 실패 = 소유주 파산 | 유한책임 → 투자금만 손실 |

**소설에서의 활용**:
- **200화 이후**: 천하재일상단을 "합자 구조"로 전환
- 이준혁: "'지분'이라는 것을 도입합니다. 10만냥 투자하면 상단의 10%를 소유합니다."
- 천만금 상단주: "상단을 나눠 갖는다? 미친 소리!"
- 이준혁: "나눠 가지면 더 커집니다. 혼자 100을 가지는 것보다, 1000의 10%가 더 큽니다."

```
이준혁: "...천마님, 검으로 이기고 돈으로 지면 그건 패배입니다."
천마: "..." 
이준혁: "전쟁에서 이기고 파산하면, 이긴 것이 아닙니다."
천마: "...쯧. 벌레 같은 계산이지만, 틀린 말은 아니로군."
```
