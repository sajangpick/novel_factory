# §2. 다음 화 주의사항 — "14화에서 뭐 조심?"
# ═══════════════════════════════════════

### 스토리 연결 (설계도 master_story_bible 기준)
- **14화**: 우승 후폭풍. 각 세력 스카우트 쏟아짐.
- 이준혁의 파트너 후보 우선순위 정리 (냉철한 분석).
- 3인격 정식 회의 (삼혼귀일경): "각자 하고 싶은 걸 하자"의 구체화 — 자본금 문제 대두.
- 천마의 300년 전 기억: "개봉에... 남겨둔 게 있어." → 안씨표국 방문 결심.
- ⚠️ 소연화·당찬은 천풍검문과 함께 본산으로 귀환. 작별.

### 캐릭터 주의
- **위소운**: 우승 후 번아웃 + 성장. 평어. "이제 시작이다" 톤.
- **이준혁**: 사업 본격 시동. 존댓말. 스카우트 제의를 기업 인수 관점으로 분석.
- **천마**: 300년 전 기억이 떠오르며 감정의 파도. 반말. 그리움을 숨기려 하지만 드러남.
- **소연화**: 작별 장면. "꼭 다시 만나요" 톤. 미련과 호기심.
- **남궁현**: 결승 후 동지 관계 확인. 파트너십 약속.

### 시스템 규칙
- 3인격 엔진: 회의 장면에서 3인격 합주(사업+무력+감정)를 충분히 보여줄 것.
- "시끄러", "나쁘지 않네" 사용 자제 유지.
- 분량: 5,000자 기준 (최대 5,500자 절대 금지).

## 업데이트 규칙
§1 현재 상태 → 최신 집필 화수를 갱신, 위치/시간/건강/무공/소지금 등을 본문 기반으로 갱신
§2 다음 화 주의 → 다음 화 내용으로 교체. 스토리 연결, 캐릭터 주의, 시스템 규칙 작성
§3 활성 떡밥 → 새 떡밥 추가(있으면), 기존 떡밥 상태 변경(힌트 진행/회수 완료 등). 회수 완료(✅)된 것은 표에서 삭제하고 §8 아카이브 안내 코멘트만 남김
§4 관계 매트릭스 → 변한 관계만 수치 갱신 (변화 없으면 유지)
§5 감정 목표 → 해당 화 행 기록 (텐션, 감정 키워드, 핵심 장면). 이준혁 감정 단계도 갱신
§6 확정 팩트 → 새로 확정된 팩트 추가 (있으면)
§7 최근 기억카드 → 해당 화 카드 추가. 6하원칙+핵심+떡밥+핵심대사 형식. 총 5화 유지(가장 오래된 것은 §8로 이동 안내)
