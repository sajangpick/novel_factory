# §2. 다음 화 주의사항 — "14화에서 뭐 조심?"
# ═══════════════════════════════════════

### 스토리 연결
- **14화**: 우승 D+1~2. 각 세력 스카우트 쏟아짐. 3인격 정식 회의: "각자 하고 싶은 걸 하자"의 구체화. 안가(안씨표국) 방문 결심.
- 13화에서 3인격 합의 완료 → 14화에서 구체적 첫 행동
- ⭐ 천마: "개봉에... 남겨둔 게 있어." (300년 전 기억, 안씨표국 떡밥)

### 캐릭터 주의
- **위소운**: 우승으로 이름이 무림에 알려짐. 왼쪽 어깨 상처 회복 중. 만류귀종 40% 발현했지만 재현 불가. "더 강해지고 싶다 + 세상을 보고 싶다"
- **이준혁**: 5단계(절망 진입) 직후이지만 결의로 전환. "사업을 하겠습니다." 위소운 이름값으로 사업 시작 전략 구상. 스카우트를 파트너 후보 목록으로 정리
- **천마**: "마교의 진실 확인" 목표 선언. 야율흑의 뒤틀린 기(氣) 떡밥. 개봉에 300년 전 남겨둔 것이 있음 → 안씨표국
- **소연화**: "내일도 만들어줄게요" — 위소운과의 관계 진전. 대회 끝나도 같이 있을 이유 필요
- **당찬**: 위소운의 우승을 인정(입 다물고 박수). 자기도 성장하고 싶은 동기 강화
- **남궁현**: 패배 후 호쾌하게 인정. "다음에 꼭 이길 테니까." 향후 라이벌+동지+사업 파트너
- **안세진**: 결승을 지켜봤다. 위소운의 마지막 검에서 무언가를 읽음 → 15화 안씨표국 방문 시 복선

### 시스템 규칙
- 3인격 엔진: 정식 회의 가능 (10~15화마다 1회 → 14화 적절)
- 코미디 7비트 필수
- 전투 없는 화 → 인간 드라마 중심. 스카우트 장면 = 이준혁 활약 기회
- "시끄러" 13화에서 1회 사용 → 14~15화에서 사용 자제
- "나쁘지 않네" 13화에서 1회 사용 → 14~15화에서 사용 자제
