## 📊 1차 타겟: 핵심 독자 (Core Reader)

### 프로필

| 항목 | 상세 |
|------|------|
| **성별** | 남성 80%, 여성 20% |
| **연령** | 25~39세 (중심: 30대 초반) |
| **직업** | 직장인, 자영업자, 취준생 |
| **독서 환경** | 출퇴근 지하철, 점심시간, 취침 전 |
| **월 결제** | 2~5만원 (웹소설 유료 결제 의향 있음) |
| **독서 플랫폼** | 카카오페이지 > 네이버시리즈 > 리디 |
| **독서 속도** | 1화(6,000자) = 10~15분 |
| **1일 소비량** | 3~10화 |

### 심리 프로필

| 심리 키워드 | 설명 | 우리 소설이 충족하는 방법 |
|-----------|------|----------------------|
| **대리 만족** | 현실에서 못하는 성공을 대리 체험 | 빚쟁이 → 천하제일 CEO |
| **지적 쾌감** | 머리 쓰는 주인공을 좋아함 | 이준혁의 데이터 분석, 경영 전략 |
| **카타르시스** | 답답한 상황 → 시원한 해결 | 고구마→사이다 패턴 |
| **소속감** | 성장하는 조직에 소속되고 싶음 | 마적 30명 → 5,000명 조직 성장 |
| **자기 투영** | "나도 저럴 수 있을까?" | 바닥에서 시작 → 공감 + 동기부여 |

### 이 독자가 가장 좋아하는 장면

1. **"0에서 시작"** 장면: 빚 3,000냥에서 출발하는 순간 → "나도 바닥인데..."
2. **"통쾌한 복수"** 장면: 과거 자신을 배신한 자를 이기는 순간 → "속 시원하다!"
3. **"조직 성장"** 장면: 부하 수가 늘고 자산이 늘어나는 순간 → "내 것이 커진다"
4. **"숫자로 보는 성과"** 장면: "자산 10만냥 달성" → 게임의 레벨업처럼 수치화된 성장

### 이 독자가 가장 싫어하는 것

1. 주인공이 바보같이 당하는 장면이 5화 이상 지속
2. 의미 없는 전투 반복 (먹렙)
3. 여러 여캐와 로맨스 (하렘 거부감)
4. 작가의 게으른 설명 ("시간이 흘러..." "몇 달 후...")
5. 현실감 없는 급성장 ("하루 만에 절정고수")

## 📊 2차 타겟: 확장 독자 (Extended Reader)

### A그룹: 경영/재벌물 독자

| 항목 | 상세 |
|------|------|
| 특징 | 경영/비즈니스 소설을 좋아하는 독자 |
| 유입 포인트 | "경영+무협? 신선하다!" |
| 핵심 요구 | 경영 전략이 현실적이어야 함 (허술하면 이탈) |
| 주의사항 | 무협 전투가 과도하면 "이건 그냥 무협이잖아" 이탈 |
| 충족 방법 | 이준혁의 경영 분석을 매 화 삽입. M&A 과정 디테일 |

### B그룹: 여성 독자 (성장형)

| 항목 | 상세 |
|------|------|
| 특징 | 성장 서사를 좋아하는 여성 독자 |
| 유입 포인트 | 설화 캐릭터 (독립적 여성상) |
| 핵심 요구 | 로맨스가 메인이 아니되, 있어야 함 |
| 주의사항 | 설화가 "보호받는 여캐"면 즉시 이탈 |
| 충족 방법 | 설화의 독립적 전투 + 동등한 파트너십 |

### C그룹: 전략/지능물 독자

| 항목 | 상세 |
|------|------|
| 특징 | 전독시, 킹덤, 데스노트 같은 두뇌전 좋아하는 독자 |
| 유입 포인트 | 3인격 시스템의 전략적 대화 |
| 핵심 요구 | 전략이 논리적이어야 함 (말이 되어야 함) |
| 주의사항 | "그냥 힘으로 이김" 전개 시 실망 |
| 충족 방법 | 매 갈등에 전략 설계 → 실행 → 변수 대응 과정 |

### D그룹: 회귀물 피로 독자

| 항목 | 상세 |
|------|------|
| 특징 | 회귀물을 너무 많이 읽어서 지친 독자 |
| 유입 포인트 | "이건 회귀물이 아니라 빙의+다인격!" |
| 핵심 요구 | 새로운 구조. "또 회귀?"가 아니어야 함 |
| 주의사항 | 회귀물 클리셰 (미래 정보 독점) 절대 사용 금지 |
| 충족 방법 | 3인격이 미래를 모름. 이준혁의 경영 지식 = 현대 지식이지 미래 지식 아님 |

## 📈 독자 여정 설계 (Reader Journey Map)

### Phase 1: 발견 (1화 전)

| 접점 | 독자 행동 | 우리의 무기 |
|------|----------|-----------|
| 썸네일 | 3초 안에 클릭 결정 | "무협+경영" 키워드가 보이는 표지 |
| 제목 | 2초 안에 흥미 판단 | 제목 후보: "무림 M&A" "천하경영록" |
| 소개글 | 5초 안에 읽을지 결정 | "빚 3,000냥 소년, 무림 최대 기업 CEO가 되다" |
| 태그 | 장르 확인 | #무협 #경영 #3인격 #성장 #카타르시스 |

### Phase 2: 1화 체험 (이탈률 목표: 30% 이하)

| 시점 | 독자 심리 | 우리의 대응 |
|------|----------|-----------|
| 첫 문장 | "재밌겠다" or "뒤로가기" | "칼이 목에 닿는 순간, 두 개의 목소리가 들렸다." |
| 1분 후 | "캐릭터 괜찮네" | 3인격 첫 대화 → 즉시 차별화 인식 |
| 5분 후 | "설정 신선하다" | 경영학적 해석 첫 등장 |
| 10분 후 | "다음 화 클릭할까?" | 절단신공: 능력 각성형 |

### Phase 3: 10화 정착 (이탈률 목표: 50% 이하)

| 화 | 독자 심리 | 핵심 제공 |
|----|----------|----------|
| 1~3화 | "신선하다" | 3인격 + 경영 + 위기 → 차별화 확인 |
| 4~5화 | "어떻게 되지?" | 첫 M&A + 첫 반격 |
| 6~8화 | "강해!" | 천마신공 첫 사용 + 30인 전투 |
| 9~10화 | "계속 읽겠다" | 조직 성장 시작 + 성장의 쾌감 |

### Phase 4: 50화 결제 전환 (유료 전환 목표: 15%)

| 화 | 독자 심리 | 핵심 제공 |
|----|----------|----------|
| 40~49화 | "너무 답답!" (고구마) | 적의 방해, 배신, 위기 |
| 50화 | "시원하다!!!!" (사이다 폭발) | 천마 무쌍 → 방화범 응징 |
| 51~55화 | "더 보고 싶다" → 결제 | 소주 입성 → 더 큰 무대 |

### Phase 5: 100화 충성 독자 (충성 독자 전환 목표: 10%)

| 지표 | 독자 행동 | 의미 |
|------|----------|------|
| 댓글 참여 | "이준혁 사랑해" "천마 최고" | 캐릭터 팬덤 형성 |
| 추천 | "이거 꼭 읽어봐" 공유 | 바이럴 시작 |
| 유료 선독 | 최신 화 유료 결제 | 핵심 수익원 전환 |
| 재독 | 1화부터 다시 읽기 | 복선 찾기 재미 → 작품 깊이 인정 |

## 📊 플랫폼별 전략

### 카카오페이지 (1순위)

| 항목 | 전략 |
|------|------|
| 장점 | 최대 독자 풀, 기다무 시스템 |
| 최적화 | 1화 6,000~7,000자, 기다무 유도 절단신공 |
| 유료 전환 | 50화 무료 → 51화부터 유료 → 50화 끝에 최대 사이다 |
| 태그 | 무협, 경영, 성장, 액션, 빙의 |

### 네이버시리즈 (2순위)

| 항목 | 전략 |
|------|------|
| 장점 | 네이버 검색 노출, 요일별 연재 |
| 최적화 | 화, 목 연재 (경쟁 약한 요일 노림) |
| 차별화 | "경영 무협" 키워드 독점 |

### 리디 (3순위)

| 항목 | 전략 |
|------|------|
| 장점 | 마니아 독자층, 완결작 선호 |
| 최적화 | 300화 완결 후 일괄 업로드 |
| 차별화 | 완결 보장 = 리디 독자 신뢰 |

## 📋 독자 피드백 대응 가이드

### 예상 긍정 반응 & 강화 전략

| 예상 반응 | 강화 방법 |
|----------|----------|
| "3인격 대화 재밌다!" | 3인격 대화 빈도 유지 (1화 최소 3회) |
| "경영 부분 신선하다!" | 경영 분석 디테일 유지 |
| "천마 최고!" | 천마 무쌍 장면 정기 배치 (20화마다 1회) |
| "조칠 귀엽다!" | 조칠 코믹 장면 유지 |

### 예상 부정 반응 & 대응 전략

| 예상 반응 | 원인 | 대응 |
|----------|------|------|
| "경영 부분 지루하다" | 숫자/분석이 과다 | 경영 분석을 대화체로 전환 → 이준혁 vs 천마 논쟁으로 |
| "전투가 부족하다" | 경영 비중이 너무 높음 | 경영:전투 = 4:6 비율 유지 (전투 약간 많게) |
| "주인공 너무 먹사냥" | 고구마가 부족 | 패배/실패 에피소드 추가 (주인공이 지는 장면) |
| "로맨스 더 줘!" | 설화 등장 부족 | 설화 단독 에피소드 20화마다 1회 |
| "화산귀환 짝퉁?" | 차별화 인식 부족 | 경영 요소를 더 전면에 배치 |

## ⚠️ 독자 이탈 위험 구간 (Red Zone)

| 구간 | 이탈 위험 | 원인 | 방어 전략 |
|------|----------|------|----------|
| 1화 | ⭐⭐⭐⭐⭐ | 첫인상 실패 | 첫 문장 3인격 임팩트 |
| 3~5화 | ⭐⭐⭐⭐ | "설정만 많고 재미없다" | 5화까지 첫 M&A 완료 |
| 20~30화 | ⭐⭐⭐ | "패턴 반복" | 새 캐릭터(왕팔) + 새 무대 |
| 45~50화 | ⭐⭐⭐⭐ | 고구마 축적 한계 | 50화 사이다 폭발 (천마 무쌍) |
| 80~100화 | ⭐⭐⭐ | "중반 처짐" | 주화입마 → 대반전 (텐션10) |
| 140~150화 | ⭐⭐⭐ | "연애 때문에 무협이 느려짐" | 데이트+자객 기습 동시 진행 |
| 200~210화 | ⭐⭐ | "정점 찍고 내리막?" | 혈마 선전포고 → 새 위기 즉시 투입 |
