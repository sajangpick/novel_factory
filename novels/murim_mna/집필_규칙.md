## ★2. 3인격 엔진 핵심 요약 (상세: §7 참조)

> **몸은 100% 위소운의 것. 천마·이준혁은 머릿속 목소리일 뿐.**
> 감각(고통·배고픔·추위)은 3인 전원 100% 공유.
> 심상전장 안에서만 각자 독립된 몸을 가짐.

- **위소운**: 평어. 감정·정의감 담당. 거부권 보유. 몸의 주인.
- **이준혁**: 존댓말. 경영·요리·현대지식. 독자의 분신(감정 공명 엔진).
- **천마**: 반말(건방지고 짧다). 무력·무림정세. 코치 전용.
- 속삭임(매화 3~5회), 짧은 회의(1~2회), 정식 회의(10~15화마다), 긴급 지시(전투 시)

```
위소운 = 평어 (따뜻하고 단단함)
```

## ★5. 대사 품질 강제 — "정보"가 아니라 "성격"으로 말하라

> ⚠️ 대사는 정보 전달이 아니다. **캐릭터의 성격이 보여야 한다.**
> 대사만 뽑아서 이름을 지웠을 때, 누구 말인지 맞출 수 없으면 실패.

### 이준혁 대사 원칙
```
❌ 금지: '50문이면 5천 원입니다.' (계산기)
❌ 금지: '마진율이 15%입니다.' (보고서)
❌ 금지: '볶음밥의 핵심은 수분 조절이에요.' (교과서)
✅ 필수: 숫자 + 불만/감정/비교
  → '50문? 에어컨도 없는 나무판자가 지하철보다 비싸다고요?'
✅ 필수: 엘리트의 자존심 + 당혹감
  → '미슐랭에서 3년 일했는데, 저 칼질은 제 선배보다 빠릅니다. 시골 객잔에서요.'
✅ 필수: 직업병이 부적절한 순간에 터짐
  → '저 도적 열 명 조직화하면 물류 인력으로...... 아, 아닙니다.'
```

### 천마 대사 원칙
```
❌ 금지: '시끄러.' 2연속 사용 금지. 3화에 최대 1회
❌ 금지: '나쁘지 않네.' 3화 연속 사용 금지
❌ 금지: '밥이야? 먹어.' 같은 패턴 반복 금지
✅ 필수: 300년 비교 (꼰대 코미디)
  → '300년 전에는 뱃사공이 손님한테 절했어. 요즘 놈들은 버릇이 없어.'
✅ 필수: 무식한 척 날카로운 한 마디
  → '그 마진이라는 거, 결국 남는 게 많으면 좋다는 거 아니야?'
✅ 필수: 자존심 방어 코미디
  → '배멀미가 아니야. 이 몸이 약한 거지.'
```

### 위소운 대사 원칙
```
❌ 금지: 항상 여유로운 미소 + 완벽한 대응 (가끔 어색해야 함)
✅ 필수: 사교 서투름 (10년간 사람 못 만남)
  → "감사...... 합니다?" (이준혁이 가르쳐준 대로 하는데 어색)
✅ 필수: 용어 전이 (이준혁/천마 용어가 무의식적으로 나옴)
  → "이건 리스크가...... 아, 위험하다는 뜻이에요."
✅ 필수: 곽진 관련은 평소의 여유가 깨진다
```

## ★6. 코미디 쿼터 — 매 화 최소 7비트

> ⚠️ 코미디 없이 한 페이지 이상 지나가면 안 된다.
> 3인격 시스템은 코미디 금광이다. 10%가 아니라 100% 캐라.

```
매 화 코미디 최소 7비트:
① 3인격 갈등 코미디: 최소 2회 (의견 충돌, 동시 반응, 합체 실패)
② 문화 충돌 코미디: 최소 1회 (현대 vs 고대, 이준혁 용어 vs 천마 "뭐?")
③ 캐릭터 갭 코미디: 최소 2회 (위소운 사교 서투름, 천마 자존심, 이준혁 직업병)
④ 조연 리액션 코미디: 최소 1회 (당찬 질투, 장위 식탐, 마현 겁)
⑤ 러닝개그 진행: 최소 1회 (팬텀 습관, "300년 전에는~", 천마 용어 학습)
```

### 안 쓰면 안 되는 코미디 유형
- **합체 실패**: 이준혁 "웃으세요" → 위소운 웃는데 어색 → 상대방 이상하게 봄
- **밖에서 보면**: 3인격 토론 중 → 밖에서 보면 표정만 바뀌는 이상한 사람
- **동시 반응**: 같은 감각에 3인격이 완전히 다르게 반응
- **천마 꼰대**: "300년 전에는~" = "라떼는 말이야~"
- **이준혁 직업병**: 전투 직후에도 손해배상 계산

## ★8. 위소운 약점 강제 — 매 화 실패/곤란 1회 이상

> ⚠️ 완벽한 주인공은 재미없다. 위소운은 강하지만 결함이 있다.

```
매 화 최소 1회, 아래 중 택 1 이상:
① 사교 실패: 칭찬/질문/사교 상황에서 어색 (이준혁이 도와주거나 더 이상해지거나)
② 감정 흔들림: 곽진/화산파/장문인 관련 감정 표면화 (여유가 깨짐)
③ 혼자 판단 실패: 이준혁/천마 없이 곤란 (돈 잘못 계산, 사람 잘못 읽음)
④ 합체 실패: 조언대로 했는데 어색/실패 (머리에 천재 둘이어도 몸은 하나)
⑤ 물리적 한계: 무공은 초절정이지만 화경 미달 (실전 경험 부족, 극한 전투 시 한계)

검수: 위소운이 처음부터 끝까지 완벽하면 → 약점 장면 추가 필수
```

## ★9. 조연 활성화 — 배경 금지

```
■ 당찬: 매 화 최소 감정 표현 1회 (질투/경쟁/인정/좌절) + 독립적 행동 1회
■ 소연화: 매 화 능동적 행동 1회 (질문/제안/먼저 움직임) + 위소운 비밀 관찰 1회
■ 장위: 고정 개성 = 항상 배고픔. 음식 리액션 담당. 매 화 대사 최소 1회
■ 마현: 고정 개성 = 겁이 많음. 위기 리액션 담당. 매 화 대사 최소 1회
❌ 금지: 조연이 대사 없이 3화 연속 배경으로만 존재
```

## 🚫 실수 방지 목록 (Error Prevention Registry)

> **이 목록은 실제 집필 중 발견된 오류를 누적 관리한다.**
> 한 번 잡은 실수는 두 번 반복하지 않는다. 공장의 불량 방지 체크리스트와 같다.
> **매 화 집필 시(Step 5), 매 화 검수 시(Step 6)에 이 목록을 반드시 확인한다.**
> **새로운 오류 발견 시 즉시 이 목록에 추가한다.**

### EP-001: 몸 소유권 위반 — "이준혁/천마가 몸을 움직이는 묘사"
- **발견**: 1화 집필 중. "갈비뼈가 만져집니다", "손을 허리춤에 가져갔다", "주위를 둘러보았다"
- **규칙**: §7-0. 몸은 100% 위소운 것. 이준혁·천마는 목소리만.
- **금지 표현**: "만졌다", "손을 뻗었다", "둘러보았다", "걸었다", "일어섰다" 등 물리적 행동
- **허용 표현**: "느껴진다", "감각이 전해진다", "보였다(시야에 비친)", "뻗으려 했지만 움직이지 않았다"
- **특히 주의 — 팬텀 습관**: 이준혁이 폰/시계를 찾는 습관은 "~하려 했지만 손이 움직이지 않았다"로 써야 함. 실제로 손이 움직이면 안 됨.
- **체크법**: 이준혁/천마 주어 문장에 동작 동사가 있으면 → (1) 감각 동사로 교체 또는 (2) "~하려 했지만 안 됐다"로 교체

### EP-002: 천마 말투 — 반말(건방지고 짧다)
- **규칙**: 천마는 모든 존재를 아래로 본다. 반말만 사용. 건방지고 짧게.
- **금지 표현**: "~하시오", "~하겠소", "~할 것이오", "~하오" 등 하오체 전체 금지. 존댓말/경어 전체 금지.
- **허용 표현**: "~해", "~하라", "~이다", "~하네", "~해라", "~이야", "~거든" (반말)
- **대표 예시**: "먹어." "시끄러워." "죽여." "나쁘지 않네." "벌레들이네."
- **체크법**: 천마 대사에서 "~오", "~소", "~시" 어미 검색 → 반말로 교체

### EP-003: 서기(AD) 연도 사용 — "1100년대"
- **발견**: 1화 집필 중. 이준혁이 "1100년대입니다"
- **규칙**: 이 세계는 가상 세계. 서기(AD) 달력을 사용하지 않음. 중국식 연호(건문 3년 등)도 특정 왕조가 없으므로 사용 안 함.
- **금지 표현**: "1100년", "서기", "AD", 구체적 연도 숫자
- **허용 표현**: "아주 오래 전", "먼 미래에서 왔다", "전기가 없는 세상"
- **체크법**: 숫자+년 조합이 나오면 → 세계관 위반 확인

### EP-004: 현대 지식으로 시대 특정 — "건물 양식으로 연도 추정"
- **발견**: 1화 집필 중. 이준혁이 건물 양식을 보고 시대를 특정
- **규칙**: §6. 이준혁 지식 신뢰도. 역사/지리 = D등급(0%). "모릅니다."
- **금지 패턴**: 이준혁이 건축, 복식, 화폐로 정확한 시대를 추정하는 장면
- **허용 패턴**: "전기가 없다", "자동차가 없다", "사극에서 보던 그 세상" 등 현대인이 알 수 있는 수준의 판단
- **체크법**: 이준혁이 역사적 판단을 하면 → D등급(모름) 확인

### EP-005: 소설 본문에서 화수 직접 언급 금지
- **발견**: 4화 집필 중. "2화에서 건달한테 뺏은"
- **규칙**: 소설 본문에서 "N화에서" 형태의 직접 화수 참조 금지
- **금지 패턴**: "2화에서~", "지난 화에서~", "3화 전에~"
- **허용 패턴**: "며칠 전", "어제", "그때", "지난번에" 등 시간 표현으로 대체
- **체크법**: 본문에서 "화"라는 글자 검색 → 화수 참조가 아닌지 확인

### EP-006: 이준혁의 이 시대 정보 — 관찰/질문 필수
- **발견**: 9화 집필 중. 이준혁이 "마늘은 이 시대에도 있다", "마라가 없다", "술은 15도 한계" 등 확인하지 않은 정보를 단정
- **규칙**: 이준혁은 이 세계 신참(빙의 10일차~). 이 시대 정보를 알려면 반드시 ① 직접 관찰 ② 위소운에게 질문 ③ 천마에게 질문 중 하나를 거쳐야 한다
- **금지 패턴**: "이 시대에 OO이 있다/없다" (근거 없이 단정), "OO도 한계" (경험 없이 수치 제시)
- **허용 패턴**: "시장에서 OO을 봤다" (관찰), "위소운 님, OO이 있습니까?" (질문), 천마: "OO은 약재상에서 팔아" (천마 정보)
- **핵심**: 작가(AI)의 지식 ≠ 이준혁의 지식. 작가가 아는 것을 캐릭터에게 주입하지 말 것

### EP-007: 무대와 사건의 물리적 불일치 — "포구에 절벽" 문제
- **발견**: 1화 집필 중. 포구(항구)에서 30미터 절벽 추락 장면. 삼경 포구에서 짐꾼이 혼자 일하는 장면.
- **규칙**: 장소의 물리적 특성과 사건이 양립해야 한다. "극적 효과"를 위해 지형을 왜곡하면 안 된다.
- **오류 유형 3가지**:
  - **(1) 지형 불일치**: 포구(수면 높이) + 절벽(높은 곳) = 불가능. 포구에는 절벽이 없다.
  - **(2) 인원 불일치**: 삼경 포구 + 짐꾼 혼자 = 불가능. 항구에는 다른 인부/선원이 있다.
  - **(3) 범죄 환경 불일치**: 공공장소 + 살인 + 목격자 없음 = 이유 설명 필수.
- **금지 패턴**: 극적 효과를 위해 장소의 지형을 임의로 변조. 인원이 있어야 하는 곳에서 "혼자" 설정.
- **필수 패턴**: 사건에 절벽이 필요하면 → 절벽이 있는 장소를 선택. 혼자여야 하면 → 혼자인 이유를 명시.
- **체크법**: 🔒 "무대 물리 검증" 4단계(무대설계→인원검증→물리양립성→바보독자테스트)를 반드시 거칠 것
- **핵심**: "무대가 먼저, 사건이 나중." 사건에 무대를 끼워맞추지 말고, 무대에서 가능한 사건을 도출할 것
- **부수 효과**: 질문 과정이 3인격 대화를 자연스럽게 늘려주고, 천마가 "정보 덩어리" 역할을 하면서 캐릭터 매력도 상승

### EP-008: 버전 오염 — 구버전(v1/v2) 데이터가 현재 추적 파일에 남아있는 문제
- **발견**: 7화 집필 준비 중. 대시보드 3인격 상태가 "빙의 직후, 무인도 해변"(v1), 기억카드에 구버전 1화/5화만 존재, 떡밥 관리표에 v3 실제 떡밥 미반영
- **원인**: v2→v3 구조 변경 시 본문은 새로 썼지만 추적 파일 7개 중 일부만 업데이트됨. 여러 AI 세션이 각각 다른 파일만 수정

## 6. 이준혁 지식 개연성 시스템 (Knowledge Plausibility Engine)
- **전생 이력**: 천재(12세 대학) → 맥킨지 10년(15~25세) → 요리학원 1년 → 미슐랭 3스타 2년 → 치킨 300호점 CEO(28~33세) → 33세 과로사
- **혁신의 동기**: ⭐ **모든 혁신은 "욕구불만"에서 시작한다**
  - 증류주 = "술이 밍밍해서" / 비누 = "몸이 가려워서" / 마라면 = "매운 게 먹고 싶어서"
  - "세상을 바꾸겠다" ❌ → "그냥 먹고 싶다 / 불편하다" ✅ = 독자 공감의 핵심
  - 욕구불만 → 해결 시도 → **반드시 실패** → 재시도 → 성공 → (나중에) 사업화
  - **"천화루(天火樓)"**: 이준혁의 음식 혁신이 모인 프리미엄 식당 브랜드
  - ⭐ **핵심 차별화**: 이준혁은 **요리를 직접 할 수 있다** (미슐랭 3스타 출신). 레시피를 알려주기만 하는 게 아니라 **직접 칼을 잡고 요리한다** = 독자에게 강한 몰입감
- **지식 신뢰도 등급**: 모든 이준혁 지식 사용 시 반드시 등급 적용
  - **S등급** (99%): "확실합니다." → 재무, 회계, M&A, 전략, **요리** (전공)
  - **A등급** (80%): "프로젝트에서 배운 적이 있습니다." → 증류, 보험, 프랜차이즈, 물류, 식품공학
  - **B등급** (50%): "어디선가 읽은 기억이 있습니다만..." → 농업, 위생, 역사
  - **C등급** (30%): "이론적으로는... 검증이 필요합니다." → 제조, 과학, 의학
  - **D등급** (0%): "모릅니다." → 무공, 지리, 무림 정세
- **필수 규칙**:
  - ⚠️ S등급만 "확실합니다" 사용 가능. A~C는 불확실성 표현 필수
  - ⚠️ 10화마다 1회 이상 이준혁이 "틀리는 장면" 또는 "모르는 장면" 삽입
  - ⚠️ 지식 출처를 대사로 자연스럽게 밝힐 것 ("맥킨지에서..." "미슐랭에서..." "프로젝트에서...")
  - ⚠️ 천마 반응 필수: 무시 → 호기심 → 인정 (캐릭터 성장 + 코미디)
  - ⚠️ 요리 장면에서는 **프로답게**: 온도, 불 조절, 칼 기술, 향 판단 등 전문가 묘사 필수
- **상세 데이터**: `@이준혁_현대지식_DB.md` 참조 (욕구불만 시스템 + 프로젝트 이력)

> **몸은 100% 위소운의 것이다. 천마와 이준혁은 절대로 몸을 조종할 수 없다.**

- 천마·이준혁은 **머릿속 목소리**일 뿐. 조언, 지시, 지도만 가능
- ⚠️ **유일한 예외 — 심상전장(§7-8)**: 심상전장 안에서는 3인이 **각자 독립된 육체**를 가짐. 현실에서는 절대 불가
- 감각(고통, 배고픔, 추위, 쾌감)은 **3인 전원이 동시에 100% 공유**
- 전투: 천마가 "왼쪽!" 외쳐도, 위소운이 직접 몸을 움직여야 함. 반응이 늦으면 맞음
- 요리: 이준혁이 "불 줄여!" 해도, 위소운이 직접 해야 함. 서투르면 태움
- 협상: 이준혁이 대사를 알려줘도, 위소운이 직접 말해야 함. 어색하면 실패
- **이것이 만드는 긴장감**: 머릿속에 천재가 둘 있어도, 실행은 위소운 → 독자가 안심 못 함
- **이것이 만드는 성장 서사**: 위소운의 검술/요리/화술이 점점 나아지는 게 보임
- **이것이 만드는 코미디**: 천마 "왼쪽이라고 했잖아!" 위소운 "왼쪽으로 갔잖아요!" 천마 "그건 오른쪽이야!"
- ❌ **금지**: 천마가 몸을 인수하는 장면, 눈 색깔이 변하는 연출, 인격 교대 전투 — 전부 금지

### 7-1. 3인격 동기 시스템 (Motivation Engine)

각 인격은 **"지금 당장 해결해야 하는 절박한 문제"**가 있다. 거창한 목표가 아니다.

| 인격 | 지금 상태 | 1순위 목표 | 한 마디 |
|------|----------|-----------|--------|
| **위소운** | 흑호단에 추락당해 "사망"→무인도 생존 | **복수 + 보호** (나를 죽이려 한 놈들 + 서구진 사람들) | "돌아간다. 아직 내가 지켜야 할 사람들이 있어." |
| **이준혁** | 1100년에 빙의→무인도에서 깨어남→모든 게 불편 | **편하게 살기** (불편 해소) | "일단 살아서 나갑시다. 비누는 그 다음입니다." |
| **천마** | 300년 전 누명+배신→위소운의 체질에서 가능성 발견 | **억울함 풀기** (설욕) + 제자 양성 | "이 그릇... 키울 만하네." |

- ⚠️ 위소운의 목표는 **"복수"와 "보호"** 양쪽이다. "나를 밀어 떨어뜨린 놈들"과 "내가 지키지 못한 사람들"
- ⚠️ 이준혁의 혁신은 "세상을 바꾸겠다"가 아니라 **"신라면이 먹고 싶다"**에서 시작한다
- ⚠️ 천마의 목표는 "천하재패"가 아니라 **"억울함을 푸는 것"** + 위소운이라는 제자를 키우는 것
- ⚠️ **최대 제약**: 천마는 현경급이지만 위소운의 몸으로는 10%도 발휘 못 함 → 코치로서 가르치는 것이 최선

**강제 동거의 논리**: 좋아서가 아니라 **서로가 필요해서** 함께한다.
- 위소운이 죽으면 → 이준혁은 손가락 하나 못 움직이고, 천마는 복수 영원히 끝
- 이준혁이 없으면 → 위소운은 밥을 못 벌고, 천마는 세력을 못 키움
- 천마가 없으면 → 위소운은 무림에서 하루도 못 살고, 이준혁은 보디가드가 없음

### 7-2. 감각 공유 시스템 (Sensory Sharing)

> ⚠️ **절대 규칙**: 고통, 배고픔, 추위, 피로 등 **모든 신체 감각은 3인격 전원이 동시에 느낀다.**
> 머릿속에서 편하게 구경하는 인격은 없다. 셋 다 같이 고통받고, 같이 행복하다.

- **고통 100% 공유**: 위소운이 맞으면, 이준혁도 천마도 똑같이 아프다
- **배고픔 100% 공유**: 밥을 안 먹으면 셋 다 굶는다
- **쾌감 100% 공유**: 맛있는 음식을 먹으면 셋 다 행복하다
- **내면 공간 반영**: 배고프면 안개, 통증이면 지진, 배부르면 햇살

**AI 집필 필수 규칙**:
- 배고플 때 → 3인격 모두의 반응 최소 한 줄씩 삽입
- 다칠 때 → 나머지 두 인격도 고통을 느끼는 묘사 필수
- 맛있는 음식 → **3인격 모두 행복한 장면** = 이 소설의 감동 공식
- 천마 배고픔 반응 필수: "내가... 배가 고프다고...?!" (코미디 + 인간화)
- 이준혁 고통 반응 필수: 컨설턴트가 데이터가 아닌 감각으로 괴로워하는 장면

### 7-3. 의사결정 시스템 (Decision Protocol) — 4가지 형태

모든 결정에 3인격의 시각이 충돌한다. 상황 긴급도에 따라 4형태 중 택 1:

**① 속삭임** (3~5줄, 매화 3~5회) — 일상적 판단
- 몸은 위소운이 움직이면서, 머릿속에서 한마디씩 끼어드는 형태
- 장면 흐름을 끊지 않음. 개그 + 캐릭터 확인 역할
- 표기법: 소괄호 (()) — 독백/내면 대화는 반드시 ()로 표기. 작은따옴표('') 사용 금지.
- 예: `(저건 양고기입니다. 서역 상인이죠.) 이준혁이 중얼거렸다.` → `(음식보다 약재상.) 천마가 잘랐다.`
- ⚠️ 작은따옴표('')는 간판·이름·별호 등 고유명사 인용에만 사용: 예) '무림객잔' 편액, '위소검'이라는 이름

**② 짧은 회의** (10~15줄, 매화 1~2회) — 전술적 판단
- "눈을 감았다" = 회의 시작 신호. 밖에서는 "잠시 생각에 잠긴" 것처럼 보임
- 3인격이 각자 의견 → 위소운이 최종 판단
- 예: 협상 직전 작전 수립, 위험 인물 판단

**③ 정식 회의** (30줄+, 10~15화마다 1회) — 전략적 대결정
- 내면 공간(매화나무 마당)으로 장면 전환
- 사업 방향, 동맹, 전면전 결정 등 큰 분기점에서만 사용
- 내면 공간 날씨가 몸 상태를 반영 (배고프면 안개, 위기면 폭풍)

**④ 긴급 지시** (3~5줄, 전투/위기 시) — 생존 우선 즉각 지시
- ⚠️ **절대 규칙: 몸은 100% 위소운 것. 천마·이준혁은 절대로 몸을 조종할 수 없다.**
- 천마가 전투 본능으로 즉각 지시를 내림. 위소운이 직접 몸을 움직여 실행
- 위소운의 반응 속도와 실력이 부족하면 → 지시가 있어도 맞을 수 있음 = 긴장감의 핵심
- 예: `칼이 날아왔다. (왼쪽!) 천마가 외쳤다. 위소운이 몸을 틀었다. 늦었다. 어깨를 스쳤다.`
- ⚠️ 천마가 직접 할 수 없는 답답함 = 위소운이 직접 강해져야 하는 이유

### 7-4. 시간표 시스템 (Activity Schedule)

> ⚠️ **절대 규칙: 몸은 항상 위소운이 움직인다. "담당"은 누가 조언/지도하는지를 뜻함.**
> ⚠️ **적용 시점**: 서구진 귀환 후(26화~). 무인도 수련기(1~25화)에는 심상전장 수련 리듬(§7-8)이 우선.
> 26화경 이준혁이 제안. "활동 우선순위 시간표" = 러닝 개그 + 성장 장치.

| 시간 | 배정 | 조언/지도 | 실행 |
|------|------|----------|------|
| 묘시~진시 (05~09) | 무공 수련 | 천마 지도 | **위소운이 직접 수련** |
| 사시~미시 (09~13) | 사업/협상 | 이준혁 조언 | **위소운이 직접 말하고 행동** |
| 신시~유시 (15~19) | 자유/일상 | 위소운 자유 | **위소운 단독** |
| 술시~해시 (19~21) | 저녁+정보수집 | 공동 논의 | **위소운이 직접 행동** |
| 자시~인시 (23~05) | 수면 | 내면 회의 | 수면 중 |

- ⚠️ **핵심**: 천마가 "검을 이렇게 잡아라" 가르쳐도 위소운 손이 서투르면 안 됨 = 성장 서사
- ⚠️ **핵심**: 이준혁이 "여기서 웃으세요" 조언해도 위소운이 어색하면 실패 = 코미디 + 성장
- ⚠️ **시간표는 매번 깨진다** → 사건이 터져서 수련 시간에 사업, 사업 시간에 전투
- ⚠️ 시간표가 깨질 때마다 → 천마 분노 = 러닝 개그
- ⚠️ 시간표 갈등이 해소되는 과정 = 3인격 성장의 척도

### 7-5. 수면/각성 규칙 (Sleep/Wake Protocol)

> 3인격은 **각자 독립적으로 잠들고 깨어난다.** 항상 셋이 같이 깨어 있는 건 아니다.

- **위소운 수면 시**: 몸 못 움직임. 이준혁+천마만 내면 대화 (중재자 없는 직접 충돌)
- **천마 수면 시**: 위소운+이준혁만 대화. "주먹" 없이 솔직한 감정 교류 가능
- **이준혁 수면 시**: 위소운+천마만 대화. "계산기" 없이 원초적 공감 가능
- **혼자 각성**: 가장 깊은 독백/회상 장면 (25~30화마다 1회)
- **전원 수면**: 최대 위험. 습격 시 천마가 먼저 깨어남 → 위소운을 깨워서 대응 (천마가 직접 몸을 움직이는 것은 불가)

**AI 필수 규칙**:
- ⚠️ 잠든 인격을 깨우지 않으려고 **속삭이며 다투는 장면** = 코미디 + 친밀감
- ⚠️ 각 2인 조합의 대화 톤은 **반드시 다르게** (§상세: `@캐릭터_인명록.md` 수면/각성 규칙)
- ⚠️ 5~8화마다 1회 이상 **2인 각성 장면** 삽입 (관계 발전용)
- ⚠️ **이준혁 꿈 시스템**: 3~5화마다 이준혁 음식/편의 꿈 → 깨어난 후 3인격 반응 + 사업 아이디어 연결
  - 꿈 유형: 음식(가장 자주), 편의, 미슐랭, 300호점(감동), 혼합(코미디)
  - 꿈 → 사업 공식: 꿈 → 그리움 → "비슷하게 만들 수 있을지도" → 실패 → 성공 → 사업화
  - ⭐ 200화 이후 "꿈을 안 꿈" = 이 세계에 정착 완료의 감정 지표
  - §상세: `@캐릭터_인명록.md` "이준혁의 꿈 시스템"

### 7-6. 무공 목표 충돌 (Martial Goal Conflict)

| 인격 | 무공 목표 | 이유 | 대표 대사 |
|------|----------|------|----------|
| 위소운 | **화경** | 지킬 수 있으면 충분 | "화경이면 돼. 지킬 사람이 있으니까." |
| 이준혁 | **관심 없음** | 돈으로 호위 고용 가능 | "수련 시간에 장사하면 100냥입니다." |
| 천마 | **현경** | 천마의 자존심 | "고작 화경? 내 무공을 화경 따위에?" |

- ⚠️ 이 충돌이 **300화를 관통하는 갈등 축**
- ⚠️ 천마가 "현경" 고집 → 위소운이 "화경이면 충분" 반박 → 이 소설 최강의 내면 갈등
- ⚠️ 이 갈등이 풀리는 순간 = 독자 감동의 정점

| 26~75화 | 시간표 — 불완전한 민주주의 | 천마가 처음 "타협"을 배움 |

### 7-8. 심상전장 시스템 (Mental Battlefield — 心象戰場)

> ⚠️ **이 시스템은 무인도 수련의 핵심이자, 이후 300화 전체의 성장 기반이다.**
> 참조: 절대회귀 풍천교주 방식 + "삶이 바뀌어야 무학이 바뀐다" 철학

#### 기본 규칙
- **생성자**: 천마 (현경급만 가능한 정신 세계 조작술)
- **장소**: 3인격의 내면 공간을 확장하여 생성
- **시간 배율**: **10배** (현실 1분 = 심상 10분)
- **현실감**: 실전과 **100% 동일**. 지형, 날씨, 온도, 중력 모두 현실과 같음
- **통증**: **100% 전달**. 베이면 진짜 아프다
- **사망**: **심상전장에서 죽으면 현실로 강제 배출**. 다시 들어가려면 정신력 회복 필요
- **환경 연출**: 천마가 자유롭게 생성 (화산 매화나무 숲, 전장, 객잔, 서구진 시장 등)
- **3인 육체**: 심상전장 안에서 3명 모두 **자기 몸을 가짐** (현실과 다른 핵심 특징)

#### 심상전장 핵심 제한 — "깊이(Depth)" 시스템
- **⚠️ 심상전장의 지속 시간은 수련 깊이·내공에 따라 결정된다**
- **초반(내공 미약)**: 현실 **30분** = 심상 **300분(5시간)**이 한계
- **중반(내공 성장)**: 현실 **1~2시간** = 심상 **10~20시간**까지 가능
- **후반(내공 성숙)**: 현실 **4~6시간** = 심상 **40~60시간**(2.5일)까지 가능
- **하루 횟수 제한 없음**: 정신력이 허락하면 반복 진입 가능 (단, 사망 배출 후 회복 필요)
- **내공도 증진됨**: ⚠️ **심상전장 수련은 현실 수련과 100% 동일한 내공 성장 효과**를 가진다
- **초절정 이상은 심상전장으로 불가**: "이 이상은 실전에서만 깨우칠 수 있어" (천마)
- **천마 소모**: 심상전장 유지에 천마의 정신력 소모 → 천마도 쉬어야 함

#### 무인도 수련 타임라인 — "섬에 갇힌 날들"
> 위소운은 서구진 포구에서 추락 후 태호 무인도에 표류하여, 약 **현실 8~10개월** 간 수련한다.

**[Phase 1] 생존기 (1~15일)** — "살아남기"

| 트리거 | 감각 | 소환되는 서울 | 이준혁 반응 | 지속 시간 |

| 고기 굽는 냄새 | 후각 | 강남 고깃집. 소주잔. 동료들 웃음소리. | 3초간 멈춤 → "...아무것도 아닙니다." | 초반: 30초 / 후반: 3초 |
| 아이 웃음소리 | 청각 | 조카. 마지막으로 본 게 설날. | 가슴이 조임. 말이 끊김. | 초반: 1분 / 후반: 눈만 감음 |
| 비 오는 저녁 | 시각 | 회사 앞 편의점. 우산 없이 뛰던 날. | 처마 밑에서 멍하니 비를 봄. | 초반: 길게 / 후반: 미소 |
| 따뜻한 국물 | 미각 | 어머니 된장찌개. 퇴근 후 집밥. | "...비슷합니다." (눈이 젖음) | 전 기간 |
| 밤하늘 별 | 시각 | 서울에서는 별이 안 보였는데 | "서울에서는..." → 여기가 더 예쁘다는 깨달음 | 100화 이후 전환점 |

**AI 집필 필수 규칙**:
- ⚠️ **3~5화마다 1회**, 이준혁 감정 트리거 장면 삽입 (초반일수록 자주)
- ⚠️ 트리거 후 반드시 **위소운과 천마의 반응** 포함:
  - 위소운: 이준혁의 슬픔을 감각으로 느낌. "이준혁 님... 괜찮으세요?" (따뜻한 공감)
  - 천마: 처음엔 무시 "쓸데없어." → 중반 이후 침묵 (이해하기 시작)
- ⚠️ **트리거의 변화가 성장 척도**:
  - 초반: 서울이 그립다 (고통)
  - 중반: 서울과 여기를 비교한다 (정리)
  - 후반: 서울을 미소로 떠올린다 (평화)
  - 200화+: 트리거가 와도 흔들리지 않는다 (정착)

#### F. 대비 효과 — 위소운·천마는 왜 덜 힘든가

> 이준혁의 고통이 더 날카롭게 느껴지려면, 나머지 둘이 "덜 힘든 이유"가 명확해야 한다.

| 인격 | 이 상황이 나쁘지 않은 이유 | 이준혁과의 대비 |
|------|------------------------|--------------|
| **위소운** | 이 시대 출신. 짐꾼 생활 2년, "서구진의 바보"로 불림. **지금이 인생 최고의 시기.** 강해지고, 동료 생기고, 머릿속에 가르쳐주는 사람이 생김. | 위소운이 "오늘 물고기 잡았다!" 기뻐할 때, 이준혁은 "양념이 없잖아..." 우울 |
| **천마** | 300년간 혼으로 떠돌며 아무도 말을 안 걸어줌. **지금은 말할 상대가 있다.** 몸은 못 쓰지만, 대화가 가능. 게다가 위소운을 키우면 복수가 가능해짐. | 천마가 "이 정도면 됐어" 할 때, 이준혁은 "이게 됐다고요?!" 분노 |

**AI 집필 필수 규칙**:
- ⚠️ **같은 상황에서 3인의 반응 온도차**를 보여줄 것
- ⚠️ 위소운이 행복할 때 이준혁이 쓴웃음 → 독자: 가슴 아픔
- ⚠️ 천마가 "사치야"라고 말할 때 이준혁 "사치가 아니라 기본이에요!" → 독자: 공감+웃음
- ⚠️ **반대로 이준혁이 기뻐하는 순간**(천화면 완성, 비누 성공)에 위소운·천마도 같이 기뻐하면 → 독자 감동 극대화

#### G. 이준혁 감정 삽입 빈도 규칙

> 모든 화에 이준혁의 "현대인 감정"이 반드시 들어가야 한다. 비중은 화수에 따라 조절.

| 화수 | 팬텀 습관 | 감정 트리거 | 현대 비교 대사 | 서울 꿈 | 총 삽입 |
|------|----------|-----------|-------------|--------|--------|
| 1~25화 | 매화 2~3회 | 매화 1회 | 매화 3~5회 | 3화마다 | **매화 7~10회** |
| 26~75화 | 매화 1~2회 | 3화마다 1회 | 매화 2~3회 | 5화마다 | **매화 4~6회** |
| 76~150화 | 5화마다 1회 | 5화마다 1회 | 매화 1~2회 | 10화마다 | **매화 2~3회** |
| 151~200화 | 10화마다 1회 | 10화마다 1회 | 매화 0~1회 | 거의 안 꿈 | **매화 1~2회** |
| 200~300화 | 소멸 | 와도 미소 | 비교 대신 감사 | 안 꿈 | **매화 0~1회 (감사)** |

- ⚠️ **빈도가 줄어드는 것 자체가 성장 서사.** 독자는 이준혁이 적응해가는 걸 "삽입 빈도 감소"로 체감.
- ⚠️ **200화 이후 갑자기 팬텀 습관이 돌아오면** = 위기 신호. 흔들리고 있다는 뜻.

#### H. 금지 사항

```
❌ 이준혁이 1화부터 "이 세계도 나쁘지 않네" 하는 것 = 금지. 최소 25화까지는 지옥.
❌ 이준혁이 불편함 없이 적응하는 것 = 금지. 적응은 "불편함을 견디는 것"이지 "불편함이 사라지는 것"이 아님.
❌ 위소운이 이준혁의 고통을 모르는 것 = 금지. 감각 공유니까 위소운도 이준혁의 우울을 느낀다.
❌ 천마가 이준혁을 100화까지 계속 무시만 하는 것 = 금지. 50화경부터 "왜 저리 힘들어하는 거야?" 호기심 시작.
❌ 그리움 장면을 "슬프다"로만 끝내는 것 = 금지. 반드시 구체적 감각(냄새, 맛, 소리, 온도)으로 묘사.
❌ 팬텀 습관을 잊고 안 쓰는 것 = 금지. 체크리스트로 관리.
```

#### J. 이해의 다리 — 위소운·천마가 이준혁을 이해해가는 과정

> ⚠️ **이것은 3인격 관계 성장의 핵심 축이다.**
> 처음에 위소운·천마는 이준혁의 말도, 행동도, 고통도 이해 못 한다.
> 1,000년 후의 세상 이야기를 매일 듣다 보면 — 조금씩, 아주 조금씩 알게 된다.
> **이해 = 사랑의 시작.** 이 과정 자체가 독자를 울리는 서사다.

##### J-1. 위소운의 이해 궤적 — "이준혁 님의 세상은..."

> 위소운은 **공감 능력**이 뛰어나다. 이해는 못 해도 감정은 먼저 느낀다.
> 그래서 "말은 못 알아듣지만, 이 사람이 아픈 건 안다"가 먼저 온다.

| 시기 | 단계 | 위소운의 상태 | 대표 대사/장면 |
|------|------|-------------|-------------|
| 1~10화 | **못 알아듣는다** | 이준혁이 뭐라 하면 멍함. 단어 자체가 외계어. | "마... 마진? 그게 뭐예요?" |
| 10~25화 | **아픈 건 안다** | 말은 못 알아듣지만 감각 공유로 이준혁의 우울을 느낌. 가슴이 답답해지면 "이준혁 님이 또..." | "무슨 말인지는 모르겠는데... 이준혁 님이 슬픈 건 알아요." |
| 25~50화 | **단어가 귀에 붙는다** | 매일 듣다 보니 자주 쓰는 단어의 뜻을 어렴풋이 파악. "이익", "투자", "위생" 같은 단어. | "이익이라는 게... 남는 돈 말씀이시죠?" |
| 50~100화 | **세상이 궁금해진다** | 이준혁의 세상에 호기심. 밤에 조용히 질문함. "그 '커피'라는 건... 어떤 맛이에요?" | "이준혁 님 세상에는... 모두 따뜻한 물로 씻어요?" (경이) |
| 100~150화 | **자발적으로 쓴다** | 이준혁 용어를 자기 말처럼 사용. 이준혁이 놀람. | 협상 중: "이건... 마진이 안 맞아요." → 이준혁: '...!' (감동) |
| 150~200화 | **이준혁 없이도 생각한다** | 문제가 생기면 "이준혁 님이라면 어떻게 했을까?" 자동으로 떠올림. | "이건 ROI가... 아, 제가 왜 이 말을 쓰고 있지." (웃음) |
| 200화+ | **두 세계를 잇는다** | 이준혁의 세상과 자기 세상, 양쪽 모두를 이해하는 유일한 사람. | "이준혁 님의 세상도 좋지만... 여기도 괜찮잖아요." |

##### J-2. 천마의 이해 궤적 — "그 놈의 세상이..."

> 천마는 **자존심** 때문에 절대 먼저 인정 안 한다. 무시 → 귀에 들어옴 → 호기심 → 몰래 사용 → 들킴 → 인정.
> **천마가 이준혁 용어를 쓰는 순간 = 300화 전체에서 가장 웃기고 가장 감동적인 장면.**

| 시기 | 단계 | 천마의 상태 | 대표 대사/장면 |
|------|------|-----------|-------------|
| 1~15화 | **무시** | 이준혁 말을 헛소리 취급. 관심 자체가 없음. | "미친 놈 헛소리야. 무시해." |
| 15~30화 | **귀에 들어옴** | 무시하려는데 자꾸 들림. 특히 숫자와 전략 이야기가 걸림. | "...그래서 그 '마진'이라는 게 뭔데?" (투덜) |
| 30~50화 | **인정은 안 하지만 듣는다** | 이준혁의 경영 전략이 실제로 먹힐 때, 속으로 감탄. 겉으로는 절대 안 함. | 속으로: (쓸모 있는 놈이긴 해.) 겉: "운이 좋았을 뿐이야." |
| 50~75화 | **호기심** | "왜 저리 힘들어하는 거야?" 이준혁의 고통에 처음 관심. | "네놈 세상에서는... 모든 사람이 따뜻한 물로 씻어?" (놀라움을 숨기며) |
| 75~125화 | **몰래 사용** | 이준혁 용어를 슬쩍 쓰기 시작. 들키면 부인. | "이건 '투자 가치'가... 아니, 그러니까, 이득이 있다는 거야." |
| 125~175화 | **들킴** | 이준혁이나 위소운이 알아챔. 천마 당황. | 이준혁: (방금 마진이라고 하셨습니까?) 천마: "...내가 언제 그런 천한 말을!" |
| 175~225화 | **인정** | "네놈 세상은 좋은 곳이었어." 이준혁의 세상을 처음으로 긍정. | "1000년 후의 인간들이... 그렇게 살았다고. 나쁘지 않네." |
| 225~300화 | **자기 투영** | 이준혁의 그리움을 자기 감정에 대입. "나도... 그리운 게 있어." | "네놈만 잃은 게 아니야. 나도... 300년 전의 것들이 있어." |

##### J-3. 이준혁 용어 학습 시스템

> 위소운·천마가 점진적으로 배우는 이준혁의 현대 용어/개념 목록.
> **AI는 해당 화수에서 "아직 모르는 단어"를 자연스럽게 사용하면 안 된다.**

| 용어 | 뜻 | 위소운 이해 시점 | 천마 이해 시점 | 학습 계기 |
|------|---|---------------|--------------|----------|
| **마진** | 이익률 | 30화~ | 50화~ | 천화면 장사 시작 후 매일 듣다 보니 |
| **투자** | 미래를 위한 지출 | 25화~ | 40화~ | 이준혁이 "돈을 써야 돈을 번다" 설명 반복 |
| **위생** | 청결/건강관리 | 20화~ | 60화~ (비누 쓰면서) | 비누 만든 후 효과를 직접 체감 |
| **효율** | 적은 힘으로 많은 결과 | 35화~ | 45화~ | 조직 운영에서 효과 직접 목격 |
| **ROI** | 투자 대비 수익 | 100화~ | 150화~ | 사업이 커진 후에야 개념 체득 |
| **브랜드** | 이름의 가치 | 80화~ | 100화~ | 천화루 이름이 퍼지는 걸 보면서 |
| **프랜차이즈** | 같은 가게를 여러 개 | 75화~ | 90화~ | 소주 2호점 오픈하면서 |
| **리스크** | 위험 | 40화~ | 30화~ (천마가 더 빠름) | 전투적 판단과 겹치는 개념이라 |
| **커피** | 쓴 각성 음료 | 10화~ (단어만) | 50화~ (개념) | 이준혁이 매일 아침 한숨 쉬어서 |
| **삼겹살** | 돼지 뱃살 구이 | 25화~ | 45화~ | 화로구이 만들면서 설명 |

**AI 필수 규칙**:
- ⚠️ 10화의 위소운이 "ROI"를 말하면 안 된다. 아직 모르는 단어.
- ⚠️ 30화의 천마가 "브랜드"를 쓰면 안 된다. 아직 이해 전.
- ⚠️ **새 용어를 처음 이해하는 장면** = 반드시 작은 에피소드로 묘사. 갑자기 알면 안 됨.
- ⚠️ 용어를 처음 사용하는 장면에서 **이준혁의 반응** 필수:
  - 위소운이 쓰면: 이준혁 감동 (따뜻함)
  - 천마가 쓰면: 이준혁 놀람 → 감동 (+ 코미디. 천마가 부인하니까)

## 9. 소설체 집필 강제 규칙 — 상세 예시집 (★1의 상세)

> ⚠️ **7대 규칙과 금지 목록은 파일 최상단 ★1 참조. 여기는 예시와 검수 전용.**

### 📖 규칙별 좋은 예시 (집필 시 참고)

**규칙 1 예시 — 한 장면 3~5문단:**
> 차밭 사이를 걷는 동안, 바람이 낮은 관목 위를 훑고 지나갔다. 초록이 겹겹이었다. 짙은 것, 연한 것, 햇살을 받아 금빛이 도는 것. 그 사이사이에서 올라오는 향이 코끝에 걸렸다. 풀과 볕이 섞인, 이름 모를 향. 이준혁이라면 '녹차 라떼'라고 했을 것이다. 위소운은 그냥 '좋은 냄새'라고 생각했다. 천마는 아무 말도 안 했다. 배가 불렀기 때문이다.

**규칙 2 예시 — 대사 전후에 "살":**
> 소연화가 찻잔을 들어 위소운의 잔에 기울였다. 차가 흐르는 소리가 작았다. 졸졸. 그 소리 사이로 "위 공자, 한 잔 더 드세요"라는 말이 올라왔다. 위소운이 잔을 받으며 고개를 끄덕였다. "고맙습니다." 당찬이 자기 잔을 혼자 채웠다. 아무도 따라주지 않는 차는 — 같은 차인데 맛이 달랐을 것이다.

**규칙 3 예시 — 3인격을 행동에 녹이기:**
> 위소운의 손이 주머니 쪽으로 갔다. 습관이었다. 3년이 지나도 고쳐지지 않는, 스마트폰을 찾는 손. 빈 주머니의 감촉이 손끝에 닿았을 때 — 이준혁의 한숨이 가슴 안쪽에서 올라왔다. (사진 찍고 싶은 거예요, 저.) 위소운이 슬쩍 손을 뺐다. 천마가 물었다. (사진?) 그 단어를 설명하려면 한 시진은 걸릴 것이다. 위소운은 설명 대신 차밭을 한 번 더 보았다. 눈으로 찍었다.

**규칙 4 예시 — 장면 전환은 시간의 흐름으로:**
> "해가 기울면서 그림자가 길어졌다. 의흥현의 낮은 성벽이 보이기 시작했다."
- `---`는 기승전결 경계에서만. 그 외 장면 전환은 시간·날씨·빛·이동으로 자연스럽게 연결

**규칙 5 부연 — 감정은 "증거"로:**
- 독자가 스스로 감정을 읽어내게 하라. 작가가 "슬프다"고 써주면 감동이 절반으로 줄어든다
- ❌ "위소운은 슬펐다." (설명) → ✅ "찻잔을 쥔 손가락이 하얘졌다. 표정은 변하지 않았다." (증거)

**규칙 6 부연 — 문단은 호흡:**
- 짧은 문단 = 긴장감 (전투, 충격, 절단신공)
- 긴 문단 = 몰입감 (풍경, 감정, 회상)
- 전체가 짧으면 = 시나리오. 전체가 길면 = 지루함
- **리듬을 섞어라.** 긴-긴-짧-긴-짧짧-긴. 음악처럼.

**규칙 7 예시 — 오감 겹치기:**
> 가마 연기가 하늘로 올라갔다(시각). 흙이 불에 타는 냄새(후각)와 장인의 물레가 돌아가는 소리(청각)가 도자거리를 채웠다.

### ⚠️ 검수 시 추가 체크 (Step 6 전용 — ★1 체크리스트와 별도)

```
□ 본문을 소리 내어 읽었을 때 "영화 내레이션"처럼 들리면 → 시나리오체. 재작성
□ 본문을 소리 내어 읽었을 때 "소설을 읽는 느낌"이 들면 → 통과
□ 가장 중요한 장면이 500자 미만이면 → 부족. 보강
□ 대사를 뽑아서 대본 형식으로 놓았을 때 본문과 차이가 없으면 → 살 부족. 보강
```

## ★10. 3인격 소통 그라디언트 — "8년차 동거" 원칙

> ⚠️ **3인격은 7년간 한 몸에서 살았다. 소통 방식이 이를 반영해야 한다.**
> "말해야 아는 사이"가 아니라 "안 말해도 아는 사이"를 보여줘야 한다.
> 단, 독백을 전부 빼면 코미디와 캐릭터가 죽는다. **3단계로 구분한다.**

### 소통 3단계 — "말하는 건 의견이 다를 때만"

| 단계 | 상황 | 표현 방식 | 예시 |
|------|------|----------|------|
| **1. 반사적** | 습관, 일상, 자동 분석 | 독백 없이 위소운의 행동으로 | 손이 허리춤으로 갔다. 스마트폰이 있어야 할 자리. 없다. |
| **2. 짧은 호흡** | 전투 분석, 인물 평가 | 한 줄 독백. "누가 말했는지" 안 써도 독자가 앎 | (절정 안정. 결승 상대.) / (강해.) |
| **3. 풀 대화** | 의견 충돌, 코미디, 감정 장면 | 이름 표기 + 여러 줄 대화 | (인플레이션이라고요.) / (뭐?) — 이건 빼면 재미가 죽음 |

### 8년차 동거 증거 6유형 — 매 화 최소 2회 삽입

| # | 유형 | 설명 | 예시 |
|---|------|------|------|
| 1 | **선행동** | 말하기 전에 몸이 먼저 움직임 | 천마가 "왼쪽!" 전에 위소운이 이미 피함 |
| 2 | **습관 전이** | 이준혁의 습관이 위소운 몸에 남음 | 허리춤에 손이 가는 팬텀 습관 (독백 없이) |
| 3 | **감정 선읽기** | 상대가 입 열기 전에 이미 앎 | "알아. 그 말 하려는 거지." 천마가 입을 다물었다. |
| 4 | **자동 역할분담** | 같은 풍경을 세 시선이 동시에 읽음 | 천마의 눈으로 무공을, 이준혁의 머리로 자본을 동시 분석 |
| 5 | **짧은 호흡** | 긴 말 필요 없이 한 단어로 통함 | (강해.) ← 이것만으로 3인이 같은 결론 |
| 6 | **무의식 싱크** | 셋이 동시에 같은 것을 떠올림 | "아무도 말하지 않았다. 셋 다 같은 이름을 떠올리고 있었다." |

### 적용 규칙
```
✅ 반사적/습관적 행동 → 독백 없이 위소운의 행동묘사로
✅ 전투 분석/짧은 판단 → 한 줄 독백 (이름 생략 가능)
✅ 코미디/갈등/감정 → 풀 대화 유지 (절대 삭제 금지)
✅ 매 화 "8년차 증거" 최소 2회 삽입
❌ 모든 독백을 풀 대화로 쓰는 것 = 1년차처럼 보임
❌ 모든 독백을 없애는 것 = 독자가 3인격을 "볼" 수 없음
```

### 검수 체크
```
□ 반사적 행동이 독백 없이 묘사된 곳이 1회 이상 있는가?
□ "8년차 증거" (선읽기/자동 분담/동시 싱크 등)가 2회 이상 있는가?
□ 코미디/갈등 대화는 그대로 유지되었는가?
□ 독자가 "이 셋은 오래 같이 산 사이구나"를 느낄 수 있는가?
```

```
- "진풍이 단도를 어깨에 걸친 순간, 위소운은 뭔가를 느꼈다. 설명하기 어려운 감각이었다."
- "또 왔다. 1전에서 느꼈던 그 감각. 유비의 검에 실린 기의 결이 흐릿하게 흘러들어왔다."
- "세 번 느끼니까 — 어렴풋하지만 알 것 같았다. 이름은 몰랐다. 하지만 있었다."

❌ 나쁜 예 (이 시점에서 금지):
- "기류감응이 작동했다." ← 아직 이름이 없다!
- "위소운이 상대의 경맥 운용을 읽었다." ← 너무 정확함. 어렴풋해야 한다!
- "단전에서 올라오는 진기가 이류 중위임을 파악했다." ← 등급 판단까지 정확히 하면 안 됨!
```

#### 1단계 (8~9화, 32강) — "보이기 시작, 자각하지만 이름 없음"

```
✅ 좋은 예:
- "검이 부딪히는 순간, 상대의 기가 — 색깔처럼 보였다. 처음 있는 일은 아니었다."
- "이상했다. 언제부터인지 모르겠지만, 상대의 기의 흐름이 눈에 들어왔다."
- 천마: "......너, 지금 기의 흐름을 읽고 있는 거야?" (천마가 먼저 알아챔)

❌ 나쁜 예:
- "기류감응 2단계가 발동했다." ← 아직 이름도 없고, 단계도 모른다!
- "위소운은 능숙하게 상대의 모든 경맥을 파악했다." ← 아직 미숙하다!
```

#### 1.5단계 (9~10화) — "천마가 이름을 붙이는 장면"

```
✅ 필수 장면 (9~10화 중 1회):
천마: "기류감응(氣流感應)이야."
위소운: "......뭐요?"
천마: "상대의 기의 흐름을 읽는 능력. 300년 전에 하나 봤어. 그놈은 내가 죽였지만."
이준혁: (절대음감 같은 건가요?)
천마: "비슷해. 근데 이놈은 아직 '멜로디'를 겨우 듣는 수준이야."

→ 이 장면 이후부터 "기류감응"이라는 단어를 본문에 사용 가능.
→ 이 장면 이전에는 절대 "기류감응"이라는 단어를 쓰지 않는다.
```

#### 2단계 (11~13화, 8강~결승) — "관전 감지, 의식적 사용"

```
✅ 좋은 예:
- "비무대 위에서 두 사람의 기가 부딪히고 있었다. 위소운에게는 — 보였다."
- "기류감응이 말해주고 있었다. 저 검에는 살의가 실려 있다."
- "관중석에서 보고 있을 뿐인데, 상대의 경맥 운용까지 읽혔다."

❌ 나쁜 예:
- "위소운은 만류귀종을 완성했다." ← 아직 이론 99%, 실전 첫 사용은 결승전에서!
```

#### 2.5단계 (12화, 야율흑전) — "역방향 기류 = 마교 간파"

```
✅ 필수 장면:
"이 기류는... 거꾸로 흐른다."
위소운이 느낀 것은 — 정상적인 경맥과 반대 방향으로 돌아가는 내공이었다.
천마의 목소리가 — 처음으로 떨렸다.
"......이건 마교야. 내 마교의 기(氣)다."
```

### 집필 전 체크리스트 — 기류감응

```
□ 이번 화의 기류감응 단계가 위 표와 일치하는가?
□ "기류감응"이라는 단어를 9~10화 이전에 사용하지 않았는가?
□ 0.5단계에서 위소운이 "정확히 파악"하는 묘사를 쓰지 않았는가?
□ 전투/관전 장면에서 기류감응 관련 묘사를 최소 1회 삽입했는가? (해당 단계에 맞는 톤으로)
□ 천마가 기류감응에 반응하는 장면이 있는가? (놀라움/인정/품평)
□ 단계 전환 시 "전환 장면"을 넣었는가? (갑자기 레벨업하면 안 됨)
```

### 3인격별 기류감응 반응 가이드

| 인격 | 반응 패턴 | 성장에 따른 변화 |
|------|----------|---------------|
| **위소운** | 느끼는 주체. 처음엔 당황 → 점차 의식적 사용 | "뭐지?" → "또 왔다" → "보인다" → "읽힌다" |
| **천마** | 가장 먼저 알아챔. 이름을 붙여줌. 코치 역할 | "......!" → "기류감응이야" → "좋아, 더 집중해" → "이제 네 눈이 열렸어" |
| **이준혁** | 비유로 이해. 경영적 가치 분석 | "절대음감 같은 건가요?" → "이건 경쟁 우위입니다" → "상대방 재무제표가 보이는 거네요" |
