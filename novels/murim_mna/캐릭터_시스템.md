## 🎭 캐릭터 개요 (빠른 참조용)

> ⚠️ 상세 카드는 각 캐릭터 항목 참조. 여기는 빠른 참조용 개요.

### 📋 1~15화 단역 상세 카드

#### 혜공 (慧空) - **"아미타불 머신건"** (7~8화)

- **정체**: 소림 방장 겸 **무림맹주(武林盟主)**, 80세 남성 | 7화 첫 등장 (개막식 축사)
- **외모**: 회색 승복, 108주 염주(손목), 흰 눈썹이 귀 밑까지 내려옴, 삭발, 마른 체구지만 등이 곧음
- **무공**: 72절기 + 금강불괴공 | **현경 초입** (천마 평가: "300년 전 맹주보다는 약하지만, 이 시대에선 꼭대기 근처")
- **성격**: 모든 상황에 "아미타불". 99% 시간을 눈 감고 있음. 진짜 큰일에만 눈을 뜸. 말이 극히 적지만 한마디가 무겁다. 공정하여 정사파 모두에게 존경받음.
- **대표 대사**:
  - "아미타불." (기본값)
  - "아미타불. 살생은 아니 되느니라." (평소)
  - "무(武)란 그침이니, 그칠 때를 아는 자가 진정한 무인이니라." (무림대회 개막식)
  - (눈을 뜨며) "...불법(佛法)이 무너지려 하는구나." → 이 한마디에 무림 전체 정적
- **습관**: 눈 감고 염주 돌리기 (1초 3알) | 차만 마심 | 말이 극히 적음 | 내공으로 목소리를 실어 광장 전체에 울림
- **맹주로서의 역할**: 무림대회 주관, 정사파 분쟁 중재, 마교 동향 감시. 20년간 맹주직을 수행하며 무림의 균형을 유지. 소림이 개봉 근처(숭산)에 위치하여 무림맹 본부와 가까움.
- **한 줄 요약**: 평소엔 졸고 있는 것 같은 노승. 눈을 뜨면 무림 전체가 긴장하는 존재. 무림맹주 겸직.

### 🎯 AI 집필 시 캐릭터 카드 활용 규칙

1. **등장 시 반드시 카드 참조**: 캐릭터가 에피소드에 등장하면 해당 카드의 성격·말투·외모·무공을 100% 준수
2. **대표 대사 변형 사용**: 카드의 대표 대사를 상황에 맞게 변형하되, 말투 톤은 유지
3. **습관은 묘사에 삽입**: 캐릭터의 습관(버릇)을 자연스럽게 행동 묘사에 녹임
4. **갈등 포인트 활용**: 캐릭터 간 갈등/감정 설정을 장면 연출에 적극 활용
5. **카드에 없는 설정 임의 창작 금지**: 인명록에 기록된 대로만 작성. 설정에 없는 건 "상상"하지 않음

## 🎬 단역 시스템 (Minor Character Engine)

`🔎 단역, 7대, 역할, 유형, Minor, Character, Archetypes, 카드, 템플릿, Card`

> **핵심 원칙**: 단역은 "있어도 그만, 없어도 그만"이 아니다.
> 주연이 칼이라면, 단역은 숫돌이다. 숫돌 없이는 칼이 날카로워지지 않는다.
> 
> **설계 철학**: 인기작 10편 분석 결과, 모든 히트작은 단역을 **전략적으로 설계**한다.
> - 화산귀환: 리액션 단역으로 주인공 강함을 "관객 시선"으로 증명
> - 전독시: 단역에게 "이름 1개 + 사연 1줄"을 주어 죽음에 무게를 부여
> - 원피스: 단역에게 명대사 1개를 줘서 수백 화 뒤 재등장 시 감동 극대화
> - 템빨: NPC(단역)에게 성격/충성심을 부여해 주역급으로 승격
>
> **활용법**: AI가 매 화 집필 시, 아래 시스템에서 적절한 유형의 단역을 **자동 배치**한다.

### 📐 단역 7대 역할 유형 (Minor Character 7 Archetypes)

### 📇 단역 카드 템플릿 (Minor Character Card)

> 모든 단역은 아래 카드 형식으로 등록한다.

> **선택**: 외모 특징, 재등장 이력, 승격 여부

```
■ 단역 카드
┌─────────────────────────────────────────────┐
│ 이름: (한자)                                   │

│ 사연 1줄: 이 사람이 왜 여기 있는지               │
│ 외모 특징: 기억되는 포인트 1가지                  │
│ 등장: 첫등장 화수 / 재등장 화수                   │

│ 승격 여부: - / 승격 검토 / 조연 승격             │
└─────────────────────────────────────────────┘
```

### 📋 단역 배치 공식 (Scene Deployment Formula)

> AI가 매 화 집필 시, 아래 공식에 따라 단역을 자동 배치한다.
> **핵심**: "단역이 왜 이 장면에 있는가?" 이유 없이 등장하는 단역은 금지.

#### 공식 1: 전투 장면 = 리액션 단역 필수

```
[전투 발생] → 리액션 단역 1명 이상 배치
- 배치 위치: 전투 직후 1~2문단 내
- 대사 패턴: "저, 저게 가능해?!" / "세상에..." / "이건 인간이 아니야..."
- 효과: 주인공 강함을 수치가 아닌 "타인의 눈"으로 체감
- 예시: "저, 저게 가능해?!" / "세상에..." / "이건 인간이 아니야..."
```

#### 공식 2: 긴장 장면 후 = 분위기 전환 단역 필수

```
[심각한 사건/전투/사망] → 분위기 전환 단역 2~5문단 내 배치
- 배치 위치: 긴장 해소 타이밍
- 대사 패턴: 일상적이고 따뜻한 한마디
- 효과: 독자가 숨 쉴 틈을 줌 (계속 긴장하면 피로)
- 예시: 장위 "만두 먹을래?" / 마현 "장위야, 분위기 좀 읽어라"
```

#### 공식 3: 새 지역 진입 = 세계관 단역 필수

```
[새 도시/문파/지역 진입] → 세계관 확장 단역 1명 이상 배치
- 배치 위치: 도착 직후 1~3문단 내
- 대사 패턴: 그 지역 말투 + 특산물/물가/풍습 언급
- 효과: 독자가 새 지역의 분위기를 즉시 체감
- 예시: 점소이 "이쪽으로 오십시오, 손님." / 손파 "아이고, 요즘 산적이 많아서..."
```

#### 공식 4: 출정/결심 = 감정 앵커 단역 필수

```
[주인공 출정/큰 결심] → 감정 앵커 단역 배치
- 배치 위치: 출발 직전
- 대사 패턴: "돌아올 거죠?" / "기다릴게요" / 말 없이 도시락 건넴
- 효과: 주인공이 돌아올 곳이 있다는 안도감 + 싸우는 이유 상기
- 예시: 소연화가 위소운에게 떡 건넴 / 당찬 "...먼저 간다." (감정 숨김)
```

#### 공식 5: 떡밥 투하 = 정보 단역 필수

```
[복선/떡밥 배치 필요] → 정보 전달 단역을 통해 자연스럽게
- 배치 위치: 객잔/다방/시장 장면, 포구/도박장
- 대사 패턴: "요즘 소문 들었어?" / "북쪽에서 이상한 놈들이..."
- 효과: 떡밥을 주인공이 직접 말하면 부자연스러움 → 단역 입으로 전달
- 예시: 객잔 손님 "요즘 소문 들었어? 북쪽에서 이상한 놈들이..." / 상인 "흑호문이 또..."
```

#### 공식 6: 적 침공/학살 = 희생 단역 필수

```
[적 침공/민간 피해] → 이름+사연 있는 희생 단역 배치
- ⚠️ 핵심: 이름 없는 "통행인 A"가 죽으면 독자는 무감각
-          이름+사연 있는 캐릭터가 죽으면 독자도 숙연
- 배치 위치: 피해 장면
- 효과: 주인공 분노의 정당성 + 독자 감정 이입 극대화
- 예시: 전투에서 이름 있는 인물의 희생 → 위소운 충격 → 독자 감정 이입
```

#### 공식 7: 5회 이상 재등장 = 승격 검토

```
[단역 재등장 5회 이상] → 조연 승격 검토 회의
- 조건: 5회 이상 등장 + 독자 반응 긍정적 (추정)
- 절차: 한 줄 태그 부여 → 대표 대사 확정 → 습관 추가 → B~C등급 검토
- 예시: 왕 노인(E-05) 4회 등장 → 서구진 사업편 추가 등장 시 승격 검토
```
