## 🏆 무림대회 출전자 — 30세 이하 열린 대회

`🔎 참가자, 구성, 다양한, 배경, 대회, 진행, 구조, 3연승, 예선, 32강`

> ⚠️ **30세 이하 누구나 참가 가능한 열린 대회**. 정파/사파/무소속 불문.
> 무림맹 주최, 3년에 1회, 개봉(開封)에서 개최. 참가자 200~300명.
> **우승 조건**: 실력뿐. 출신은 상관없음.
> **파워 밸런스**: 초절정(위소운 유일) > 절정급 > 절정 진입 직전 > 일류 고수 > 일류 초입 > 이류 상위
> ⚠️ 9대문파 + 5대가문 출전자는 최소 일류 이상이지만, **전체 참가자의 절반은 비(非)명문 출신**.
> ⚠️ 이름 없는 곳에서 온 무인이 명문 제자를 이기는 것 = 대회의 가장 큰 재미.

### 참가자 구성 — 다양한 배경

| 분류 | 비율 | 특징 | 스토리 역할 |
|------|------|------|-----------|
| **구파일방 제자** | ~30% | 정파 엘리트. 명문 자존심. | "무소속? 예선에서 떨어질 걸." |
| **오대세가 자제** | ~15% | 가문의 명예. 가전무공. 장비 풍부. | 이준혁 "저 장비 다 합치면..." |
| **지방 무관/도장 출신** | ~20% | 명문은 아니지만 실전 강함. 잡초 생명력. | 위소운과 공감. "나도 이름 없는 곳" |
| **떠돌이/독학 무인** | ~15% | 스승 없이 독학. 예측불가 무공. | "어디서 저런 기술을?" — 변수 |
| **상단/표국 소속** | ~10% | 호위무사 출신. 실전적. 돈 벌기 위해 참가. | 이준혁 "인재 발굴... 아닙니다." |
| **사파/회색지대** | ~3% | "무소속"으로 등록. 어두운 기술. | 천마 "이 기운... 사파야." |
| **⚠️ 마교/마교예하** | ~2% | 신분 완전 위장. 정보 수집+실력 과시. | ⭐ 천마 감지 → 핵심 복선 |
| **특이 배경** | ~5% | 산중 은둔, 이민족(서역/남만), 소수 문파 | 세계관 확장 |

### 📋 대회 진행 구조 — 3연승 예선 → 32강 토너먼트

> ⚠️ 전 지역에서 **300~500명**이 개봉으로 몰려온다.
> **3연승 예선**으로 **32명**을 선발한 후, 토너먼트 진행.

#### 예선 — 3연승 무대 방식 (挑戰三連勝)

> ⚠️ **핵심 규칙**: 출전 희망자가 무대에 올라가면, 다른 지원자가 도전자로 나선다.
> **연속 3명을 물리치면 32강 진출 확정.** 1패라도 하면 탈락. 시드 없음. 재도전 불가.

| 항목 | 내용 |
|------|------|
| **무대** | 개봉 무림맹 광장. ⭐ **비무대 1개만 운영** — 모든 관중의 시선이 한 곳에 집중. 경기장 열기 폭발 |
| **진행** | 출전자가 무대에 올라감 → 도전자(다른 지원자) 3명을 연속으로 이겨야 통과 |
| **도전자** | 자발적 지원. 누구든 손을 들 수 있음. 강한 상대를 골라 끌어내리려는 전략도 가능 |
| **휴식** | 1전 끝나면 짧은 휴식(약 5분)만 허용 |
| **탈락** | 3연승 중 1패 = **즉시 탈락**. 재도전 불가 |
| **시드** | ⭐ **없음**. 명문이든 무소속이든 모두 3연승 통과 필수 |
| **32명 확정** | 6일간 매일 5~6명 선발하여 32명 확정. 32명 채워지면 예선 종료 |

**3연승 예선이 만드는 드라마:**
- 명문파 제자라도 3연승 못 하면 탈락 → "이름값"이 통하지 않는 시스템
- 강한 출전자를 일부러 도전해서 끌어내리려는 연합 전략
- 무명인이 3연승 → "저 놈이 누구야?" 술렁임
- 마교/사파는 **일부러 평범하게 이김** (실력 숨김)

#### 예선이 만드는 스토리 장면

| 장면 | 내용 | 역할 |
|------|------|------|
| **위소운 예선** | 무소속이 무대에 올라감. 3명을 가볍게 이김. "저 놈 누구야?" | 첫 실력 공개. 기대감. |
| **곽진 예선** | 화산파 대제자. 압도적 3연승. 관중 환호. | 정파 스타의 등장. 위소운과 대비 |
| **남궁현 예선** | 3명 모두 검을 뽑기도 전에 끝남. | "소검신은 다르다" — 최강 확인 |
| **야율흑 예선** | 무표정으로 "적당히" 이김. 실력 감춤. | 천마: "저 놈... 숨기고 있어." |
| **탈락 드라마** | 명문 제자가 3번째 도전자에게 패배. "3년 준비했는데..." | 대회의 잔인함 |
| **3인격 해설** | 예선 전체를 3인격이 해설. 천마 "벌레들이네." / 이준혁 "채용 면접이네요." | 코미디+정보 전달 |

#### 본선 토너먼트 구조 — 32강 → 결승

```
32강 (32명, 16경기) → 16강 (16명, 8경기) → 8강 (8명, 4경기) → 4강 (4명, 2경기) → 결승 → 우승
```

#### ⭐ 위소운 대진표 (확정)

| 라운드 | 상대 | 결과 | 스토리 핵심 |
|--------|------|------|-----------|
| **32강** | 중급 정파 무인 | 빠른 승리 | 실력 본격 공개 |
| **16강** | ⭐ **곽진** (화산파) | **압도적 승** | "형, 오랜만이야." 정체 공개. 곽진 공포. |
| **8강** | ⭐ **호령** (사파 흑도) | 고전 끝에 승 | 첫 "진짜 위험한" 상대. 살인검. |
| **4강** | ⭐ **야율흑** (마교) | 격전 끝에 승 | 천마 "내 마교의 기다." 마교 떡밥. |
| **결승** | ⭐ **남궁현** (남궁세가) | 사투 끝에 승 | 만류귀종 첫 사용. 우승. |

> ⚠️ **마교/사파 세력은 모든 라운드에 존재**. 마교의 규모를 보여주는 장치.

| 사연 | 지하 비무장에서 "개봉 권왕" 타이틀 3년. 대회 상금으로 도박 빚을 갚으려고 참가. 위소운 1~2전을 보고 "강하지만 내 경험이면 한 방 먹일 수 있다"고 판단 |
| 한마디 | "규칙? 나는 규칙 없는 데서 싸워왔는데. ...... 어디, 한번 해보지." (입꼬리 올리며) |
| **패배 묘사** | 첫 3합은 더티 파이트(머리 박치기, 모래 뿌리기 시도 — 비무 규칙 위반 경고)로 위소운을 당황시킴 → 하지만 위소운이 **적응한 뒤 5합 만에 제압**. 곽철의 주먹이 위소운 뺨을 스침(처음으로 위소운에게 "닿은" 도전자). 위소운이 곽철의 팔을 잡아 비틀어 항복 유도. |
| 패배 후 반응 | (팔을 잡힌 채) "...... 야, 진짜 강하네. 하하!" (쿨하게 인정) → 관중 박수 |
| **관중 반응** | "3연승! 무소속 위소운, 32강 진출 확정!" → 술렁임 → "저 놈 대체 누구야?" |
| 3인격 반응 | 천마: "뺨을 스쳤어. 실전 감각은 있는 놈이네." / 이준혁: '저 사람...... 경호원으로...... (직업병)' / 위소운: (뺨을 만지며 쓴웃음. 아직 실전 감각이 부족하다는 자각) |
| 비고 | ⭐ 이 캐릭터는 위소운에게 "처음 닿은 상대"라는 의미. 노력 증명 원칙: 위소운이 무적이 아님을 보여줌. 나중에 개봉 정보원 또는 천화루 경호원 후보 |
