## 2. 배경 — 소설 시작 전 (과거)

> ⚠️ 아래는 **1화 이전에 이미 일어난 일**이다. 본편에서는 회상/대화로만 등장한다.

### 위소운의 18년
| 나이 | 사건 |
|------|------|
| 0세 | 고아 출생 |

### 빙의 + 무인도 수련 (현실 3년 = 삼혼귀일경 7년)
| 시기 | 사건 |
|------|------|
| 18세 | 죽음의 경계에서 **이준혁+천마 동시 빙의** |
| 18~21세 | 무인도 수련 3년 |
| | - 삼혼귀일경(10배 배율) 약 7년분 수련 |
| | - 천마에게 정파제일고수(소무진)의 무공 전수 |
| | - 화산파 기본기 + 태허귀일결 + 자창 무공 융합 |
| | - 이준혁에게 경영/요리/현대 지식 학습 |
| | - **총 수련 14년** (화산파 7년 + 삼혼귀일경 7년) |
| | - **초절정급** 달성. 화경 근접. |

> **위소운이 초절정인 근거**:
> - 화산파 시절부터 **무공 천재**. 문제는 체질(내공 축적이 느림)뿐이었음
> - 천마 = **현경급** 무공의 달인. 300년 전 정파제일고수(소무진)의 무공까지 보유
> - 삼혼귀일경 7년 = **현경급 스승에게 1:1 실전 수련 7년**. 하루에 수십 번 죽고 다시 일어남
> - 체질 문제 → 태허귀일결로 해결. 내공 축적이 정상화됨
> - 화산파 7년(기본기) + 삼혼귀일경 7년(실전) = **14년 무공 인생**
> - 이후 사업에 치우치면 수련 시간이 줄어들기 때문에, **지금이 무공 성장의 전성기**
> - **화경은 머지않아 돌파** 가능. 무림대회 실전 경험 + 기류감응 2단계 도약이 계기
> - ⭐ **만류귀종(萬流歸宗)**: 삼혼귀일경 5년차부터 연구 중. "정공과 마공은 같은 강의 양쪽 둑" → 모든 무공을 하나로 융합하는 최상위 철학
> - ⭐ **기류감응(氣流感應)**: 상대의 기류를 읽는 천재 능력. 무림대회에서 2단계 도달 → 만류귀종 이론 완성의 계기

### 삼혼귀일경 7년 동안의 3인격 관계 변화

> ⚠️ **이 변화가 1화 시작 시점의 3인격 상태를 결정한다.**

| 삼혼귀일경 시기 | 관계 | 핵심 |
|-------------|------|------|
| 1~100일 | **충돌** — 서로 못 참음 | 이준혁 공포+거부, 천마 모욕감, 위소운 혼란 |
| 101~300일 | **타협** — 규칙을 만듦 | 생존을 위해 어쩔 수 없이 협력 |
| 301~500일 | **신뢰** — 등을 맡김 | 이준혁의 소금밥, 위소운의 18년 이야기 |
| 501~700일 | **이해** — 상대 생각을 앎 | 천마가 변하기 시작 |
| 701~1000일 | **이심전심** — 말이 필요 없음 | 세 사람이 하나처럼 |

**천마의 변화 (핵심)**:
- 이준혁이 소금 하나로 맛있는 밥을 만들기 위해 매일 노력하는 모습
- 위소운의 18년 — 맞고, 굶고, 버림받은 아이의 이야기를 듣고
- **마교천하** → **천하재패** (최강이면서 존경받는 문파)로 목표 전환
- 공포로 지배하는 것이 아니라, 힘이 있되 존경받는 길을 선택

## 4. 3인격 목표 — 무(武) + 인(仁) + 상(商)

> ⚠️ **이 세 목표가 소설 300화를 관통하는 핵심 엔진이다.**
> "각자 하고 싶은 걸 하자" — 하지만 세 목표는 결국 하나로 수렴한다.

### 천마 — 武 (무력과 명예)
| 항목 | 내용 |
|------|------|
| **원래 목표** | 마교천하 (삼혼귀일경 진입 전) |
| **현재 목표** | **천하재패** — 무림 최강의 문파 건설 |
| **핵심** | 힘이 있으나 **존경받는** 문파. 마교가 아님. |
| **변화 이유** | 7년간 이준혁의 소금밥 + 위소운의 18년 인생 |
| **겉으로** | "최강 문파를 세운다" (자존심상 천하재패라고 포장) |
| **속으로** | 이번에는 배신당하지 않는, 사람을 믿을 수 있는 세력을 만들고 싶다 |

### 위소운 — 仁 (보호와 안전)
| 항목 | 내용 |
|------|------|
| **목표** | 안 맞고 배 안 고픈 세상 |
| **상태** | 힘이 생겨 **기본은 달성**. 구체적 결과물(고아 보호 등) 구축 필요 |
| **핵심** | 천마의 문파가 커지면 자연스럽게 완성됨 |
| **구체화** | 흑천상단이 있는 마을에서 고아가 일자리·밥을 얻을 수 있다 |

### 이준혁 — 商 (사업과 시스템)
| 항목 | 내용 |
|------|------|
| **목표** | 천화루 300호점 (상업 제국) |
| **핵심** | 상가(商家) 담당. 문파의 경제력을 만드는 사람 |
| **측정 기준** | 1호점 → 10호점 → 100호점 → 300호점 (독자가 센다) |
| **동기** | "전생에도 300개 열었다. 이번엔 죽지 않고" |

### 3목표 연결 구조
```
  천마의 최강 문파 (武)
       │
       ├── 위소운의 안전한 세상 (仁) ← 문파 안에서 자연스럽게 실현
       │
       └── 이준혁의 상업 제국 (商) ← 문파의 경제 기반 제공
       
  → 세 갈래가 하나의 세력으로 수렴
  → "각자 하고 싶은 걸 하자"이지만, 결국 같은 곳을 향한다
```

### ⭐ 위소운-소연화 관계 발전 라인 (1~75화)

> ⚠️ **이 관계는 "관찰→호기심→이해→신뢰→애정"으로 서서히 발전한다.**
> 급격한 로맨스 금지. 위소운의 사교 서투름 + 소연화의 영리한 관찰이 만드는 느린 전개.
> 핵심: 소연화는 "비밀을 캐는 영리한 눈"이지만, 위소운의 비밀을 다 알게 된 후에도 곁에 있는 사람.

#### 여정편 (1~6화) — "7일의 관찰" (이미 집필 완료)

> ⚠️ 기존 본문에 아래 요소가 이미 자연스럽게 삽입되어 있음. 수정 불필요.

| 화 | 소연화 행동 | 위소운 반응 | 관계 온도 |
|----|-----------|-----------|----------|
| **1화** | 손 관찰 → 먼저 다가감 → "같이 가시겠습니까?" | 반 박자 늦은 자기소개 (사교 서투름) | 🌡️ 호기심 |
| **2화** | 옆에서 걸음 → "외롭지 않으셨어요?" → "비밀이 많으시네요" | "시끄러운 편이라서요, 섬이" (3인격 때문에 진실) | 🌡️ 호기심+관찰 |
| **3화** | 차 따르기 → 하얀 손가락 발견 → 달빛 검 목격 → **심장이 뜀** | "고맙습니다" (소연화에게만 자연스러움) | 🌡️🌡️ 설렘 시작 |
| **4화** | 요리 분석("사부님이 좋아하셔서") → "먼 곳을 보세요" | "생각이 좀 많은 편이에요" (웃으며 넘김) | 🌡️🌡️ 교감 |
| **5화** | 말없이 차를 놓음 (배려) → 퍼즐 조각 모으기 | "고맙습니다" (평소보다 작은 목소리) | 🌡️🌡️ 신뢰 |
| **6화** | "괜찮아요?" (이름 안 붙이고 직접 호명) → 등록 동행 → "긴장돼요?" | "응" (소연화에게만 반말 섞임) | 🌡️🌡️🌡️ 친밀 |

> ⭐ **6화까지의 핵심**: 소연화는 위소운의 "퍼즐 조각"을 모으고 있다.
> 매화검법, 하얀 손가락, 먼 곳을 보는 눈, 사교 서투름, 달빛 아래 검.
> 아직 그림은 완성되지 않았지만 — 조각이 쌓이고 있다.
> 그리고 당찬은 이 모든 것을 뒤에서 보고 있다.

#### 무림대회편 (7~13화) — "20일의 거리 좁히기"

| 화 | 소연화-위소운 장면 | 당찬 반응 | 관계 온도 |
|----|------------------|----------|----------|
| **7화** | 예선 관전. 소연화가 위소운의 3연승을 보고 "아" — 한마디 | 당찬: 소연화 표정 봄. 입 다묾 | 🌡️🌡️🌡️ |
| **8~9화** | 쉬는 날 함께 개봉 시장 구경. 소연화가 음식 설명(요리 지식) | 당찬: 따라가지만 셋이서 어색 | 🌡️🌡️🌡️ |
| **10화** | 곽진전 후 소연화: "그 사람이... 사형이었어요?" 첫 직접 질문 | 당찬: 위소운 과거에 충격 | 🌡️🌡️🌡️🌡️ |
| **11~12화** | 부상 치료 장면. 소연화가 약을 바르는데 손이 떨림 | 당찬: 물러섬 시작 | 🌡️🌡️🌡️🌡️ |
| **13화** | 우승 후 소연화: "축하해요." 위소운: "고마워요." (반말 자연스러움) | 당찬: 조용히 축하 (성장) | 🌡️🌡️🌡️🌡️🌡️ |

#### 기(起) 시작편 (14~25화) — "각자의 자리"

| 화(경) | 소연화-위소운 | 핵심 |
|--------|------------|------|
| 14~16화 | 천화련 설립 과정에서 소연화가 자발적으로 돕겠다고 함 | "저도 할 수 있는 게 있어요" |
| 17~20화 | 소연화가 천화루 요리 R&D에 참여 (요리 지식 활용) | 이준혁+소연화 = 요리 토론 케미 |
| 21~25화 | ⭐ 소연화가 아버지(소풍명)에게 서구진 관리를 제안 | 서구진 = 위소운의 고향, 소연화의 역할 |

#### ⭐ 서구진 관리 — 천풍검문(소가) 위탁 (25~30화경)

> ⚠️ **서구진 = 위소운의 원점. 흑호문이 횡포를 부리던 곳.**
> 위소운이 떠난 후에도 서구진 사람들은 보호받아야 한다.
> 소연화의 아버지 소풍명(蘇風明, 천풍검문 장문인)이 서구진 일대를 관리.
> **천풍검문 = 태호 지역 중소문파** → 서구진(태호 서쪽 포구)과 지리적으로 인접.

| 항목 | 내용 |
|------|------|
| **관리 주체** | 천풍검문(天風劍門) — 소풍명 장문인 |
| **관리 지역** | 서구진(胥口鎭) 포구 및 주변 촌락 |
| **관리 방식** | 천화표국 서구진 분소(分所) 설치. 천풍검문 제자 5~10명 파견. 치안 유지 + 상인 보호 |
| **흑호문 처리** | 위소운 귀환 시 졸개 제압(1화). 이후 천풍검문이 남은 세력 정리 |
| **경제** | 천화상회 서구진점(소규모). 어부/짐꾼 보호. 포구 세금 공정화 |
| **위소운과의 의미** | "내가 지키지 못했던 사람들을 — 소연화의 가문이 지켜준다." |

**스토리 효과:**
- 위소운: "서구진에 천풍검문이 있으면 — 안심할 수 있어." (소연화에 대한 신뢰의 표현)
- 소연화: 위소운의 과거(서구진의 바보)를 알게 됨 → 퍼즐 완성에 가까워짐
- 당찬: 서구진 분소장으로 파견됨 → 소연화와 물리적 거리 → 성장의 계기
- 천풍검문: 소규모 문파에서 천화련 지역 파트너로 격상 → 소풍명의 판단력 증명
- 이준혁: "태호 지역 물류 거점입니다. 천풍검문이면 안전하고요." (사업+인연 일석이조)

> ⚠️ **이 설정이 만드는 감정선**:
> 1화에서 소연화가 서구진 천향루에서 위소운을 처음 만남
> → 25~30화에서 소연화의 가문이 그 서구진을 지킴
> → "처음 만난 곳을 지켜주는 사람" = 독자가 자연스럽게 관계를 응원하게 됨

### 무림대회 설정 (武林大會)

> ⚠️ **무림대회는 30세 이하 누구나 참가 가능한 열린 대회다.**
> 구파일방+오대세가 전용이 아니다. 무림의 모든 젊은 무인이 실력을 증명하는 자리.
> 이것이 대회를 더 풍성하고, 위소운의 우승을 더 가치 있게 만든다.

#### 대회 기본 정보

| 항목 | 내용 |
|------|------|
| **명칭** | 무림대회 (武林大會) / 개봉무림대회 |
| **장소** | 개봉(開封) — 무림의 중심 도시 |
| **주관** | 무림맹 (武林盟) |
| **참가 자격** | ⭐ **30세 이하, 누구나**. 정파/사파/무소속 불문. 단, 마교는 공식 불가(몰래 참가 가능) |
| **주기** | 3년마다 1회 |
| **규모** | 300~500명 지원 → **3연승 예선** → 본선 **32명** → 토너먼트 → 결승 |
| **상금/명예** | 우승자 = "무림제일신예(武林第一新銳)". 상금 + 각 세력 스카우트 경쟁 |
| **시드** | ⭐ **없음**. 명문이든 무소속이든 모두 3연승 예선을 통과해야 한다 |

#### 예선 — 3연승 무대 방식 (挑戰三連勝)

> ⚠️ **핵심 규칙**: 출전 희망자가 무대에 올라가면, 관중/다른 지원자가 도전자로 나선다.
> **연속 3명을 물리치면 32강 진출 확정.** 1패라도 하면 탈락. 시드 없음.

| 항목 | 내용 |
|------|------|
| **무대** | ⭐ 개봉 무림맹 광장. **비무대 1개만 운영** — 모든 관중의 시선이 한 곳에 집중. 경기장의 열기가 폭발. |
| **진행** | 출전자가 무대에 올라감 → 도전자(다른 출전자) 3명을 연속으로 이겨야 통과 |
| **도전자** | 자발적 지원. 누구든 손을 들 수 있음. 강한 상대를 골라 끌어내리려는 전략도 가능 |
| **휴식** | 1전 끝나면 차 한 잔(약 5분) 정도의 짧은 휴식만 허용 |
| **탈락** | 3연승 중 1패라도 하면 **즉시 탈락**. 재도전 불가 |
| **32명 확정** | 32명이 채워지면 예선 종료 |
| **관전** | 관중석 자유 관람. 이 과정에서 출전자들의 실력이 공개됨 |

**3연승 예선이 만드는 드라마:**
- 명문파 제자라도 3연승 못 하면 탈락 → "이름값"이 통하지 않는 공정한 시스템
- 강한 출전자를 일부러 도전해서 탈락시키려는 전략 → "저 녀석을 끌어내려라!"
- 3번째 상대가 의도적으로 강한 도전자를 보내는 연합 전략

- 마교/사파 출전자는 **일부러 평범하게 이김** (실력 숨김)

### 천화련(天火聯) — 조직 구조 (무림대회 이후)

> ⚠️ **천화련 = 지주회사(그룹). 산하에 3개 사업부.**
> 현대로 치면 **삼성그룹** = 천화련, 산하에 삼성물산+CJ푸드빌+신세계.

#### 전체 구조

```
천화련(天火聯) ← 그룹 전체 / 위소운 총수, 이준혁 경영, 천마 무력 총괄
│
├── 🗡️ 천화표국(天火鏢局) — 무력 + 물류 + 호위
│   ├── 본사 겸 개봉지국 = 옛 안씨표국 (300년 뿌리)
│   ├── 소주지국, 항주지국... (각 성 1~2개)
│   ├── 물류 호위 (천화루/천화상회 상품 배송 보호)
│   └── 정보 수집 (각 지역 동향 파악)
│
├── 🍜 천화루(天火樓) — 객잔+식당 통합 (중국식 주루 모델)
│   ├── ⭐ 1층 = 식당/주점 (이준혁 레시피), 2~3층 = 객실(숙박)
│   ├── 개봉 본점, 소주점, 항주점... (각 성 체인)
│   └── "먹고 자고"를 한 건물에서 = 객잔이 곧 천화루
│
└── 🏪 천화상회(天火商會) — 유통/소매 (백화점형 매장)
    ├── 천화루 옆 또는 1층 한편에 매장
    ├── 이준혁 발명품 주력: 증류주, 비누, 양념, 생활용품
    └── "귀족만 쓰던 물건을 모두에게" = 시장 혁명
```

> ⚠️ **천화루 = 객잔+식당 통합**: 중국식 주루(酒樓)/객잔(客棧)은 원래 **한 건물**.
> 1층에서 먹고 마시고, 2~3층에서 잔다. 객잔과 식당을 따로 분리하는 건 비현실적.
> 천화루 하나가 "숙박+식사+주점"을 전부 해결. 천화상회는 바로 옆에 배치.
>
> ⚠️ **지역 거점 구성**: 각 성마다 **천화루(숙식) + 천화상회(쇼핑) + 천화표국 지국(호위/물류)**
> 여행자 동선: 천화루에서 묵고+먹고 → 옆 천화상회에서 쇼핑 → 짐은 천화표국이 배송
> 이준혁: "고속도로 휴게소 모델입니다. 한곳에서 다 해결."
> 천마: "그게 뭔데."

#### 천화상회(天火商會) — 이준혁의 핵심 사업

> ⭐ **"귀족만 쓰던 것을 모두에게."** 이것이 천화상회의 슬로건.

| **위생** | 천화비누 | 귀족만 향료 사용. 일반인은 물로만 | 저가 비누 대량생산. "모든 사람이 깨끗해질 수 있다" |
| **양념** | 천화장(간장류), 천화유(참기름) | 소금+된장 위주 | 다양한 양념 = 천화루 맛의 비밀 |

#### 당근과 채찍의 효과 — 소문의 힘

```
[1차] 호남 흑사련 제압 (76~100화)
  → 소문: "보호세 뜯다가 밟혔대" → 섬서/강서 파트너십 순조로움

[2차] 산서 철기방 제압 (151~175화)  
  → 소문: "화경급도 졌대" → 복건/광동이 먼저 연락

[결과] 12개 성 중 10개는 평화적 파트너십, 2개만 무력 장악
  → 이준혁: "전체 확장 비용의 80%를 이 2곳에 쓴 겁니다. 하지만 나머지 10곳의 협상 비용이 0이었으니..."
  → 천마: "그래서 남는 장사라는 거야?"
  → 이준혁: "...ROI로 보면 역대급이죠."
```

> ⚠️ **3인격 갈등 포인트**: 적대적 장악 시 항상 의견 충돌
> - 천마: "전부 밟으면 빠르잖아." (무력 선호)
> - 이준혁: "전쟁은 비용이에요. 파트너십이 싸요." (효율 선호)
> - 위소운: "대의명분이 있으면 싸운다. 없으면 안 싸운다." (중재)
> - 흑사련/철기방은 **명확한 악당**이므로 위소운도 동의 = 대의명분 충족

| **위치** | 개봉 외곽. 큰 길에서 살짝 벗어난 2층 건물 + 뒤편 수련장 |
| **규모** | 30~50명 (소규모지만 정예) |
| **창시조** | "안현(安玄) 선생" — ⭐ **안가는 이 사람이 누구인지 모름.** 그냥 "강한 무인"으로만 전해짐. 실체: 천마가 중원제일인(소무진, 현경급)으로 변장 |
| **현 표국주** | **안세진(安世珍)** — 안현의 후손. 10대째. 일류 고수. |
| **창시조 정체** | ⭐ **3중 비밀**: ① 안가는 "안현 선생"만 앎 ② 실은 중원제일인(소무진) 행세 ③ 진짜 정체는 천마(마교교주) |
| **비밀 금고** | 표국 지하에 봉인. "창시조가 남긴 것. 열쇠를 가진 자가 온다" |
| **사업** | 소규모 호위 업무 (약재/귀금속 운송). 개봉에서 신용도 높음 |

#### 안씨표국 무공 — 귀원검법(歸元劍法) + 청원심법(淸源心法)

> ⚠️ **천마가 중원제일인(소무진, 현경급)으로 변장하여 직접 가르친 정파 무공.**
> 단, **검법도 심법도 절반만 전수**하고 떠남.
> **이 무공의 기류(氣流) 패턴이 안가 신분 확인의 열쇠.**

##### 청원심법(淸源心法) — "맑은 근원" (내공심법)

> 소무진 내공(태허귀일결)을 기반으로 천마가 간략화한 정파 심법. 전체 **9층(層)**.
> ⭐ **핵심 경지 대응**: 5층=절정 / 7층=화경 / 9층=현경

| 층 | 이름 | 경지 | 원래(안씨표국) | 천화표국 전수 | 위소운 | 효과 |
|---|------|------|-------------|-------------|--------|------|
| 1층 | **정수(靜水)** | 삼류 | ✅ | ✅ | 완성 | 기초 내공. 호흡 안정. 기류 패턴의 씨앗 형성 |
| 2층 | **류천(流泉)** | 이류 | ✅ | ✅ | 완성 | 내력 축적. 기류 패턴 안정화 = **신분 확인 가능** |
| 3층 | **청담(淸潭)** | 일류 초입 | ✅ | ✅ | 완성 | 내력 강화 |
| 4층 | **심연(深淵)** | 일류 후기 | ✅ (여기까지) | ✅ | 완성 | 옛 안씨표국의 천장이었던 곳 |
| **5층** | **원류(源流)** | ⭐ **절정** | ❌ | ⭐ **✅ 전수** | 완성 | 기류가 "흐름"으로 변화. 천화표국 표준 목표 |
| **6층** | **귀원(歸源)** | 초절정 | ❌ | ❌ (천재만 자력) | ⭐ **현재** | 내공 순환 자가 회복. **위소운 독점 영역** |
| **7층** | **활수(活水)** | ⭐ **화경** | ❌ | ❌ | 배움(미도달) | 내공이 "살아 있는 물"처럼 움직임 |
| 8층 | **만류(萬流)** | 화경 후기 | ❌ | ❌ | 배움(미도달) | 모든 기류를 하나로 모으는 단계 |
| **9층** | **청원(淸源)** | ⭐ **현경** | ❌ | ❌ | 배움(미도달) | 소무진의 경지. 이론상 완성 |

> **옛 안씨표국**: 1~4층만 전수 → 일류 후기가 천장
> **천화표국 전수 정책**: 1~5층 + 검법 12초식 전부 → **절정이 표준 목표** (천재는 초절정 가능)
> **위소운**: 9층까지 배웠으나 6층(초절정)까지 익힘. 7층(화경) 돌파 예정.
> **6층 이상은 위소운 독점** = 조직 내 절대 권위.
>
> 이준혁: "9층까지 교과서는 다 받았는데, 시험은 6층까지만 합격한 겁니다."
> 천마: "5층이면 충분해. 그 이상은 그 놈들이 감당할 그릇이 아니야."

##### 귀원검법(歸元劍法) — "원점으로 돌아가는 검" (검법)

> 소무진의 만검귀일(萬劍歸一)을 기반으로 간략화. 전체 **12초식**.
> ⭐ **천화표국 전수 정책: 12초식 전부 전수.** (옛 안씨표국은 전반 6초식만 전수됐음)

| 구분 | 초식 | 특성 | 옛 안씨표국 | 천화표국 |
|------|------|------|-----------|---------|
| **전반부** | 1~6식 | 방어·호위 특화. 짐을 지고도 사용. 다수 상대 방어에 강함. | ✅ | ✅ |
| **후반부** | 7~12식 | 공격·결정타 위주. 전반과 합쳐야 "귀원" 완성. | ❌ (미전수) | ⭐ **✅ 전수** |

##### 전수 범위와 한계 — "반만 준 세트"

| 구분 | 옛 안씨표국 (300년간) | 천화표국 전수 정책 (변경 후) | 위소운 독점 |
|------|-------------------|------------------------|-----------|
| **심법** | 1~4층 (일류 한계) | **1~5층** (절정) | 6~9층 (초절정~현경) |
| **검법** | 전반 6초식만 | **12초식 전부** | — (동일) |
| **도달 경지** | 일류 후기 | **절정** (천재: 초절정) | 현재 6층(초절정) |

> **300년간의 잠금**: 심법 4층 + 검법 6초식 = 일류 한계 (천마의 안전장치)
> **위소운 합류 후**: 5층 + 12초식 전부 전수 = 절정급 정예 부대로 격상
> **천마가 원래 절반만 가르친 이유**: "금고 지키는 놈들이 너무 강하면 눈에 띈다"
> **지금 전부 전수하는 이유**: "이제 숨을 필요가 없어. 싸울 준비를 해야 해."

> **⭐ 무림대회에서의 발견 — 안세진이 위소운을 찾아오다:**
>
> **[장면 1] 무림대회 관중석 — 안세진의 의문**
> - 안세진이 관중으로 무림대회를 관람 중
> - 위소운이 경기에서 무공을 펼치는 것을 봄
> - 안세진: "......저 검법의 기초 동작이... 우리 귀원검법과 같다?"
> - 300년간 전해 내려온 검법의 뿌리를 처음으로 목격하는 순간
> - 심장이 빨라짐. "창시조... 안현 선생님의 검과 같은 뿌리라는 건가?"
>
> **[장면 2] 무림대회 직후 — 안세진이 위소운을 찾아옴**
> - 안세진이 위소운을 직접 찾아옴 (위소운이 먼저 가는 게 아님)
> - 안세진: "실례합니다. 위 소협의 검법을... 어디서 배우셨습니까?"
> - 위소운: (기류감응으로 안세진의 내공을 감지) "......이 기류... 청원심법?"
> - 천마: (내면에서 침묵. 그리고) "......찾았네. 내 곳을."
> - 서로가 서로를 알아보는 순간 = 소설의 명장면 중 하나
>
> **[장면 3] 안씨표국 방문 — 진실의 공개**
> - 안세진의 안내로 안씨표국 방문
> - 위소운이 문도들의 수련을 봄 → 귀원검법 전반부에서 **소무진 검법의 흔적**을 확인
> - 천마: "......내가 그 놈 행세를 해서 세운 곳이야."
> - 위소운: "중원제일인으로... 변장을요?"
> - 천마: "그 놈의 검으로 사람들을 지키게 했어. 내 검으로는... 지킬 수 없으니까."
> - 안세진은 아직 모름. 그저 "같은 뿌리의 무공을 쓰는 자"라는 것만 확인
> - 위소운이 유언("열쇠를 가진 자가 온다")과 청원심법 기류 패턴으로 신분 확인
> - 안세진: "......기다렸습니다." (300년 유언의 실현. 눈이 붉어짐)
>
> **핵심**: 안세진은 이 시점에서도 **창시조가 "중원제일인"인 줄 모름**.
> 그저 "안현 선생님의 검과 같은 뿌리의 무공을 쓰는 자가 드디어 왔다"는 것만 앎.
> 창시조의 정체(중원제일인→실은 천마)는 **나중에 밝혀질 초대형 떡밥**.

> **⭐ 후반 6초식 전수 — 안씨표국 충성 확보의 결정타:**
> - 위소운이 귀원검법 후반부 6초식을 안씨표국에 전수
> - 안세진: "......300년간... 미완이었습니다. 이제야 완성이..." (눈물)
> - 안효림(불만파): "......이 초식이... 전반부와 이어지면..." → 직접 써보고 → 무릎 꿇음
> - **효과**: 전반부만으로 일류였던 안씨표국이 → 12초식 완성으로 **절정급 도달 가능**
> - **의미**: "원래 주인이 돌아왔다"는 증거. 무력 제압보다 강한 충성 확보.
> - **3인격**: 천마: "그래... 원래 그 놈 검은 12초식이야. 반만 줬으니까." / 이준혁: "돈으로 충성을 사는 것보다 기술 이전이 효과적입니다. 경영학 기본이에요."
> - ⭐ **떡밥 3중 구조**:
>   - 1차 떡밥 (무림대회): 안세진이 위소운 검법에서 자기 검법을 발견 ✅ 회수
>   - 2차 떡밥 (안씨표국 방문): "열쇠를 가진 자" 유언 실현 ✅ 회수
>   - 3차 떡밥 (미래): 창시조 "안현 선생" = 중원제일인 = 실은 천마 → ❌ 미회수. 초대형 떡밥

#### 금고(안가) 접근 시스템 — 이중 잠금

| 잠금 | 열쇠 | 누가 가졌나 |
|------|------|-----------|
| **1차: 신분 확인** | 청원심법의 기류 패턴 | 안씨표국 표국주 (대대로 전수) |
| **2차: 금고 개봉** | 천마신공의 기운 결 | ⭐ 오직 위소운만 (천마에게 배운 "비밀번호") |

> 안씨표국은 300년간 "언젠가 열쇠를 가진 분이 오신다"는 유언을 지켜왔다.
> 1차 잠금(신분 확인)은 표국주가 열어주고, 2차 잠금(금고)은 위소운이 여는 구조.
> 이 장면 = 천마의 300년 전 준비 + 현재 3인격의 유대 = 감동의 정점 중 하나.

#### 안씨표국 내부 관리 — "주인 귀환 후"

> ⭐ **이것은 "인수"가 아니다. 원래 주인이 돌아온 것이다.**
> 안씨표국의 창시조 = 천마. 300년 만에 주인(천마의 대리인=위소운)이 돌아온 것.
> 하지만 300년 동안 자기들끼리 운영해온 사람들이 갑자기 "새 주인"을 받아들이기 쉽지 않다.

**내부 반응 3유형:**

| 유형 | 비율 | 반응 | 대응 |
|------|------|------|------|
| **환영파** | ~40% | "창시조의 유언이 실현됐다!" 감격 | 핵심 인력으로 등용 |
| **관망파** | ~40% | "두고 보자. 실력이 있나?" | 성과로 증명 → 자연스러운 합류 |
| **불만파** | ~20% | "300년 동안 우리가 지켰는데, 갑자기 와서 주인?" | 🗡️ **무력으로 제압** or 💼 **지점장으로 파견** |

**불만 세력 처리 — 당근과 채찍:**

| 처리 | 대상 | 방법 | 효과 |
|------|------|------|------|
| 🗡️ **무력 제압** | 반항하는 자 | 위소운 직접 비무 → 실력 차이 체감 | "이 정도면... 따를 수밖에 없다" |
| 💼 **지점장 파견** | 능력 있지만 불만인 자 | 천화루 지방 지점 책임자로 보냄 | 키워주기 + 본사에서 떼어놓기 + 성과 내면 인정 |
| 🤝 **인정** | 성과 낸 지점장 | "당신이 이 지역을 이만큼 키웠다" | 불만 → 충성으로 전환. 가장 강한 로열티 |

> **이준혁**: "불만이 있는 사람을 죽이면 1명을 잃습니다. 지점장으로 보내면 매장 1개를 얻습니다."
> **천마**: "300년 전에는 그냥 죽였는데." 
> **위소운**: "이준혁 님 방식이 맞아요. 하지만 반항하면... 그때는 보여줘야죠."

### 개봉(開封) 기존 세력 — 본사 주변의 판도

> ⚠️ 개봉은 무림의 수도. 천화련이 본사를 두려면 이 세력들과의 관계를 설정해야 한다.

#### 개봉 세력 지도

```
[최상위] 무림맹(武林盟) — 무림의 정부. 감시자. 직접 장사 안 함.
[최상위] 황실/금의위 — 정치적 영향력. 226화+ 본격 등장.
[상위]   천하재일상단 — 개봉 상권의 40%. 최종 M&A 타겟(276~300화).
─────────────────────────────────────────
[중간]   개봉상회(商會) — 상인 길드. 점포 허가/규칙 관장.
[중간]   진무관(鎭武館) — 개봉 최대 무관+호위 조직. 제자 200~300명.
[중간]   만리표국(萬里鏢局) — 개봉 최대 표국. 장거리 물류 독점.
─────────────────────────────────────────
[하위]   안씨표국(→천화표국 본사) — ⭐ 우리 편. 300년 토박이 → 천화련 본사로 확장.

```

#### 개봉상회(開封商會) — 상인 연합

| **성격** | 개봉 상인들의 길드. 점포 개업 허가, 거래 규칙, 분쟁 조정 |
| **회주** | **진만복(陳萬福)** — 60대 노상인. 비무력. 40년 인맥으로 개봉 상권 조율. |

#### 진무관(鎭武館) — 개봉 최대 무관

### 기(起): 각자의 꿈 (14~75화)

> 무림대회 우승 후, 3인격이 각자의 목표를 향해 움직이기 시작.

**핵심 사건 (확정):**

  - 무림대회 우승 직후, "각자 하고 싶은 걸 하자" → 천마: "그래서 줄 게 있어."
  - 안씨표국 방문 → 위소운이 귀원검법에서 소무진 검법 흔적 발견 → 감동
  - 안세진(표국주)이 1차 잠금 해제 → 위소운이 2차 잠금(천마신공) 개봉
  - 이준혁이 처음으로 계산을 포기하는 장면 (감동)
- ⭐ **증류주 양조장 설립** (안씨표국 근처) → **첫 사업. 첫 수입원.**
  - **발효주 장인 스카우트**: 주덕삼(周德三) — 개봉 최고 양조사, 30년 경력 (퇴직 상태)
  - 2단계 공정: 발효(주덕삼 담당) → 증류(안씨세가 독점 비밀)
  - 이준혁이 증류 기술을 안씨세가 가족에게만 직접 전수
  - 안효림(장남)을 증류장 실무 책임자로 발탁 → 불만→능력 발휘→충성 전환 계기
  - ⭐ **VIP 홍보**: 팔기 전에 고위층에게 먼저 선물+만찬
    - 무림대회 인맥(문파 장로, 세가 가주)에게 시제품 선물
    - 개봉 유지(진만복, 풍만장 등)를 만찬 초대 → 이준혁 풀코스+증류주 페어링
    - "못 사게 하면 더 사고 싶다" = 럭셔리 브랜드 론칭 전략
    - 입소문이 퍼진 뒤 정식 납품 시작
  - 천화표국이 완제품을 개봉 객잔/주점에 납품 (유통 전담)
  - ⚠️ 기술 보호: 장인은 발효만, 안씨는 증류만, 표사는 배달만 = 3단 분리
- ⭐ **천화 연구방(硏究房) 설립** (증류장 옆 별채) — 조리사 훈련소 + 양념 R&D
  - 재료 전문가 스카우트: 손약령(孫若玲) — 약재상 3대째, 재료 감별의 달인
  - 천연 감칠맛 가루(천화선분) 개발: 건표고+다시마+멸치+건새우 = 천연 MSG
  - 현대 한국식 양념(참기름, 양념장 등) 중 이 시대 재료로 가능한 것 연구·개발
  - 조리사 훈련: 지방 지점 파견용 조리사 양성 (이준혁 레시피 표준화)
  - 해안 재료 조달 루트 개척: 손약령 인맥 + 천화표국 물류 = 시너지
  - ⭐⭐ **양념 독점 = 프랜차이즈 핵심 경쟁력**
    - 요리사 손맛도 중요하지만 **양념(천화선분·천화장)이 진짜 무기**
    - 다른 객잔이 요리사를 빼가도 → 양념 배합을 모르면 → **그 맛 재현 불가**
    - 양념 제조법 = 이준혁+손약령만 앎 → 지점에는 **완성된 양념만 공급**

  - ⭐⭐⭐ **무공 식품가공법 — 절대 보안 계층**
    - 1100년대에 분쇄기·믹서기·건조기가 없음 → **위소운의 내공으로 현대 기계 대체**

  - ⭐⭐ **합원진(合源陣) — 생산 연속성 시스템** (위소운 부재 대비)

## 8. 천마의 안가(隱家) 네트워크 — 사업 확장의 숨은 엔진

> ⚠️ **안가는 1개가 아니다. 마교교주였던 천마가 300년 전 전국 각지에 만들어둔 비밀 거점이다.**
> 개봉 안가는 그중 하나일 뿐. 사업이 확장되는 지역마다 새 안가가 열린다.

### 안가 시스템 개요

| 항목 | 내용 |
|------|------|
| **명칭** | 마교 교주 비밀 안가(隱家) — 비밀 거점/금고 |
| **설치자** | 천마 (300년 전 마교교주 시절) |
| **총 개수** | 천마 본인도 정확히 모름. "주요 도시에는 다 있어." |
| **300년 경과** | 일부는 훼손/발각됐을 수 있음. 열어봐야 안다. = 서스펜스 |
| **내용물** | 금/은/보석 + 무공 서적 + 약재 + 무기 (안가마다 다름) |
| **핵심 기능** | 사업 확장 시 해당 지역 안가를 열어 **자금 + 거점** 확보 |

### 개봉 안가 — 1차 자금 (무림대회 우승 직후)

| 항목 | 내용 |
|------|------|
| **위치** | 개봉(開封) 성내 |
| **타이밍** | 무림대회 우승 직후. "각자 하고 싶은 걸 하자" 결정 후 |
| **의미** | ⭐ **천마가 이준혁에게 주는 선물**. 7년간 소금밥의 보답. |
| **자금 규모** | 약 5,000~10,000냥 (천화루 1~3호점 + 초기 운영자금) |
| **천마 대사** | "밥값이야. 7년 동안 네놈이 해준 밥값. 빚진 거 싫어." |
| **이준혁 반응** | "이 ROI는...... 계산이 안 됩니다." (처음으로 계산을 포기하는 순간) |
| **위소운 반응** | 천마의 진심을 느낌. 겉으론 "밥값"이라지만 속으론 감사. |

### 안가 네트워크 — 사업 확장 연동

> 사업이 새 지역으로 확장될 때마다, 천마가 해당 지역 안가를 "열어준다."
> = 이준혁에게 자금을, 위소운에게 거점을, 천마에게 옛 기억을 선물하는 장치.

| 안가 | 지역 | 오픈 시기(경) | 용도 | 스토리 기능 |
|------|------|-------------|------|-----------|
| **개봉 안가** | 개봉 | ~10화 (무림대회 직후) | 1차 사업 자금 | 이준혁 선물. 3인격 유대 확인 |
| **소주 안가** | 소주 | 41~60화경 | 프랜차이즈 자금 | 천마의 과거 단서 1. "여기서... 한 놈을 죽였어." |
| **강남 안가** | 강남 | 76~100화경 | 대규모 확장 자금 | 천마의 과거 단서 2. 정마대전 관련 |
| **사천 안가** | 사천 | 151~175화경 | 전략 거점 | 천마의 핵심 과거. 마교 본거지 근처 |
| **개봉 대안가** | 개봉 지하 | 226~250화경 | 최종 전쟁 자원 | 천마의 마지막 비밀. 최종 결전 자금 |

> ⚠️ **안가의 3중 기능**: (1) 사업 자금 (2) 천마 과거 떡밥 (3) 3인격 유대 장면
> ⚠️ **안가를 열 때마다** 천마의 "밥값" 대사가 변형되며 감정이 깊어진다.
> - 개봉: "밥값이야." (퉁명)
> - 소주: "......또 밥값이야." (익숙해짐)
> - 강남: "네놈 밥값이 비싸." (인정)
> - 사천: (말없이 열어준다) (진심)
> - 최종: "빚은...... 다 갚았다." (완결)
