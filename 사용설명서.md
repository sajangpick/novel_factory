# Novel Alchemist — 대시보드 사용 설명서

---

## 접속 방법

| 환경 | 주소 |
|------|------|
| 이 컴퓨터 | http://localhost:3000 |
| 다른 컴퓨터 | https://novel-factory.onrender.com |

접속하면 대시보드가 나타납니다. 좌측 사이드바에 메뉴가 있습니다.

---

## 집필 흐름 (매일 반복)

```
① 현재 상태 → ② 전략 브리핑 → (③ 작품 자료 참고) → ④ 본문 집필 → ⑤ 품질 검수 → 확정
```

사이드바 상단 **"집필 프로젝트"** 메뉴를 **위에서 아래 순서**로 사용합니다.

---

## ① 현재 상태

> 지금까지 쓴 에피소드의 상태를 한눈에 확인하는 페이지

### 화면에 보이는 것

- **에피소드 동기화 패널**: 모든 에피소드가 카드로 나열됩니다
- **기억 카드 목록**: 확정된 에피소드의 핵심 요약

### 에피소드 상태 색깔

| 색깔 | 상태 | 의미 |
|------|------|------|
| 🟢 초록 | 확정됨 | 기억 카드가 만들어진 에피소드 |
| 🟡 노랑 | 미확정 | 본문은 있지만 아직 확정 안 한 에피소드 |
| 🟠 주황 | 수정됨 | 확정 후 본문이 바뀐 에피소드 (재확정 필요) |
| 🔴 빨강 | 삭제됨 | 기억 카드는 있는데 본문이 없는 에피소드 |

### 버튼

| 버튼 | 기능 |
|------|------|
| **확정** | AI가 해당 화를 분석 → 기억 카드 + 대시보드 자동 업데이트 |
| **롤백** | 잘못 확정한 경우 되돌리기 |

---

## ② 전략 브리핑

> 다음 화를 쓰기 전에 AI가 방향을 제안해주는 페이지

### 사용 순서

1. 화수를 확인합니다 (자동으로 다음 화가 선택됨)
2. **"AI 방향 제안"** 버튼을 클릭합니다
3. AI가 **3가지 스토리 방향**을 제안합니다
4. 마음에 드는 것을 참고하여 방향을 정합니다

### 추가 기능

| 기능 | 설명 |
|------|------|
| 떡밥 관리 | 보류 중인 떡밥 목록, 회수 시점 알림 |
| 캐릭터 캐스팅 | 이번 화에 등장할 인물 선택 (자동완성) |
| 주의사항 메모 | 이번 화에서 주의할 점 기록 |

> ⚠️ AI 제안을 다시 받으면 비용이 발생합니다. 확인 팝업이 뜹니다.

---

## ③ 작품 자료

> 이 소설(무림MNA)에만 해당하는 설정·인물·전략 자료를 모아본 페이지

### 포함된 자료 (22개 파일)

| 카테고리 | 주요 파일 |
|---------|----------|
| 🏛️ 지휘부 | 마스터 스토리 바이블, 소설 진행 마스터 |
| 📋 스토리 | 스토리 아크 상세, 무림대회 설정 |
| 👤 인물 | 캐릭터(주인공/조연/무림대회), 캐릭터 시스템 |
| ⚙️ 집필엔진 | 3인격 엔진, 감정 엔진, 기류감응 가이드 |
| 🗡️ 무공 | 주인공 무공 상세 |
| 🏢 조직세력 | 천화련 조직, 안씨표국, 세력도 |
| 💰 경제사업 | 경제 시스템, 사업 음식기술, 현대지식 DB |
| 🧭 전략방향 | 테마·주제의식, 경쟁작 차별화, 독자 타겟 분석 |

### 사용법

1. 왼쪽에서 카테고리를 펼쳐 파일을 클릭합니다
2. 오른쪽에 내용이 표시됩니다
3. 집필 전 관련 설정을 미리 확인하면 오류를 줄일 수 있습니다

> 💡 장르 공통 자료(무공 시스템, 지리, 음식, 무기 등)는 사이드바 하단 **"자료실 > 장르 공통 DB"** 에 있습니다.

## ④ 본문 집필

> AI가 소설 본문을 생성하고, 직접 편집하는 페이지

### 새 에피소드 쓰기

1. 화수를 선택합니다
2. **"AI 본문 생성"** 버튼을 클릭합니다
3. AI가 본문을 생성합니다 (이전 화 + 기억 카드 + 설계도를 참고)
4. 생성된 본문을 읽고, 필요하면 직접 수정합니다
5. **"저장"** 버튼을 클릭합니다

### 버튼

| 버튼 | 기능 |
|------|------|
| **AI 본문 생성** | AI가 새 본문을 만듭니다 |
| **저장** | 현재 내용을 파일 + 데이터베이스에 저장 |
| **확정하기** | 검수 없이 바로 확정 (기억 카드 자동 생성) |
| **폐기** | 이 화의 본문을 삭제 |

> 💡 저장하면 파일과 데이터베이스에 **동시 저장**됩니다. 다른 컴퓨터에서도 바로 반영됩니다.

## ⑤ 품질 검수 ⭐ (핵심 기능)

> AI가 참고자료와 대조하여 오류를 찾고, 바로 수정하는 페이지

### 화면 구성

- **좌측**: 행번호가 붙은 본문
- **우측**: "AI 검수 실행" 버튼
- **하단**: 빨간펜 지시창

---

### [A] 자동 검수 — "AI 검수 실행" 버튼

**AI가 알아서 오류를 찾아줍니다.**

1. 우측의 **"AI 검수 실행"** 버튼을 클릭합니다
2. 2~3분 기다립니다 (AI가 참고자료 전체를 읽고 대조합니다)
3. 발견된 이슈가 빨간펜 영역에 카드로 나타납니다

**AI가 찾는 것들:**

| 항목 | 예시 |
|------|------|
| 설정 충돌 | 인물 정보, 무공, 조직 구조가 바이블과 다른 경우 |
| 말투 위반 | 천마가 존댓말을 하거나, 이준혁이 반말하는 경우 |
| 현대어 사용 | M&A, CEO, 인프라 등 무협 세계에 없는 단어 |
| 무공 오류 | 전수 범위나 방식이 바이블 규칙과 다른 경우 |
| 스토리 모순 | 이전 화와 충돌하는 내용 |

**이슈 카드 사용법:**

```
① 행번호 버튼 클릭 → 해당 행으로 자동 스크롤
② "AI 수정" 버튼 클릭 → AI가 수정안을 만듭니다
③ 원본(빨간 취소선)과 수정본(초록색)이 비교 표시됩니다
④ "적용" 버튼 클릭 → 본문에 반영 + 자동 저장
```

---

### [B] 수동 교정 — 빨간펜 지시창

**AI가 못 찾은 문제는 직접 지시합니다.**

1. 본문에서 **문제가 있는 행을 클릭**합니다
2. 입력창에 행번호가 자동 입력됩니다
3. 뒤에 지시 내용을 적습니다
4. **"지시"** 버튼을 클릭합니다
5. AI가 분석 후 이슈 카드로 응답합니다
6. 이슈 카드에서 **"AI 수정" → "적용"** 으로 수정합니다

**지시 예시:**

| 입력 | AI가 하는 일 |
|------|-------------|
| `194행: 현대어를 무협 용어로 고쳐줘` | 194행의 현대어를 찾아 시대에 맞게 수정 제안 |
| `384행: 천마 편지가 바이블에 없어` | 384행 내용을 바이블과 대조하여 오류 검증 |
| `이 화에서 천마 말투 전부 검사해줘` | 전체 본문에서 천마 말투 위반 검색 |
| `15화 열어줘` | 15화로 이동 |
| `200행 보여줘` | 200행으로 스크롤 |

---

### [C] 확정하기

검수와 수정이 끝나면:

1. 하단의 **"확정하기"** 버튼을 클릭합니다
2. AI가 본문을 분석하여 **기억 카드 + 대시보드**를 자동 업데이트합니다
3. 확정된 에피소드는 이후 화 집필 시 AI가 참조합니다

## 자료실 (가끔 사용)

사이드바 하단의 **"자료실 ▼"** 를 클릭하면 펼쳐집니다.

| 메뉴 | 용도 | 언제 사용 |
|------|------|----------|
| **장르 공통 DB** | 집필 규칙, 문체, 무공 시스템, 지리, 음식, 무기, 의복, 용어집 등 | 장르(무협) 공통 설정 확인/수정할 때 |
| **인명록 관리** | 캐릭터 정보 검색/수정 | 새 캐릭터 추가, 정보 수정할 때 |
| **AI 교정** | 별도 빨간펜 교정 도구 | 추가 교정할 때 |

> 💡 장르 공통 DB에는 다른 무협소설을 쓸 때도 재사용할 수 있는 15개 파일이 있습니다.
> 이 소설 전용 자료는 집필 프로젝트의 **"작품 자료"** 에서 확인하세요.

## 자주 묻는 질문

**Q: 품질 검수에서 수정한 내용이 본문 집필에도 반영되나요?**

네. 파일과 데이터베이스에 동시 저장되므로, 모든 페이지에서 수정된 내용이 보입니다.

---

**Q: 다른 컴퓨터에서 사용하려면?**

`https://novel-factory.onrender.com` 주소로 접속하면 됩니다.
데이터는 클라우드 DB에 있으므로 어디서든 동일합니다.

---

**Q: AI 검수가 오래 걸려요 (2~3분)**

정상입니다. AI가 바이블, 인명록, 세계관 DB를 전부 읽고 대조하기 때문입니다.

---

**Q: "분석중..."이 1분 넘게 지속돼요**

AI 사용량 제한에 걸린 경우 자동으로 30초 대기 후 재시도합니다.
에러가 아니니 기다리시면 됩니다.

---

**Q: 확정을 잘못 눌렀어요**

"현재 상태" 페이지에서 해당 에피소드의 **"롤백"** 버튼을 누르면 되돌릴 수 있습니다.

---

**Q: AI 비용은 얼마인가요?**

| 작업 | 비용 |
|------|------|
| AI 검수 실행 1회 | 약 200원 |
| AI 수정 1건 | 약 7원 |
| 빨간펜 지시 1건 | 약 140원 |
| 본문 생성 1화 | 약 300~500원 |
| **에피소드 1화 풀코스** | **약 700원** |

---

**Q: 이전 화를 수정하면 다음 화에 영향이 있나요?**

이전 화를 수정 후 재확정하면 기억 카드가 업데이트됩니다.
다음 화 집필 시 AI가 갱신된 기억 카드를 참조합니다.
